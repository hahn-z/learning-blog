## 文章格式

每篇文章包含 YAML frontmatter：

```yaml
---
title: "文章标题"
slug: "url-slug"
category: "分类名称"
tech_stack: "技术栈"
created_at: "创建时间"
updated_at: "更新时间"
reading_time: 预估阅读分钟数
tags: [标签1, 标签2]
---
```
