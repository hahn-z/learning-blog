# 015 - 服务发现机制与负载均衡策略

> **难度：** ⭐⭐⭐ | **前置知识：** 服务注册（014）、Nacos 基础（011）| **预估用时：** 50 分钟

## 概念讲解

**一句话定义：** 服务发现是指从注册中心获取目标服务的所有可用实例列表，负载均衡则是根据特定策略选择其中一个实例发起调用。

**现实类比：** 就像你点外卖——平台帮你找到附近所有营业中的餐厅（服务发现），然后根据距离、销量、评分等因素推荐一家（负载均衡）。如果推荐的餐厅打烊了，平台会自动换一家（故障转移）。

**技术场景：** 订单服务需要调用用户服务获取用户信息。Nacos 返回 3 个用户服务实例，负载均衡器根据随机/轮询/权重策略选择一个。被选中的实例响应慢或超时，重试时切换到另一个实例。

### 自定义负载均衡器 — WeightedRandomBalancer.php

```php
<?php

declare(strict_types=1);

namespace App\LoadBalancer;

use Hyperf\LoadBalancer\LoadBalancerInterface;
use Hyperf\LoadBalancer\LoadBalancerManager;
use Hyperf\LoadBalancer\Node;

class WeightedRandomBalancer implements LoadBalancerInterface
{
    private ?LoadBalancerManager $manager = null;

    public function setManager(LoadBalancerManager $manager): void
    {
        $this->manager = $manager;
    }

    /**
     * ✅ Select instance based on weight from Nacos metadata
     */
    public function select(array ...$arguments): ?Node
    {
        if ($this->manager === null) {
            return null;
        }

        $nodes = $this->manager->getNodes();

        if (empty($nodes)) {
            return null;
        }

        // Calculate total weight
        $totalWeight = 0;
        foreach ($nodes as $node) {
            $weight = $node->weight ?? 1;
            $totalWeight += $weight;
        }

        if ($totalWeight <= 0) {
            // Fallback to random if no weights
            return $nodes[array_rand($nodes)];
        }

        // Weighted random selection
        $random = mt_rand(1, $totalWeight);
        $currentWeight = 0;

        foreach ($nodes as $node) {
            $weight = $node->weight ?? 1;
            $currentWeight += $weight;

            if ($random <= $currentWeight) {
                return $node;
            }
        }

        return $nodes[array_rand($nodes)];
    }
}
```

### 一致性哈希负载均衡器 — ConsistentHashBalancer.php

```php
<?php

declare(strict_types=1);

namespace App\LoadBalancer;

use Hyperf\LoadBalancer\LoadBalancerInterface;
use Hyperf\LoadBalancer\LoadBalancerManager;
use Hyperf\LoadBalancer\Node;
use Swoole\Coroutine;

class ConsistentHashBalancer implements LoadBalancerInterface
{
    private ?LoadBalancerManager $manager = null;
    private int $virtualNodes = 150;  // 🟡 Virtual nodes for better distribution
    private array $hashRing = [];

    public function setManager(LoadBalancerManager $manager): void
    {
        $this->manager = $manager;
        $this->buildHashRing();
    }

    /**
     * ✅ Consistent hashing for sticky sessions or cache-friendly routing
     * Same key always routes to same instance (until instances change)
     */
    public function select(array ...$arguments): ?Node
    {
        if ($this->manager === null) {
            return null;
        }

        $nodes = $this->manager->getNodes();

        if (empty($nodes)) {
            return null;
        }

        // Use first argument as hash key (e.g., user_id)
        $hashKey = $arguments[0]['hash_key'] ?? 'default';
        $hash = crc32((string) $hashKey);

        // Find next node in hash ring
        foreach ($this->hashRing as $ringHash => $node) {
            if ($ringHash >= $hash) {
                return $node;
            }
        }

        // Wrap around to first node
        return $this->hashRing[array_key_first($this->hashRing)];
    }

    private function buildHashRing(): void
    {
        $this->hashRing = [];
        $nodes = $this->manager->getNodes();

        foreach ($nodes as $node) {
            $nodeKey = $node->host . ':' . $node->port;

            for ($i = 0; $i < $this->virtualNodes; $i++) {
                $virtualKey = "{$nodeKey}#{$i}";
                $hash = crc32($virtualKey);
                $this->hashRing[$hash] = $node;
            }
        }

        ksort($this->hashRing);
    }
}
```

### RPC 客户端使用服务发现 — UserServiceClient.php

```php
<?php

declare(strict_types=1);

namespace App\Service;

use Hyperf\Guzzle\ClientFactory;
use Hyperf\LoadBalancer\LoadBalancerInterface;
use Hyperf\LoadBalancer\LoadBalancerManager;
use Hyperf\ServiceGovernance\DriverInterface;
use Psr\Log\LoggerInterface;

class UserServiceClient
{
    public function __construct(
        private ClientFactory $clientFactory,
        private LoggerInterface $logger,
        private DriverInterface $driver
    ) {}

    /**
     * ✅ Call user service with automatic service discovery and load balancing
     */
    public function getUser(int $userId): ?array
    {
        $serviceName = 'user-service';

        // 1. Get instances from Nacos
        $instances = $this->driver->getNodes($serviceName);

        if (empty($instances)) {
            $this->logger->error('No instances found', ['service' => $serviceName]);
            return null;
        }

        // 2. Load balancer selects instance
        $selected = $instances[array_rand($instances)];

        $url = "http://{$selected->host}:{$selected->port}/api/users/{$userId}";

        // 3. Make request with retry
        $client = $this->clientFactory->create();

        for ($attempt = 1; $attempt <= 3; $attempt++) {
            try {
                $response = $client->get($url, ['timeout' => 2]);

                $data = json_decode($response->getBody()->getContents(), true);

                if ($data['code'] === 0) {
                    return $data['data'];
                }

                $this->logger->warning('User service returned error', [
                    'url' => $url,
                    'response' => $data,
                ]);

                break;
            } catch (\Throwable $e) {
                $this->logger->error('User service request failed', [
                    'attempt' => $attempt,
                    'url' => $url,
                    'error' => $e->getMessage(),
                ]);

                // 🟡 Retry with different instance
                $selected = $instances[array_rand($instances)];
                $url = "http://{$selected->host}:{$selected->port}/api/users/{$userId}";
            }
        }

        return null;
    }

    /**
     * ✅ Call with sticky routing (consistent hash)
     */
    public function getUserSticky(int $userId): ?array
    {
        $serviceName = 'user-service';
        $instances = $this->driver->getNodes($serviceName);

        if (empty($instances)) {
            return null;
        }

        // Use user ID as hash key for sticky routing
        $hash = crc32((string) $userId);
        $selected = $instances[$hash % count($instances)];

        $url = "http://{$selected->host}:{$selected->port}/api/users/{$userId}";

        $client = $this->clientFactory->create();

        try {
            $response = $client->get($url, ['timeout' => 2]);
            $data = json_decode($response->getBody()->getContents(), true);

            return $data['code'] === 0 ? $data['data'] : null;
        } catch (\Throwable $e) {
            $this->logger->error('User service sticky call failed', [
                'user_id' => $userId,
                'instance' => "{$selected->host}:{$selected->port}",
                'error' => $e->getMessage(),
            ]);
            return null;
        }
    }
}
```

### 负载均衡测试脚本 — test-loadbalancer.php

```php
<?php
// ✅ Test load balancer strategies

declare(strict_types=1);

$iterations = 1000;

echo "🧪 Load Balancer Strategy Test\n";
echo "Iterations: {$iterations}\n\n";

// Mock instances with different weights
$instances = [
    ['ip' => '10.0.1.11', 'port' => 9501, 'weight' => 1],
    ['ip' => '10.0.1.12', 'port' => 9501, 'weight' => 2],
    ['ip' => '10.0.1.13', 'port' => 9501, 'weight' => 3],
];

// 1. Random strategy
echo "1️⃣ Random Strategy:\n";
$counts = ['11' => 0, '12' => 0, '13' => 0];

for ($i = 0; $i < $iterations; $i++) {
    $selected = $instances[array_rand($instances)];
    $key = substr($selected['ip'], -2);
    $counts[$key]++;
}

foreach ($counts as $ip => $count) {
    $percent = round($count / $iterations * 100, 1);
    echo "   10.0.1.{$ip}: {$count} ({$percent}%)\n";
}

// 2. Weighted random
echo "\n2️⃣ Weighted Random Strategy:\n";
$counts = ['11' => 0, '12' => 0, '13' => 0];
$totalWeight = array_sum(array_column($instances, 'weight'));

for ($i = 0; $i < $iterations; $i++) {
    $random = mt_rand(1, $totalWeight);
    $currentWeight = 0;

    foreach ($instances as $instance) {
        $currentWeight += $instance['weight'];
        if ($random <= $currentWeight) {
            $key = substr($instance['ip'], -2);
            $counts[$key]++;
            break;
        }
    }
}

foreach ($counts as $ip => $count) {
    $percent = round($count / $iterations * 100, 1);
    echo "   10.0.1.{$ip}: {$count} ({$percent}%)\n";
}

echo "\n✅ Test complete!\n";
```

## 注意事项

| 约束 | 说明 | 违反后果 |
|------|------|----------|
| 🔴 健康检查过滤 | 只选择 healthy=true 的实例 | 调用已下线/异常实例 |
| 🟡 权重范围 | 权重建议 1-10，避免过大 | 权重过大导致分配不均 |
| 🟢 实例缓存 | 本地缓存实例列表，避免频繁请求 Nacos | 每次都查 Nacos，性能差 |
| 🔴 超时重试 | 重试要换实例，不能重试同一实例 | 雪崩效应 |
| 🟡 一致性哈希 | 虚拟节点数影响分布均匀度 | 虚拟节点太少导致倾斜 |

## 避坑指南

| 错误 | 现象 | 正确做法 |
|------|------|----------|
| 未缓存实例 | 每次请求都查 Nacos | 用本地缓存，定期刷新 |
| 重试同一实例 | 超时后继续重试同一 IP | 重试前随机选择新实例 |
| 权重为 0 | 实例永远不会被选中 | 权重最小设为 1 |
| 未过滤健康状态 | 调用不健康的实例 | 只选择 healthy=true |
| 一致性哈希无虚拟节点 | 实例上下线导致大量路由变动 | 设置 100-200 个虚拟节点 |

## 练习题

### 🟢 基础
1. 实现一个简单的随机负载均衡器，从 3 个实例中选择一个。
2. 修改实例权重，验证加权随机策略的分布是否符合预期。

### 🟡 进阶
3. 实现一个基于响应时间的自适应负载均衡：响应快的实例获得更高权重。

### 🔴 开放
4. 设计一个方案：当某个实例错误率超过阈值时，暂时将其从负载均衡中摘除（熔断机制）。

## 举一反三

| 场景 | 适配方式 | 关键差异 |
|------|----------|----------|
| 会话保持 | 使用一致性哈希或基于 IP 路由 | 确保同一用户访问同一实例 |
| 地域优化 | 优先选择同地域实例 | 减少跨地域延迟 |
| 灰度发布 | 按用户 ID 或权重分配到不同版本 | 实现金丝雀发布 |
