1. 概念讲解：什么是面向对象？

面向对象编程（Object-Oriented Programming, OOP）是一种编程范式，核心思想是用"对象"来组织代码。

现实世界中，你看到的每样东西都可以是一个"对象"——一只猫、一辆车、一个银行账户。每个对象都有：

  • 属性（Attribute）：描述对象的特征（猫的名字、颜色）
  • 方法（Method）：对象能做的事（猫会叫、会跑）

    • 类（Class） 是创建对象的蓝图/模板。对象（Object） 是类的具体实例。

    类比关系：
  类 = 汽车设计图纸
  对象 = 按图纸造出来的每一辆车

    四大支柱（本文聚焦前两个）

    支柱 含义 本文
    封装 把数据和行为绑在一起，隐藏内部细节 ✅
    抽象 只暴露必要接口，隐藏复杂实现 ✅
    继承 子类复用父类的代码 下一篇
    多态 同一接口，不同实现 下一篇

    3. 代码演进：从过程式到面向对象

    v1：过程式写法（不推荐）

    # v1: Procedural style - data and logic are separated

def create_student(name, age, grades):
    return {"name": name, "age": age, "grades": grades}

def average_grade(student):
    return sum(student["grades"]) / len(student["grades"]) if student["grades"] else 0

def student_info(student):
    return f"{student['name']}, age {student['age']}, avg: {average_grade(student):.1f}"

# Usage
s1 = create_student("Alice", 20, [85, 90, 78])
print(student_info(s1))  # Alice, age 20, avg: 84.3

    问题： 数据（字典）和函数散落各处，没有绑定关系，容易传错参数。

    v2：基本类写法

    # v2: Basic class - bind data and methods together

class Student:
    def __init__(self, name, age, grades=None):
        self.name = name
        self.age = age
        self.grades = grades or []

    def average(self):
        return sum(self.grades) / len(self.grades) if self.grades else 0

    def info(self):
        return f"{self.name}, age {self.age}, avg: {self.average():.1f}"

s1 = Student("Alice", 20, [85, 90, 78])
print(s1.info())  # Alice, age 20, avg: 84.3

    v3：完整版（封装 + 属性管理 + repr）

    # v3: Full OOP with encapsulation, property, and repr

class Student:
    """Represents a student with grades tracking."""

    # Class attribute: shared across all instances
    school_name = "Python University"

    def __init__(self, name: str, age: int):
        self.name = name          # public attribute
        self._age = age           # protected (convention only)
        self.__grades = []        # private (name mangling)

    # --- Property: controlled access to grades ---
    @property
    def grades(self):
        """Read-only view of grades."""
        return self.__grades[:]

    def add_grade(self, score: float):
        """Add a grade with validation."""
        if 0 <= score <= 100:
            self.__grades.append(score)
        else:
            raise ValueError(f"Invalid score: {score}")

    # --- Property: age with validation ---
    @property
    def age(self):
        return self._age

    @age.setter
    def age(self, value):
        if value < 0 or value > 150:
            raise ValueError(f"Invalid age: {value}")
        self._age = value

    # --- Methods ---
    def average(self) -> float:
        """Calculate average grade."""
        return sum(self.__grades) / len(self.__grades) if self.__grades else 0.0

    def __repr__(self):
        return f"Student({self.name!r}, age={self._age}, avg={self.average():.1f})"

    def __str__(self):
        return f"📚 {self.name} (avg: {self.average():.1f})"


# --- Class method & static method ---
class StudentManager:
    """Manages a collection of students."""

    _registry = {}  # class-level storage

    @classmethod
    def register(cls, student: Student):
        cls._registry[student.name] = student
        print(f"Registered: {student}")

    @classmethod
    def get(cls, name: str):
        return cls._registry.get(name)

    @staticmethod
    def is_passing(average: float) -> bool:
        """No cls/self needed — pure utility."""
        return average >= 60.0


# --- Usage ---
if __name__ == "__main__":
    s1 = Student("Alice", 20)
    s1.add_grade(85)
    s1.add_grade(92)
    s1.add_grade(78)

    print(s1)           # 📚 Alice (avg: 85.0)
    print(repr(s1))     # Student('Alice', age=20, avg=85.0)

    StudentManager.register(s1)
    print(StudentManager.is_passing(s1.average()))  # True

    s1.age = 21         # uses setter
    # s1.age = -5       # ValueError!
    # s1.add_grade(150) # ValueError!

    5. 注意事项

    项目 说明
    self 必须 实例方法第一个参数永远是 self，Python 不会自动省略
    __init__ 不是构造函数 它是初始化方法，真正构造是 __new__，但日常不用管
    类属性共享 所有实例共享同一个类属性，修改时要注意
    _ vs __ _ 是约定（别人还能改），__ 是名称改写（_Class__attr）
    @property 不是真私有 只是控制访问入口，执着的人还是能绕过

    6. 避坑指南

    ❌ 错误写法 ✅ 正确写法 原因
    def init(self): def __init__(self): 双下划线！少写一个就不执行
    忘写 self 参数 方法第一个参数写 self Python 不会隐式传递
    在类外直接 obj.__grades 用 @property 提供访问器 私有变量外部访问会报 AttributeError
    可变对象做默认值 grades=[] grades=None 然后 or [] 默认值在类定义时创建，所有实例共享
    __repr__ 返回中文大段 返回 ClassName(field=value) 格式 repr() 用于调试，要能重建对象

    7. 练习题

    🟢 入门：创建一个 Dog 类

    # Create a Dog class with:
# - name, breed attributes
# - bark() method that returns f"{name} says Woof!"
# - __str__ returns "🐕 name (breed)"

    参考答案

    class Dog:
    def __init__(self, name: str, breed: str):
        self.name = name
        self.breed = breed

    def bark(self):
        return f"{self.name} says Woof!"

    def __str__(self):
        return f"🐕 {self.name} ({self.breed})"

rex = Dog("Rex", "German Shepherd")
print(rex.bark())  # Rex says Woof!
print(rex)         # 🐕 Rex (German Shepherd)

    🟡 中级：BankAccount 带封装

    创建 BankAccount 类：私有余额，deposit/withdraw 方法带校验，@property balance 只读，__repr__ 显示余额。

    参考答案

    class BankAccount:
    def __init__(self, owner: str, balance: float = 0):
        self.owner = owner
        self.__balance = balance

    @property
    def balance(self):
        return self.__balance

    def deposit(self, amount: float):
        if amount <= 0:
            raise ValueError("Deposit must be positive")
        self.__balance += amount
        return self.__balance

    def withdraw(self, amount: float):
        if amount <= 0:
            raise ValueError("Withdraw must be positive")
        if amount > self.__balance:
            raise ValueError("Insufficient funds")
        self.__balance -= amount
        return self.__balance

    def __repr__(self):
        return f"BankAccount({self.owner!r}, balance={self.__balance:.2f})"

acc = BankAccount("Alice", 100)
acc.deposit(50)
print(acc)              # BankAccount('Alice', balance=150.00)
print(acc.balance)      # 150.0
# acc.balance = 999     # AttributeError: can't set attribute

    🔴 挑战：Temperature 类（双单位转换）

    创建 Temperature 类，存储开尔文温度，用 @property 提供 celsius 和 fahrenheit 的读写（设置时自动转换回开尔文）。

    参考答案

    class Temperature:
    def __init__(self, celsius: float = 0):
        self._kelvin = celsius + 273.15

    @property
    def celsius(self):
        return self._kelvin - 273.15

    @celsius.setter
    def celsius(self, value):
        self._kelvin = value + 273.15

    @property
    def fahrenheit(self):
        return self.celsius * 9 / 5 + 32

    @fahrenheit.setter
    def fahrenheit(self, value):
        self.celsius = (value - 32) * 5 / 9

    def __repr__(self):
        return f"Temperature(celsius={self.celsius:.1f})"

t = Temperature(100)  # boiling point
print(t.fahrenheit)    # 212.0
t.fahrenheit = 32     # freezing point
print(t.celsius)       # 0.0
print(repr(t))         # Temperature(celsius=0.0)

    8. 知识点总结

    面向对象基础
├── 类定义 class ClassName:
│   ├── __init__(self, ...) → 初始化实例属性
│   └── 方法 def method(self, ...)
├── self 关键字
│   ├── 引用当前实例
│   └── 必须是实例方法第一个参数
├── 属性类型
│   ├── 公有 (self.name) → 任何人可访问
│   ├── 保护 (self._name) → 约定不改
│   └── 私有 (self.__name) → 名称改写 _Class__name
├── 特殊方法
│   ├── __repr__ → repr(obj), 开发调试
│   └── __str__ → str(obj)/print(obj), 用户展示
├── 装饰器方法
│   ├── @property → 像属性一样调用方法
│   ├── @classmethod → 操作类级别数据
│   └── @staticmethod → 纯工具函数
└── 类属性 vs 实例属性
    ├── 类属性 → 所有实例共享
    └── 实例属性 → self.xxx, 各自独立
