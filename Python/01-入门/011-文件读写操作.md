## 一、概念讲解

### 为什么要学文件操作？

程序的数据不能只活在内存里。断电就没了。文件操作让你的程序能够**持久化数据**——保存配置、记录日志、处理数据文件。

### Python 文件操作的核心：`open()` 函数

```python
# Basic but manual - must remember to close
file = open("data.txt", "r")
content = file.read()
file.close()

# Better: with statement (auto-close)
with open("data.txt", "r") as f:
    content = f.read()
# File is automatically closed here
```

### 文件打开模式

| 模式 | 说明 | 文件不存在时 | 文件存在时 |
|------|------|-------------|-----------|
| `r` | 只读 | ❌ 报错 | 从头读 |
| `w` | 只写 | ✅ 创建 | ⚠️ 清空 |
| `a` | 追加 | ✅ 创建 | 末尾追加 |
| `x` | 独占创建 | ✅ 创建 | ❌ 报错 |
| `r+` | 读写 | ❌ 报错 | 从头读写 |
| `rb` / `wb` | 二进制模式 | 同上 | 同上 |

### 文件路径基础

```python
import os

# Current working directory
print(os.getcwd())

# Join paths (cross-platform)
path = os.path.join("data", "files", "report.txt")

# Better: use pathlib (Python 3.4+)
from pathlib import Path
p = Path("data") / "files" / "report.txt"
```

## 三、完整代码示例

### 3.1 基础读写

```python
# === Basic file read/write ===

# Write to a file
with open("hello.txt", "w", encoding="utf-8") as f:
    f.write("Hello, Python!\n")
    f.write("File I/O is easy.\n")

# Read entire file
with open("hello.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)
# Output:
# Hello, Python!
# File I/O is easy.

# Read line by line
with open("hello.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())  # strip() removes trailing newline

# Read all lines into a list
with open("hello.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()
    print(lines)  # ['Hello, Python!\n', 'File I/O is easy.\n']
```

### 3.2 追加与多行写入

```python
# === Append and multi-line write ===

# Append to existing file
with open("hello.txt", "a", encoding="utf-8") as f:
    f.write("This line is appended.\n")

# Write multiple lines at once
lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
with open("hello.txt", "a", encoding="utf-8") as f:
    f.writelines(lines)
```

### 3.3 CSV 文件处理

```python
# === CSV file handling ===
import csv

# Write CSV
data = [
    ["name", "age", "city"],
    ["Alice", 30, "Beijing"],
    ["Bob", 25, "Shanghai"],
    ["Charlie", 35, "Shenzhen"],
]

with open("people.csv", "w", newline="", encoding="utf-8") as f:
    writer = csv.writer(f)
    writer.writerows(data)

# Read CSV
with open("people.csv", "r", encoding="utf-8") as f:
    reader = csv.reader(f)
    for row in reader:
        print(row)
# ['name', 'age', 'city']
# ['Alice', '30', 'Beijing']

# CSV with DictReader/DictWriter
with open("people.csv", "r", encoding="utf-8") as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(f"{row['name']} is {row['age']} years old")
```

### 3.4 JSON 文件处理

```python
# === JSON file handling ===
import json

# Write JSON
config = {
    "app_name": "MyApp",
    "version": "1.0.0",
    "debug": True,
    "users": ["Alice", "Bob"],
}

with open("config.json", "w", encoding="utf-8") as f:
    json.dump(config, f, indent=2, ensure_ascii=False)

# Read JSON
with open("config.json", "r", encoding="utf-8") as f:
    loaded = json.load(f)
    print(loaded["app_name"])  # MyApp
    print(loaded["users"])     # ['Alice', 'Bob']
```

### 3.5 pathlib 现代文件操作

```python
# === Modern file operations with pathlib ===
from pathlib import Path

# Create Path object
p = Path("data")

# Create directory (if not exists)
p.mkdir(exist_ok=True)

# Write and read using Path methods
file_path = p / "notes.txt"
file_path.write_text("Hello from pathlib!\n", encoding="utf-8")
content = file_path.read_text(encoding="utf-8")
print(content)

# List all .txt files in directory
for txt_file in p.glob("*.txt"):
    print(txt_file.name, txt_file.stat().st_size, "bytes")

# File info
print(file_path.exists())   # True
print(file_path.is_file())  # True
print(file_path.suffix)     # .txt
print(file_path.stem)       # notes
print(file_path.parent)     # data
```

### 3.6 大文件处理

```python
# === Processing large files efficiently ===

# Method 1: Read line by line (memory efficient)
def count_lines(filepath):
    count = 0
    with open(filepath, "r", encoding="utf-8") as f:
        for line in f:
            count += 1
    return count

# Method 2: Chunked reading for binary files
def read_in_chunks(filepath, chunk_size=8192):
    with open(filepath, "rb") as f:
        while chunk := f.read(chunk_size):
            yield chunk

# Method 3: Process CSV line by line (streaming)
import csv
def process_large_csv(filepath):
    with open(filepath, "r", encoding="utf-8") as f:
        reader = csv.DictReader(f)
        for row in reader:
            yield row["name"]
```

## 五、注意事项

| 事项 | 说明 | 建议 |
|------|------|------|
| 编码 | Windows 默认 GBK，Linux/Mac 默认 UTF-8 | **始终指定 `encoding="utf-8"`** |
| 文件关闭 | 忘记 `close()` 会导致资源泄露 | 始终用 `with` 语句 |
| 路径分隔符 | Windows `\`，Linux/Mac `/` | 用 `os.path.join` 或 `pathlib` |
| 写模式覆盖 | `"w"` 模式会清空文件 | 需追加用 `"a"`，需保护用 `"x"` |
| 大文件内存 | `read()` 一次性加载全部内容 | 用逐行读取或分块读取 |
| 换行符 | Windows `\r\n`，Linux `\n` | Python 自动处理（文本模式） |

## 六、避坑指南

### ❌ 坑1：忘记指定编码

```python
# ❌ Windows 上读中文文件可能乱码
with open("data.txt", "r") as f:
    content = f.read()  # UnicodeDecodeError!

# ✅ 始终指定编码
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()
```

### ❌ 坑2：忘记关闭文件

```python
# ❌ 没有关闭文件
f = open("data.txt", "w")
f.write("some data")
# If exception occurs here, file stays open

# ✅ 使用 with 语句自动关闭
with open("data.txt", "w") as f:
    f.write("some data")
```

### ❌ 坑3：一次读取超大文件

```python
# ❌ 10GB 文件直接 read() -> 内存爆炸
content = open("huge.log").read()

# ✅ 逐行处理
with open("huge.log", "r") as f:
    for line in f:
        if "ERROR" in line:
            print(line.strip())
```

### v3：封装为工具类

```python
# Reusable file utility
import json
from pathlib import Path
from typing import Any

class FileManager:
    """Simple file manager for common operations."""

    @staticmethod
    def read_text(filepath: str, encoding: str = "utf-8") -> str:
        return Path(filepath).read_text(encoding=encoding)

    @staticmethod
    def write_text(filepath: str, content: str, encoding: str = "utf-8") -> None:
        Path(filepath).write_text(content, encoding=encoding)

    @staticmethod
    def read_json(filepath: str, encoding: str = "utf-8") -> Any:
        return json.loads(Path(filepath).read_text(encoding=encoding))

    @staticmethod
    def write_json(filepath: str, data: Any, encoding: str = "utf-8") -> None:
        Path(filepath).write_text(
            json.dumps(data, indent=2, ensure_ascii=False),
            encoding=encoding,
        )

# Usage:
# fm = FileManager()
# fm.write_json("config.json", {"key": "value"})
# config = fm.read_json("config.json")
```
