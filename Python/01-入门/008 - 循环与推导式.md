# 循环与推导式

> **难度标注：** ⭐⭐ 初级
>
> 适合已掌握条件判断，准备学习循环和推导式的读者。

## 1. 概念讲解

循环是让计算机重复执行某段代码的机制。Python 提供了 `for` 和 `while` 两种循环，以及独特的推导式（comprehension）语法来优雅地创建序列。

**核心概念：**

- **for 循环**：遍历可迭代对象（列表、字符串、字典、range 等），逐个取出元素
- **while 循环**：条件为真时反复执行，适合不确定次数的循环
- **break / continue**：break 立即跳出整个循环，continue 跳过当前轮次
- **循环的 else 子句**：循环正常结束（没被 break）时执行 else 块——这是 Python 的独有特性
- **推导式**：用一行代码从可迭代对象创建新的列表/集合/字典，是 Python 最优雅的特性之一
- **生成器表达式**：类似列表推导式但惰性求值，节省内存

**为什么推导式重要？**

推导式是 Python 区别于其他语言的重要特征。同样的功能，Java 需要 5-8 行，Python 推导式一行搞定，且可读性更强。熟练掌握推导式是写出 Pythonic 代码的关键。

## 3. 完整代码示例

```python
"""
Python 循环与推导式 - 完整示例
Covering: for, while, break/continue, else, comprehensions, generator expressions
"""

# --- 1. for loop basics ---
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
# apple, banana, cherry

# range(start, stop, step)
for i in range(5):
    print(i, end=" ")  # 0 1 2 3 4
print()

for i in range(2, 10, 3):
    print(i, end=" ")  # 2 5 8
print()


# --- 2. enumerate and zip ---
# enumerate: get index + value
for idx, fruit in enumerate(fruits):
    print(f"{idx}: {fruit}")  # 0: apple, 1: banana, 2: cherry

# zip: iterate in parallel
prices = [3.5, 2.8, 4.0]
for fruit, price in zip(fruits, prices):
    print(f"{fruit}: ${price}")  # apple: $3.5, banana: $2.8, cherry: $4.0


# --- 3. Iterate over dict ---
person = {"name": "Alice", "age": 30, "city": "Beijing"}

# Keys (default)
for key in person:
    print(key)

# Key-value pairs
for key, value in person.items():
    print(f"{key} = {value}")


# --- 4. while loop ---
# Basic while
count = 0
while count < 5:
    count += 1
print(f"count = {count}")  # count = 5

# while with break (infinite loop pattern)
while True:
    response = "quit"  # simulated input
    if response == "quit":
        break
print("Exited loop")


# --- 5. break, continue, and else ---
# break: exit the entire loop
for n in range(10):
    if n == 5:
        break
print(f"Stopped at {n}")  # Stopped at 5

# continue: skip current iteration
evens = []
for n in range(10):
    if n % 2 != 0:
        continue
    evens.append(n)
print(f"Evens: {evens}")  # Evens: [0, 2, 4, 6, 8]

# loop else: runs only if loop wasn't broken
for n in range(2, 10):
    for i in range(2, n):
        if n % i == 0:
            break
    else:
        print(f"{n} is prime")  # 2, 3, 5, 7


# --- 6. List comprehension ---
# Basic: create a new list
squares = [x ** 2 for x in range(10)]
print(squares)  # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# With condition (filter)
even_squares = [x ** 2 for x in range(10) if x % 2 == 0]
print(even_squares)  # [0, 4, 16, 36, 64]

# With ternary (map + filter)
labels = ["even" if x % 2 == 0 else "odd" for x in range(5)]
print(labels)  # ['even', 'odd', 'even', 'odd', 'even']

# Nested comprehension (flatten a 2D list)
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
flat = [x for row in matrix for x in row]
print(flat)  # [1, 2, 3, 4, 5, 6, 7, 8, 9]


# --- 7. Dict comprehension ---
# Basic dict comprehension
word = "hello"
char_count = {c: word.count(c) for c in set(word)}
print(char_count)  # {'h': 1, 'e': 1, 'l': 2, 'o': 1}

# Swap keys and values
original = {"a": 1, "b": 2, "c": 3}
swapped = {v: k for k, v in original.items()}
print(swapped)  # {1: 'a', 2: 'b', 3: 'c'}


# --- 8. Set comprehension ---
# Remove duplicates while transforming
nums = [1, 2, 2, 3, 3, 3, 4]
unique_squares = {x ** 2 for x in nums}
print(unique_squares)  # {1, 4, 9, 16}


# --- 9. Generator expression ---
# Lazy evaluation - doesn't build the full list in memory
total = sum(x ** 2 for x in range(1_000_000))
print(f"Sum of squares: {total}")

# Compare memory usage
import sys
list_comp = [x for x in range(10000)]
gen_expr = (x for x in range(10000))
print(f"List size: {sys.getsizeof(list_comp)} bytes")   # ~87616 bytes
print(f"Gen size: {sys.getsizeof(gen_expr)} bytes")     # ~200 bytes


# --- 10. Practical: word frequency counter ---
text = "the quick brown fox jumps over the lazy dog the fox"

word_freq = {}
for word in text.split():
    word_freq[word] = word_freq.get(word, 0) + 1

# Same thing with dict comprehension + count
word_freq_v2 = {word: text.split().count(word) for word in set(text.split())}

# Best approach: collections.Counter (production code)
from collections import Counter
word_freq_v3 = Counter(text.split())

print(word_freq_v3.most_common(3))  # [('the', 3), ('fox', 2), ...]
```

## 5. 注意事项

| 注意点 | 说明 |
|--------|------|
| 不要修改正在遍历的列表 | 遍历时增删元素会导致跳过或重复，用副本 `for x in lst[:]` |
| `range()` 是左闭右开 | `range(5)` 是 0-4，不包含 5 |
| 推导式不要写太长 | 超过两行就该用普通 for 循环，可读性优先 |
| `while` 要确保终止条件 | 忘了更新变量就是死循环 |
| 生成器只能遍历一次 | 消费完后再次遍历会是空的 |
| 嵌套推导式从左到右读 | `[x for row in matrix for x in row]`，外层循环在前 |
| `for/else` 的 else 含义 | else 是"没有被 break 打断"时执行，不是"循环条件不满足" |

## 6. 避坑指南

| ❌ 错误写法 | ✅ 正确写法 | 原因 |
|------------|-----------|------|
| `for i in range(len(lst)): lst[i]` | `for item in lst:` | Pythonic 方式是直接遍历 |
| `for i in range(len(lst)): print(i, lst[i])` | `for i, v in enumerate(lst):` | 用 `enumerate` 获取索引 |
| `[[], [], []] * 3` 创建二维数组 | `[[] for _ in range(3)]` | `*` 是浅拷贝，修改一个会影响所有 |
| 推导式里写复杂逻辑 | 简单推导式 + 函数调用 | 推导式超过一行就不可读了 |
| `sum([x**2 for x in range(N)])` | `sum(x**2 for x in range(N))` | 生成器表达式更省内存 |
| 遍历时删除列表元素 | 用列表推导式过滤 | `[x for x in lst if x != target]` |
| `for i in range(10000000)` 处理大文件 | 逐行读取或生成器 | 避免一次性加载到内存 |

    lo, hi = 0, len(lst) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if lst[mid] == target:
            return mid
        elif lst[mid] < target:

        else:

│   ├── 遍历: for item in iterable
│   ├── range(start, stop, step)
│   ├── enumerate() → (index, value)
│   ├── zip() → 并行遍历
│   ├── dict.items() → (key, value)
│   └── itertools 模块
├── while 循环
│   ├── while condition: ...
│   ├── while True + break
│   └── 适合不确定迭代次数的场景
├── 流程控制
│   ├── break → 跳出整个循环
│   ├── continue → 跳过本轮
│   └── else → 未被 break 时执行
├── 推导式
│   ├── 列表 [expr for x in iter if cond]
│   ├── 字典 {k:v for ...}
│   ├── 集合 {expr for ...}
│   └── 嵌套 [x for a in A for x in a]
├── 生成器表达式
│   ├── (expr for x in iter)
│   └── 惰性求值，省内存
└── 性能
    ├── 推导式 > for 循环 > map/filter
    ├── 生成器 > 列表（大数据量）
    └── 避免在推导式中调用昂贵函数
```

## 9. 举一反三

| 场景 | 推荐方式 | 示例 |
|------|---------|------|
| 批量转换数据 | 列表推导式 | `[x.strip() for x in lines]` |
| 过滤数据 | 带条件推导式 | `[x for x in data if x > 0]` |
| 同时需要索引和值 | `enumerate()` | `for i, v in enumerate(lst):` |
| 并行遍历多个序列 | `zip()` | `for a, b in zip(list1, list2):` |
| 大数据求和/统计 | 生成器表达式 | `sum(x for x in data)` |
| 统计频次 | `collections.Counter` | `Counter(iterable)` |
| 展平嵌套结构 | 嵌套推导式 | `[x for row in matrix for x in row]` |
| 字典转换 | 字典推导式 | `{k: v.upper() for k, v in d.items()}` |

squares = [x ** 2 for x in range(10)]

# Filter with condition
even_squares = [x ** 2 for x in range(10) if x % 2 == 0]

```

一行搞定，可读性反而更强。

### v3 — 生成器 + 惰性管道（实战）

```python
def process_data_pipeline(data_source):
    """A lazy data processing pipeline using generator expressions."""
    
    # Stage 1: filter (lazy)
    valid = (item for item in data_source if item is not None)
    
    # Stage 2: transform (lazy)
    normalized = (item.strip().lower() for item in valid if isinstance(item, str))
    
    # Stage 3: collect results (only this triggers evaluation)
    return {
        word: count 
        for word, count in __import__("collections").Counter(normalized).items()
        if count > 1
    }

# Works with any iterable, including infinite ones
sample = ["Hello", " world ", None, "hello", "WORLD", "hello"]
result = process_data_pipeline(sample)
print(result)  # {'hello': 3, 'world': 2}
```

v3 版本用生成器表达式构建惰性管道，每一步都是延迟求值，只在最终收集时才触发计算——处理大数据集时内存友好，代码还优雅。
