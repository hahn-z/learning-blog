📖 概念讲解

什么是变量？

变量是存储数据的容器。你可以把它想象成一个贴了标签的盒子：

标签：name     标签：age     标签：is_student
 ┌──────────┐  ┌──────────┐  ┌──────────┐
 │ "张三"   │  │   25     │  │  True    │
 └──────────┘  └──────────┘  └──────────┘
  string         int           bool

Python是动态类型语言，不需要提前声明变量类型，解释器会自动推断：

name = "张三"       # Python 知道这是 str
age = 25            # Python 知道这是 int

Python的6种基本数据类型

类型 关键字 示例 说明
整数 int 42, -7, 0 任意大小，没有溢出
浮点数 float 3.14, -0.5, 1e3 64位双精度
字符串 str "hello", '世界' 不可变序列
布尔值 bool True, False 逻辑值
空值 NoneType None 表示"没有值"
复数 complex 3+4j 科学计算用

动态类型 vs 静态类型

# Python: Dynamic typing
x = 42        # x is int
x = "hello"   # Now x is str — totally fine!

# Compare with Java (static typing):
# int x = 42;
# x = "hello";  // ERROR! Type mismatch

动态类型的优势是灵活，劣势是IDE提示和错误检查不如静态语言。Python 3.5+ 引入了类型提示来弥补：

def greet(name: str) -> str:
    return f"Hello, {name}"

数值类型详解

整数 (int)： Python的int没有大小限制，不会溢出

big = 10 ** 100  # A googol! Works fine in Python

浮点数 (float)： 64位双精度，注意精度问题

0.1 + 0.2  # = 0.30000000000000004, not 0.3!

布尔值 (bool)： True本质是1，False本质是0

True + True   # = 2
False + 5     # = 5

数值类型操作

# numbers.py - Numeric types in depth

# Integer operations
a = 17
b = 5

print(f"a + b = {a + b}")     # 22 - Addition
print(f"a - b = {a - b}")     # 12 - Subtraction
print(f"a * b = {a * b}")     # 85 - Multiplication
print(f"a / b = {a / b}")     # 3.4 - True division (returns float)
print(f"a // b = {a // b}")   # 3  - Floor division
print(f"a % b = {a % b}")     # 2  - Modulo (remainder)
print(f"a ** b = {a ** b}")   # 1419857 - Exponent

# Integer has no limit
big_number = 2 ** 1000
print(f"\n2^1000 has {len(str(big_number))} digits")

# Float precision issue
print(f"\n0.1 + 0.2 = {0.1 + 0.2}")         # 0.30000000000000004
print(f"0.1 + 0.2 == 0.3 = {0.1 + 0.2 == 0.3}")  # False!

# Solution: use round() or Decimal
print(f"round(0.1 + 0.2, 1) = {round(0.1 + 0.2, 1)}")  # 0.3

# Boolean is a subclass of int
print(f"\nTrue + True = {True + True}")     # 2
print(f"False + 5 = {False + 5}")           # 5
print(f"isinstance(True, int) = {isinstance(True, int)}")  # True

类型检查与转换

# type_conversion.py - Type checking and conversion

# Check types
x = 42
print(f"type(x) = {type(x)}")                        # <class 'int'>
print(f"isinstance(x, int) = {isinstance(x, int)}")  # True
print(f"isinstance(x, (int, float)) = {isinstance(x, (int, float))}")  # True

# Type conversion
# String to number
s = "123"
n = int(s)
f = float("3.14")
print(f"int('123') = {n}, type: {type(n)}")
print(f"float('3.14') = {f}, type: {type(f)}")

# Number to string
print(f"str(42) = '{str(42)}'")
print(f"str(3.14) = '{str(3.14)}'")

# Float to int (truncates, not rounds!)
print(f"int(3.9) = {int(3.9)}")    # 3, not 4!
print(f"int(-3.9) = {int(-3.9)}")  # -3, not -4!

# Boolean conversion
print(f"\nbool(1) = {bool(1)}")       # True
print(f"bool(0) = {bool(0)}")       # False
print(f"bool('') = {bool('')}")      # False
print(f"bool('hello') = {bool('hello')}")  # True
print(f"bool([]) = {bool([])}")      # False
print(f"bool([1]) = {bool([1])}")     # True
print(f"bool(None) = {bool(None)}")   # False

实用示例：单位转换器

# converter.py - Practical type conversion example

def celsius_to_fahrenheit(celsius: float) -> float:
    # Convert Celsius to Fahrenheit
    return celsius * 9 / 5 + 32

def fahrenheit_to_celsius(fahrenheit: float) -> float:
    # Convert Fahrenheit to Celsius
    return (fahrenheit - 32) * 5 / 9

# Get user input and convert
temp_str = input("Enter temperature: ")
unit = input("Unit (C/F): ").upper()

try:
    temp = float(temp_str)

    if unit == "C":
        result = celsius_to_fahrenheit(temp)
        print(f"{temp}°C = {result:.1f}°F")
    elif unit == "F":
        result = fahrenheit_to_celsius(temp)
        print(f"{temp}°F = {result:.1f}°C")
    else:
        print("Please enter C or F")
except ValueError:
    print(f"'{temp_str}' is not a valid number")

项目 说明 建议
变量命名 区分大小写，Name ≠ name 使用 snake_case 风格
关键字冲突 class = 5 会报错 避免使用Python关键字

变量命名规范

# Good naming
user_name = "Alice"
max_count = 100
is_valid = True
PI = 3.14159        # Constants use UPPER_CASE

# Bad naming
x = "Alice"          # Meaningless
userName = "Alice"   # Not Python style (that's camelCase)
2nd_place = "Bob"    # Can't start with number
class = "math"       # Keyword conflict

🚫 避坑指南

❌ 浮点数直接比较

if 0.1 + 0.2 == 0.3:
    print("equal")  # Never executes!

✅ 正确做法： 使用 round() 或 math.isclose()

import math
if math.isclose(0.1 + 0.2, 0.3):
    print("equal")  # Works!

if round(0.1 + 0.2, 10) == round(0.3, 10):
    print("equal")  # Also works

❌ 用 == 判断 None

x = None
if x == None:  # Works but not recommended
    print("is None")

✅ 正确做法： 使用 is

if x is None:   # Pythonic way
    print("is None")

❌ 忘记转换 input() 的类型

age = input("Age: ")  # Always returns str!
# age * 2 would be "2525" if input was "25", not 50

✅ 正确做法： 显式转换

age = int(input("Age: "))
double = age * 2  # Now it's 50

❌ 修改变量时忘记重新赋值

x = 10
x + 5   # x is still 10, result discarded!
x = x + 5  # Now x is 15

✅ 正确做法： 使用增强赋值运算符

x = 10
x += 5  # x is now 15

❌ 链式比较的误解

x = 5
# This is NOT (x > 3) or (x > 10)
# It IS 3 < x and x < 10
result = 3 < x < 10  # True — Python supports chained comparison!

🌳 知识点总结

变量与数据类型
├── 变量
│   ├── 赋值：=, +=, -=, *=, /=
│   ├── 命名：snake_case, UPPER_CASE
│   ├── 动态类型：type() 查看类型
│   └── 多重赋值：a, b = b, a
├── int（整数）
│   ├── 无大小限制
│   ├── 进制：0b二进制, 0o八进制, 0x十六进制
│   └── 运算：+ - * / // % **
├── float（浮点数）
│   ├── 64位双精度
│   ├── 精度问题：0.1 + 0.2 ≠ 0.3
│   └── 解决：round() 或 decimal 模块
├── bool（布尔值）
│   ├── True = 1, False = 0
│   ├── Falsy: 0, 0.0, "", [], {}, None, False
│   └── Truthy: everything else
├── None（空值）
│   ├── 表示"没有值"
│   ├── 用 is 判断，不用 ==
│   └── 函数默认返回 None
├── 类型操作
│   ├── type(obj) → 获取类型
│   ├── isinstance(obj, type) → 类型判断
│   └── int(), float(), str(), bool() → 类型转换
└── 进阶
    ├── Type Hints: x: int = 42
    └── 可变性：int/float/str/bool 不可变
