# 函数定义与参数

> **难度标注：** ⭐⭐ 初级
>
> 适合已掌握基本语法和流程控制，准备学习函数封装的读者。

## 1. 概念讲解

函数是代码复用的基本单位。Python 的函数系统极其灵活——支持多种参数传递方式、默认值、可变参数、关键字参数，甚至可以把函数当参数传递。

**核心概念：**

- **def 定义函数**：用 `def` 关键字定义命名函数，`lambda` 定义匿名函数
- **参数类型**：位置参数、默认参数、可变位置参数（`*args`）、可变关键字参数（`**kwargs`）
- **仅关键字参数**（keyword-only）：`*` 之后的参数必须用关键字传递
- **仅位置参数**（positional-only，3.8+）：`/` 之前的参数只能按位置传递
- **作用域**：LEGB 规则（Local → Enclosing → Global → Built-in）
- **高阶函数**：函数可以作为参数传递、作为返回值

**为什么函数重要？**

没有函数的代码就是一团面条。函数让代码模块化、可复用、可测试。好的函数设计是软件工程的基础——一个函数只做一件事，名字说清楚它做什么，参数设计合理，返回值明确。

## 3. 完整代码示例

```python
"""
Python 函数定义与参数 - 完整示例
Covering: def, lambda, args/kwargs, scope, higher-order functions
"""

# --- 1. Basic function definition ---
def greet(name: str, greeting: str = "Hello") -> str:
    """Return a greeting message.

    Args:
        name: The person's name.
        greeting: The greeting word. Defaults to "Hello".

    Returns:
        A formatted greeting string.
    """
    return f"{greeting}, {name}!"

print(greet("Alice"))              # Hello, Alice!
print(greet("Bob", greeting="Hi")) # Hi, Bob!


# --- 2. All parameter types in one function ---
def demo_params(a, b, /, c, d=10, *args, e, f=20, **kwargs):
    """
    a, b: positional-only (before /)
    c, d: normal positional (d has default)
    *args: variable positional
    e, f: keyword-only (after *args), f has default
    **kwargs: variable keyword
    """
    print(f"a={a}, b={b}, c={c}, d={d}")
    print(f"args={args}")
    print(f"e={e}, f={f}")
    print(f"kwargs={kwargs}")

# demo_params(1, 2, 3, 4, 5, 6, e=7, f=8, g=9, h=10)
# a=1, b=2, c=3, d=4
# args=(5, 6)
# e=7, f=8
# kwargs={'g': 9, 'h': 10}


# --- 3. *args and **kwargs ---
# *args: collect extra positional args into a tuple
def total(*numbers):
    return sum(numbers)

print(total(1, 2, 3, 4, 5))  # 15

# **kwargs: collect extra keyword args into a dict
def build_profile(**info):
    return info

profile = build_profile(name="Alice", age=30, city="Beijing")
print(profile)  # {'name': 'Alice', 'age': 30, 'city': 'Beijing'}

# Unpacking with * and **
def add(a, b, c):
    return a + b + c

nums = [1, 2, 3]
print(add(*nums))  # 6

config = {"a": 10, "b": 20, "c": 30}
print(add(**config))  # 60


# --- 4. Default parameter trap ---
# WRONG: mutable default
def append_to(item, target=[]):
    target.append(item)
    return target

print(append_to(1))  # [1]
print(append_to(2))  # [1, 2]  <- Bug! Default list is shared

# CORRECT: use None as default
def append_to_fixed(item, target=None):
    if target is None:
        target = []
    target.append(item)
    return target

print(append_to_fixed(1))  # [1]
print(append_to_fixed(2))  # [2]  <- Fixed!


# --- 5. Multiple return values ---
def min_max(numbers):
    """Return both min and max of a sequence."""
    return min(numbers), max(numbers)

lo, hi = min_max([3, 1, 4, 1, 5, 9])
print(f"Min={lo}, Max={hi}")  # Min=1, Max=9


# --- 6. Lambda expressions ---
# Simple anonymous functions
square = lambda x: x ** 2
print(square(5))  # 25

# Common use: sort with key
students = [("Alice", 85), ("Bob", 92), ("Charlie", 78)]
students_sorted = sorted(students, key=lambda s: s[1], reverse=True)
print(students_sorted)  # [('Bob', 92), ('Alice', 85), ('Charlie', 78)]

# Lambda limitation: single expression only
# For complex logic, always use def


# --- 7. Scope (LEGB Rule) ---
x = "global"

def outer():
    x = "enclosing"
    
    def inner():
        x = "local"
        print(f"inner: {x}")  # local
    
    inner()
    print(f"outer: {x}")  # enclosing

outer()
print(f"global: {x}")  # global

# Using nonlocal and global
counter = 0

def make_counter():
    count = 0
    
    def increment():
        nonlocal count
        count += 1
        return count
    
    return increment

c = make_counter()
print(c())  # 1
print(c())  # 2
print(c())  # 3


# --- 8. Higher-order functions ---
# Function as argument
def apply_twice(func, value):
    return func(func(value))

print(apply_twice(lambda x: x + 1, 5))  # 7

# map, filter, reduce
numbers = [1, 2, 3, 4, 5]

# map: transform each element
doubled = list(map(lambda x: x * 2, numbers))
print(doubled)  # [2, 4, 6, 8, 10]

# filter: keep elements that match condition
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)  # [2, 4]

# reduce: accumulate into single value
from functools import reduce
product = reduce(lambda a, b: a * b, numbers)
print(product)  # 120

# But comprehensions are usually more Pythonic
doubled_v2 = [x * 2 for x in numbers]      # same as map
evens_v2 = [x for x in numbers if x % 2 == 0]  # same as filter


# --- 9. Practical: flexible config builder ---
def create_api_client(
    base_url: str,
    *,
    timeout: int = 30,
    retries: int = 3,
    api_key: str = None,
    **extra_options
) -> dict:
    """Create an API client configuration.

    Keyword-only arguments after * ensure clarity at call site.
    """
    config = {
        "base_url": base_url,
        "timeout": timeout,
        "retries": retries,
    }
    if api_key:
        config["auth"] = f"Bearer {api_key}"
    config.update(extra_options)
    return config

client = create_api_client(
    "https://api.example.com",
    timeout=60,
    api_key="secret123",
    debug=True  # extra option
)
print(client)
# {'base_url': 'https://api.example.com', 'timeout': 60, 'retries': 3,
#  'auth': 'Bearer secret123', 'debug': True}


# --- 10. Closures and partial application ---
from functools import partial

def power(base, exponent):
    return base ** exponent

# Create specialized functions
square_fn = partial(power, exponent=2)
cube_fn = partial(power, exponent=3)

print(square_fn(5))  # 25
print(cube_fn(3))    # 27
```

## 5. 注意事项

| 注意点 | 说明 |
|--------|------|
| 默认参数只计算一次 | 可变默认值（列表、字典）会在调用间共享，这是经典 Bug |
| `*args` 是元组 | 收集的位置参数是 tuple，不可修改 |
| `**kwargs` 是字典 | 收集的关键字参数是 dict |
| `lambda` 只能包含表达式 | 不能有赋值、if 语句等（只能用条件表达式） |
| `nonlocal` vs `global` | `nonlocal` 修改外层函数变量，`global` 修改模块级变量 |
| 函数是一等公民 | 可以赋值给变量、作为参数、作为返回值 |
| 类型注解不影响运行 | `def f(x: int) -> str:` 只是提示，运行时不强制 |

## 6. 避坑指南

| ❌ 错误写法 | ✅ 正确写法 | 原因 |
|------------|-----------|------|
| `def f(x=[])` | `def f(x=None): if x is None: x=[]` | 可变默认值在调用间共享 |
| `def f(a, b=[]):` | `def f(a, b=None):` | 同上，列表/字典/集合都中招 |
| `lambda x: if x > 0: x else 0` | `lambda x: x if x > 0 else 0` | lambda 只能用表达式 |
| 在循环中创建 lambda | 用 `partial` 或默认参数绑定 | 闭包捕获变量引用，不是值 |
| `global` 到处用 | 重构代码，用类或传参 | `global` 让代码难以测试和调试 |
| 10 个参数的函数 | 拆分成多个函数或用配置对象 | 参数太多说明函数职责不清 |
| `def f(*args): print(args[0])` | `def f(first, *args):` | 命名参数比索引更清晰 |

### 🟡 中级

# Q1
def is_palindrome(s):
    cleaned = s.lower().replace(" ", "")
    return cleaned == cleaned[::-1]

print(is_palindrome("race car"))  # True
print(is_palindrome("hello"))     # False

# Q2
def multiply_all(*numbers):
    result = 1
    for n in numbers:
        result *= n
    return result

# Q3
def make_multiplier(n):
    def multiplier(x):
        return x * n
    return multiplier

double = make_multiplier(2)
triple = make_multiplier(3)
print(double(5))   # 10
print(triple(5))   # 15

# Q4
def safe_get(data, *keys, default=None):
    current = data
    for key in keys:
        if isinstance(current, dict) and key in current:
            current = current[key]
        else:
            return default
    return current

print(safe_get({"a": {"b": {"c": 42}}}, "a", "b", "c"))  # 42
print(safe_get({"a": 1}, "x", "y", default="N/A"))        # N/A

# Q5
def memoize(func):
    cache = {}
    def wrapper(*args):
        if args not in cache:
            cache[args] = func(*args)
        return cache[args]
    return wrapper

@memoize
def fib(n):
    if n < 2:
        return n
    return fib(n - 1) + fib(n - 2)

print(fib(100))  # 354224848179261915075 (instant)

# Q6
import functools

def type_safe(func):
    hints = func.__annotations__
    sig = func.__code__.co_varnames[:func.__code__.co_argcount]
    
    @functools.wraps(func)
    def wrapper(*args, **kwargs):
        for name, val in zip(sig, args):
            if name in hints and not isinstance(val, hints[name]):
                raise TypeError(
                    f"{name}: expected {hints[name].__name__}, "
                    f"got {type(val).__name__}"
                )
        return func(*args, **kwargs)
    return wrapper

@type_safe
def add(a: int, b: int) -> int:
    return a + b

print(add(1, 2))   # 3
# add("a", 1)      # TypeError: a: expected int, got str

## 9. 举一反三

| 场景 | 推荐方式 | 示例 |
|------|---------|------|
| 不确定参数数量 | `*args` / `**kwargs` | `def log(*args): print(*args)` |
| 配置项很多 | keyword-only + 默认值 | `def f(*, timeout=30, retry=3)` |
| 需要记住状态 | 闭包或类 | `def make_counter(): ...` |
| 缓存计算结果 | `@functools.lru_cache` | `@lru_cache(maxsize=128)` |
| 通用包装/增强 | 装饰器 | `@timer`, `@retry`, `@auth` |
| 安全取嵌套值 | 递归 + `**kwargs` | `safe_get(data, "a", "b")` |
| 简单回调/排序 | `lambda` | `sorted(lst, key=lambda x: x[1])` |
| 批量数据处理 | `map()` 或推导式 | `[f(x) for x in data]` |

### v2 — 灵活参数 + 类型注解（进阶）

```python
from typing import Optional

def calculate(
    *values: float,
    operation: str = "sum",
    precision: int = 2,
    ignore_errors: bool = False,
) -> Optional[float]:
    """Perform calculation on values with configurable options."""
    if not values:
        return None
    
    nums = list(values)
    
    if operation == "sum":
        result = sum(nums)
    elif operation == "avg":
        result = sum(nums) / len(nums)
    elif operation == "max":
        result = max(nums)
    elif operation == "min":
        result = min(nums)
    else:
        raise ValueError(f"Unknown operation: {operation}")
    
    return round(result, precision)

print(calculate(1, 2, 3, 4, 5))                     # 15.0
print(calculate(1, 2, 3, 4, 5, operation="avg"))    # 3.0
print(calculate(10, 20, 30, operation="max"))        # 30.0
```

v2 版本加入了 `*args`、keyword-only 参数、类型注解、文档字符串——函数签名本身就是 API 文档。

### v3 — 装饰器 + 闭包 + 高阶函数（实战）

```python
import functools
import time
from typing import Callable, TypeVar, Any

T = TypeVar("T")

def retry(max_attempts: int = 3, delay: float = 1.0):
    """Decorator: retry a function on failure."""
    def decorator(func: Callable) -> Callable:
        @functools.wraps(func)
        def wrapper(*args, **kwargs) -> Any:
            last_error = None
            for attempt in range(1, max_attempts + 1):
                try:
                    return func(*args, **kwargs)
                except Exception as e:
                    last_error = e
                    print(f"Attempt {attempt}/{max_attempts} failed: {e}")
                    if attempt < max_attempts:
                        time.sleep(delay)
            raise last_error
        return wrapper
    return decorator

def log_calls(func: Callable) -> Callable:
    """Decorator: log function calls with timing."""
    @functools.wraps(func)
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        elapsed = time.time() - start
        print(f"[{func.__name__}] args={args} kwargs={kwargs} -> {result} ({elapsed:.4f}s)")
        return result
    return wrapper

@retry(max_attempts=3, delay=0.5)
@log_calls
def fetch_data(url: str, *, timeout: int = 30) -> dict:
    """Simulate fetching data from an API."""
    import random
    if random.random() < 0.5:  # 50% chance of failure
        raise ConnectionError(f"Failed to connect to {url}")
    return {"url": url, "status": "ok", "timeout": timeout}

# Usage - will auto-retry on failure
result = fetch_data("https://api.example.com/data", timeout=60)
print(result)
```

v3 版本展示了 Python 函数系统的真正威力：装饰器工厂（带参数的装饰器）、`functools.wraps` 保留元信息、高阶函数组合、闭包捕获状态——这是生产级 Python 代码的常见模式。
