字符串的本质

Python中，字符串是不可变的Unicode字符序列。三个关键点：

  1. 不可变：创建后不能修改，每次"修改"实际是创建新字符串
    1. 序列：有序、可迭代、可通过索引访问
      1. Unicode：原生支持中文、emoji等所有字符
        2. s = "Hello"
# s[0] = "h"  # ERROR! Strings are immutable
s = "hello"    # This creates a NEW string, doesn't modify the old one

        字符串的四种创建方式

        # Single quotes
s1 = 'Hello'

# Double quotes (identical to single)
s2 = "Hello"

# Triple quotes (multi-line)
s3 = """Line 1
Line 2
Line 3"""

# Raw strings (no escape processing)
s4 = r"C:\new\test"  # \n and \t are literal, not escape sequences

        单引号和双引号完全等价，选择标准是：字符串内容包含哪种引号就用另一种。

        字符串格式化演进

        方式 语法 Python版本 推荐度
        % 格式化 "Hello %s" % name 所有版本 ⭐⭐
        .format() "Hello {}".format(name) 2.6+ ⭐⭐⭐
        f-string f"Hello {name}" 3.6+ ⭐⭐⭐⭐⭐

        f-string 是现代Python的标准写法，性能最好、语法最简洁。

        切片（Slicing）

语法: s[start:stop:step]

        索引与切片

        # slicing.py - String indexing and slicing

s = "Python"

# Indexing
print(f"s[0] = '{s[0]}'")    # P
print(f"s[-1] = '{s[-1]}'")  # n
print(f"s[3] = '{s[3]}'")    # h

# Basic slicing: s[start:stop] (stop is exclusive)
print(f"s[0:3] = '{s[0:3]}'")   # Pyt
print(f"s[2:5] = '{s[2:5]}'")   # tho
print(f"s[:3] = '{s[:3]}'")     # Pyt (omit start = 0)
print(f"s[3:] = '{s[3:]}'")     # hon (omit end = len)

# Negative index slicing
print(f"s[-3:] = '{s[-3:]}'")   # hon (last 3 chars)
print(f"s[:-2] = '{s[:-2]}'")   # Pyth (all but last 2)

# Step slicing: s[start:stop:step]
print(f"s[::2] = '{s[::2]}'")     # Pto (every 2nd char)
print(f"s[1::2] = '{s[1::2]}'")   # yhn (every 2nd, starting from index 1)
print(f"s[::-1] = '{s[::-1]}'")   # nohtyP (reverse!)
print(f"s[::-2] = '{s[::-2]}'")   # nhy (reverse every 2nd)

# Practical examples
email = "user@example.com"
domain = email.split("@")[1]  # example.com
username = email.split("@")[0]  # user
print(f"\nUsername: {username}, Domain: {domain}")

# Reverse a string
text = "ABBA"
print(f"'{text}' reversed: '{text[::-1]}'")
print(f"Is palindrome: {text == text[::-1]}")  # True

        实用字符串处理示例

        # practical_strings.py - Real-world string processing

def clean_user_input(text: str) -> str:
    # Clean and normalize user input
    return text.strip().lower()

def format_phone(phone: str) -> str:
    # Format phone number: 13812345678 -> 138-1234-5678
    phone = phone.replace("-", "").replace(" ", "")
    if len(phone) == 11 and phone.isdigit():
        return f"{phone[:3]}-{phone[3:7]}-{phone[7:]}"
    return phone

def mask_email(email: str) -> str:
    # Mask email: alice@example.com -> a***e@example.com
    if "@" not in email:
        return email
    user, domain = email.split("@")
    if len(user) <= 2:
        masked = user[0] + "***"
    else:
        masked = user[0] + "***" + user[-1]
    return f"{masked}@{domain}"

def truncate(text: str, max_len: int = 20, suffix: str = "...") -> str:
    # Truncate text with ellipsis
    if len(text) <= max_len:
        return text
    return text[:max_len - len(suffix)] + suffix

# Demo
print("--- User Input Cleaning ---")
inputs = ["  Hello  ", "WORLD", "  Python  "]
for raw in inputs:
    print(f"'{raw}' -> '{clean_user_input(raw)}'")

print("\n--- Phone Formatting ---")
phones = ["13812345678", "138-1234-5678", "123"]
for p in phones:
    print(f"{p} -> {format_phone(p)}")

print("\n--- Email Masking ---")
emails = ["alice@example.com", "bob@test.org", "x@y.com"]
for e in emails:
    print(f"{e} -> {mask_email(e)}")

print("\n--- Text Truncation ---")
texts = ["Hello World", "This is a very long sentence that needs truncation"]
for t in texts:
    print(f"'{t}' -> '{truncate(t, 20)}'")

        ⚠️ 注意事项

        项目 说明 建议
        不可变性 字符串操作返回新字符串 s = s.upper() 而不是 s.upper()
        编码 Python 3默认UTF-8 涉及文件读写时指定 encoding='utf-8'
        原始字符串 r"" 不处理转义 正则表达式和Windows路径用 r""
        join vs + + 拼接大量字符串效率低 批量拼接用 "".join(list)
        index vs find index() 找不到会抛异常 不确定是否存在时用 find()
        切片越界 不会报错，返回空串 s[100:] 返回 "" 而非报错
        中文长度 一个中文算一个字符 len("你好") = 2

        🚫 避坑指南

        ❌ 拼接大量字符串用 +

        # Inefficient! Creates a new string each iteration
result = ""
for i in range(10000):
    result += str(i)  # O(n^2) time complexity

        ✅ 正确做法： 使用 join()

        # Efficient! O(n) time complexity
parts = [str(i) for i in range(10000)]
result = "".join(parts)

        ❌ 忘记字符串不可变

        s = "hello"
s.upper()  # Returns "HELLO" but s is still "hello"!
print(s)   # hello

        ✅ 正确做法： 重新赋值

        s = "hello"
s = s.upper()  # s is now "HELLO"

        ❌ 判断包含用 find() >= 0

        if s.find("hello") >= 0:  # Unnecessary
    print("found")

        ✅ 正确做法： 使用 in 运算符

        if "hello" in s:  # Pythonic!
    print("found")

        ❌ 格式化浮点数用字符串截断

        pi_str = str(3.14159)[:4]  # "3.14" — fragile and wrong for larger numbers

        ✅ 正确做法： 使用格式化

        pi_str = f"{3.14159:.2f}"  # "3.14" — correct and clear

        ❌ Windows路径转义问题

        path = "C:\new\test"  # \n and \t are escape sequences!
print(path)  # C:
# ew    est

        ✅ 正确做法： 使用原始字符串

        path = r"C:\new\test"  # Raw string, no escape
# Or use forward slashes (works on Windows too!)
path = "C:/new/test"

              v3：完整的文本处理器

              # v3: Text processor with multiple utilities
import re
from collections import Counter

class TextProcessor:
    # A simple text processing toolkit

    def __init__(self, text: str):
        self.text = text

    def word_count(self) -> int:
        return len(self.text.split())

    def char_count(self, ignore_spaces: bool = True) -> int:
        if ignore_spaces:
            return len(self.text.replace(" ", ""))
        return len(self.text)

    def top_chars(self, n: int = 5) -> list:
        chars = [c.lower() for c in self.text if c.isalpha()]
        return Counter(chars).most_common(n)

    def reverse_words(self) -> str:
        return " ".join(self.text.split()[::-1])

    def extract_numbers(self) -> list:
        return [int(n) for n in re.findall(r'\d+', self.text)]

    def summary(self) -> str:
        lines = [
            f"Characters: {self.char_count()}",
            f"Words: {self.word_count()}",
            f"Top chars: {self.top_chars(3)}",
            f"Reversed: {self.reverse_words()}",
        ]
        return "\n".join(lines)

# Usage
tp = TextProcessor("Hello World 123 Python 456")
print(tp.summary())
