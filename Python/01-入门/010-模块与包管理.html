一、概念讲解

什么是模块（Module）？

模块就是一个 .py 文件。没错，就这么简单。你写的每一个 Python 文件，都是一个模块。

# my_module.py
def greet(name):
    return f"Hello, {name}!"

PI = 3.14159

当你在另一个文件里 import my_module，就能用 my_module.greet("Alice") 了。

什么是包（Package）？

包就是一个文件夹，里面有个 __init__.py 文件（可以为空）。文件夹里放多个模块，就组成了一个包。

my_package/
├── __init__.py
├── math_tools.py
└── string_tools.py

为什么要用模块和包？

理由 说明
代码复用 写一次，到处 import
命名空间隔离 避免变量名冲突
可维护性 代码分门别类，好找好改
团队协作 各人负责不同模块，互不干扰
标准库扩展 Python 自带 200+ 模块，按需加载

三、完整代码示例

3.1 基础导入方式

# === Basic import styles ===

# Style 1: Import entire module
import math
print(math.sqrt(16))       # 4.0
print(math.pi)             # 3.141592653589793

# Style 2: Import specific items
from math import sqrt, pi
print(sqrt(25))            # 5.0
print(pi)                  # 3.141592653589793

# Style 3: Import with alias
import datetime as dt
now = dt.datetime.now()
print(now.strftime("%Y-%m-%d %H:%M:%S"))

# Style 4: Import all (avoid in production!)
from math import *
print(pow(2, 10))          # 1024.0

3.2 自定义模块

# === File: string_utils.py ===

def truncate(text, max_length=50, suffix="..."):
    """Truncate text to max_length, appending suffix if needed."""
    if len(text) <= max_length:
        return text
    return text[:max_length - len(suffix)] + suffix

def count_words(text):
    """Count words in a text string."""
    return len(text.split())

def reverse_words(text):
    """Reverse the order of words in a text."""
    words = text.split()
    return " ".join(reversed(words))

# Only runs when executed directly, not when imported
if __name__ == "__main__":
    sample = "Python modules are powerful tools for code organization"
    print(truncate(sample, 30))    # Python modules are powerfu...
    print(count_words(sample))     # 8
    print(reverse_words(sample))   # organization code for tools...

3.3 自定义包

# === Package structure ===
# my_tools/
#   __init__.py
#   math_helpers.py
#   text_helpers.py

# --- my_tools/__init__.py ---
from .math_helpers import add, multiply
from .text_helpers import shout, whisper

# --- my_tools/math_helpers.py ---
def add(a, b):
    return a + b

def multiply(a, b):
    return a * b

# --- my_tools/text_helpers.py ---
def shout(text):
    return text.upper() + "!!!"

def whisper(text):
    return text.lower() + "..."

# === Usage ===
# from my_tools import add, shout
# print(add(3, 5))       # 8
# print(shout("hello"))  # HELLO!!!

3.5 相对导入与绝对导入

# === Absolute vs Relative imports ===

# Absolute import (recommended)
from my_tools.math_helpers import add

# Relative import (use inside packages)
from .math_helpers import add       # Same package
from ..sibling_pkg.module import fn  # Parent's sibling package

# __all__ controls what "from module import *" exports
__all__ = ["add", "multiply"]  # Explicitly exported names

五、注意事项

事项 说明 建议
__init__.py Python 3.3+ 可以省略（命名空间包） 建议始终保留，保持兼容
import * 污染命名空间，可能覆盖同名变量 生产代码禁止使用
循环导入 A 导入 B，B 又导入 A 重构代码，提取公共部分到 C
搜索路径 Python 按 sys.path 顺序查找模块 添加路径用 sys.path.insert(0, path)
缓存机制 __pycache__/ 存编译后的字节码 开发时修改模块可能需要重启
相对导入 只能在包内部使用 入口脚本用绝对导入

六、避坑指南

❌ 坑1：模块名与标准库冲突

# ❌ 文件命名为 random.py
import random
print(random.randint(1, 10))  # 可能导入自己而不是标准库！

# ✅ 使用有意义的模块名
import my_random_utils

❌ 坑2：循环导入

# ❌ a.py imports b.py, b.py imports a.py -> ImportError

# ✅ 延迟导入或提取公共模块
def func_a():
    from b import func_b  # Import inside function
    return func_b()

❌ 坑3：直接修改 sys.path

# ❌ 硬编码路径
sys.path.append("/home/user/my-project/src")

# ✅ 使用项目结构和相对路径
import os
sys.path.insert(0, os.path.dirname(os.path.abspath(__file__)))
