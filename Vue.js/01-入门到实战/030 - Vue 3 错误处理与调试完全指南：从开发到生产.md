## 概念讲解

### Vue 3 错误处理机制

Vue提供了多层错误处理机制，帮助开发者捕获和处理运行时错误。

```
错误处理层级
├── 全局错误处理 (app.config.errorHandler)
├── 错误边界 (onErrorCaptured)
├── 生命周期钩子错误
└── 组件渲染错误
```

### 调试工具

| 工具 | 用途 |
|------|------|
| Vue DevTools | 组件状态、事件、路由追踪 |
| Chrome DevTools | 性能分析、网络、存储 |
| Vite Debug Mode | 热更新、模块依赖可视化 |
| Source Maps | 生产环境错误定位 |

## 完整代码示例

```vue
<!-- ErrorDebugDemo.vue -->
<template>
  <div class="error-demo">
    <h2>错误处理与调试演示</h2>

    <section>
      <h3>1. 错误边界</h3>
      <button @click="triggerError">触发错误</button>
      <p v-if="error">{{ error.message }}</p>
    </section>

    <section>
      <h3>2. 错误监听</h3>
      <button @click="triggerAsyncError">触发异步错误</button>
      <p>捕获的错误: {{ asyncErrorCount }}</p>
    </section>

    <section>
      <h3>3. 开发环境调试</h3>
      <button @click="debugComponent">调试组件状态</button>
    </section>
  </div>
</template>

<script setup>
import { ref, onErrorCaptured } from 'vue'

const error = ref(null)
const asyncErrorCount = ref(0)

// Error boundary
onErrorCaptured((err, instance, info) => {
  console.error('Error captured:', err)
  console.error('Component:', instance)
  console.error('Info:', info)
  error.value = err
  return false  // Prevent error propagation
})

function triggerError() {
  throw new Error('手动触发的错误')
}

function triggerAsyncError() {
  setTimeout(() => {
    throw new Error('异步错误')
  }, 100)
}

function debugComponent() {
  console.log('Component state:', {
    error: error.value,
    asyncErrorCount: asyncErrorCount.value
  })
}

// Global error handler (main.js)
// app.config.errorHandler = (err, instance, info) => { ... }
</script>

<style scoped>
.error-demo { padding: 20px; }
section { margin: 16px 0; padding: 16px; border: 1px solid #eee; border-radius: 8px; }
button { margin: 4px; padding: 6px 14px; cursor: pointer; }
</style>
```

## 注意事项

| 机制 | 适用场景 | 注意点 |
|------|---------|--------|
| onErrorCaptured | 组件内错误捕获 | 只捕获后代组件错误 |
| errorHandler | 全局错误处理 | 未捕获的错误都会到达这里 |
| try/catch | 同步代码错误 | 不适用于组件渲染错误 |

## 避坑指南

### ❌ 错误处理中再次抛出错误
```javascript
onErrorCaptured((err) => {
  console.error(err)
  throw err  // ❌ 可能导致无限循环
})
```
```javascript
// ✅ 返回false阻止传播
onErrorCaptured((err) => {
  console.error(err)
  return false
})
```

```javascript

  console.error(err)
}
```
