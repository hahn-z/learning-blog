# 模板语法与数据绑定

> 📊 **难度：初级** | 🏷️ Vue.js 3.x | ⏱️ 阅读约 15 分钟

---

## 📖 概念讲解

Vue的模板语法是声明式地将DOM与底层数据绑定的方式。所有Vue模板都是合法的HTML，能被浏览器和HTML解析器解析。

**插值表达式：**

- `{{ msg }}` - 文本插值，最基本的数据绑定方式
- `{{ ok ? 'YES' : 'NO' }}` - 支持JavaScript表达式
- `{{ rawHtml }}` - 原始HTML需要用`v-html`指令

**指令系统：**

指令是带有`v-`前缀的特殊属性，其值预期是单个JavaScript表达式。

| 指令 | 用途 | 示例 |
|------|------|------|
| `v-bind` | 动态绑定属性 | `v-bind:src="imgUrl"` |
| `v-on` | 监听事件 | `v-on:click="handler"` |
| `v-model` | 双向绑定 | `v-model="name"` |
| `v-if` | 条件渲染 | `v-if="show"` |
| `v-for` | 列表渲染 | `v-for="item in list"` |
| `v-show` | 显示/隐藏 | `v-show="visible"` |
| `v-html` | 渲染HTML | `v-html="content"` |
| `v-text` | 更新文本 | `v-text="msg"` |

**缩写形式：**

- `v-bind:src` → `:src`
- `v-on:click` → `@click`
- `v-slot:name` → `#name`

**动态参数：** 可以用方括号包裹JavaScript表达式作为指令参数：
```html
<a :[attrName]="url"> ... </a>
<button @[eventName="handler"> ... </button>
```

**修饰符：** 以`.`指明的特殊后缀，指出指令应以特殊方式绑定：
- `.prevent` - 调用`event.preventDefault()`
- `.stop` - 调用`event.stopPropagation()`
- `.trim` - 自动过滤首尾空白字符
- `.number` - 自动将输入值转为数字
- `.lazy` - 在change事件后同步（而非input事件）


## 💻 完整代码

```vue
<!-- TemplateSyntaxDemo.vue -->
<template>
  <div class="demo">
    <!-- 1. Text interpolation -->
    <h2>1. Text Interpolation</h2>
    <p>Message: {{ message }}</p>
    <p>Computed: {{ message.toUpperCase() }}</p>
    <p>Ternary: {{ count > 5 ? 'Many' : 'Few' }}</p>

    <!-- 2. v-bind: dynamic attribute binding -->
    <h2>2. Attribute Binding (v-bind)</h2>
    <img :src="imageUrl" :alt="imageAlt" />
    <a :href="link.url" :target="link.target">{{ link.text }}</a>
    <!-- Dynamic class and style binding -->
    <div :class="{ active: isActive, 'text-danger': hasError }">
      Dynamic CSS classes
    </div>
    <div :style="{ color: textColor, fontSize: fontSize + 'px' }">
      Dynamic inline styles
    </div>

    <!-- 3. v-on: event handling -->
    <h2>3. Event Handling (v-on)</h2>
    <button @click="count++">Count: {{ count }}</button>
    <button @click="greet">Greet</button>
    <!-- With modifiers -->
    <form @submit.prevent="handleSubmit">
      <input v-model="inputText" />
      <button type="submit">Submit</button>
    </form>
    <a href="https://vuejs.org" @click.prevent="handleLinkClick">
      Prevented navigation
    </a>

    <!-- 4. v-model: two-way binding -->
    <h2>4. Two-Way Binding (v-model)</h2>
    <input v-model.trim="name" placeholder="Name (trimmed)" />
    <input v-model.number="age" type="number" placeholder="Age (number)" />
    <textarea v-model="bio" placeholder="Bio"></textarea>
    <select v-model="selected">
      <option value="">Please select</option>
      <option v-for="opt in options" :key="opt.value" :value="opt.value">
        {{ opt.label }}
      </option>
    </select>
    <p>Name: {{ name }}, Age: {{ age }}, Selected: {{ selected }}</p>

    <!-- 5. v-html: rendering raw HTML -->
    <h2>5. Raw HTML (v-html)</h2>
    <div v-html="richText"></div>
    <!-- WARNING: Never use v-html with user input! XSS risk -->

    <!-- 6. Dynamic arguments -->
    <h2>6. Dynamic Arguments</h2>
    <button @[eventName]="handleDynamic">Dynamic event: {{ eventName }}</button>
    <button @click="toggleEvent">Toggle event type</button>
  </div>
</template>

<script setup>
import { ref, reactive } from 'vue'

// Basic state
const message = ref('Hello Vue 3 Template!')
const count = ref(0)
const inputText = ref('')

// Attribute binding
const imageUrl = ref('https://picsum.photos/200/100')
const imageAlt = ref('Random image')
const isActive = ref(true)
const hasError = ref(false)
const textColor = ref('#42b883')
const fontSize = ref(16)
const link = reactive({
  url: 'https://vuejs.org',
  target: '_blank',
  text: 'Vue.js Official'
})

// Form state
const name = ref('')
const age = ref(null)
const bio = ref('')
const selected = ref('')
const options = [
  { value: 'vue', label: 'Vue.js' },
  { value: 'react', label: 'React' },
  { value: 'angular', label: 'Angular' }
]

// Raw HTML content
const richText = ref('<strong>Bold</strong> and <em>italic</em> text')

// Dynamic argument
const eventName = ref('click')

// Event handlers
function greet() {
  alert(`Hello! Count is ${count.value}`)
}
function handleSubmit() {
  console.log('Form submitted:', inputText.value)
}
function handleLinkClick() {
  alert('Navigation prevented by .prevent modifier')
}
function handleDynamic() {
  alert(`Dynamic event "${eventName.value}" triggered!`)
}
function toggleEvent() {
  eventName.value = eventName.value === 'click' ? 'mouseenter' : 'click'
}
</script>
```

## ⚠️ 注意事项

| 注意点 | 说明 | 建议 |
|--------|------|------|
| 模板表达式限制 | 只支持单行表达式，不能使用语句 | 复杂逻辑放computed中 |
| v-html安全风险 | 直接渲染HTML存在XSS风险 | 永远不要对用户输入用v-html |
| v-model本质 | 是v-bind + v-on的语法糖 | 理解原理有助于自定义组件 |
| 动态参数限制 | 表达式不能包含空格和引号 | 复杂参数用computed计算 |
| 修饰符顺序 | 修饰符顺序影响执行结果 | .stop应放在最前面 |
| class绑定 | 支持对象、数组和字符串 | 对象语法最常用 |

## 🚫 避坑指南

### 1. 在插值中使用语句
❌ **错误：** `{{ let a = 1; return a }}`（模板中不能用语句）
✅ **正确：** `{{ 1 }}` 或在computed中处理

### 2. v-html用于用户输入
❌ **错误：** `<div v-html="userInput" />`（XSS漏洞！）
✅ **正确：** 使用文本插值`{{ userInput }}`，或先做HTML转义

### 3. 忘记key属性
❌ **错误：** `<div v-for="item in list">{{ item }}</div>`
✅ **正确：** `<div v-for="item in list" :key="item.id">{{ item }}</div>`

### 4. v-for和v-if混用
❌ **错误：** `<div v-for="item in list" v-if="item.active">`
✅ **正确：** 先用computed过滤，再v-for渲染过滤后的列表

### 5. v-model在自定义组件上不生效
❌ **错误：** 子组件没有emit update:modelValue事件
✅ **正确：** 子组件内`emit('update:modelValue', newValue)`

## 📝 知识点总结

```
模板语法知识树
├── 插值
│   ├── {{ }} → 文本
│   ├── v-html → HTML
│   └── v-text → 文本（替换整个内容）
├── 指令
│   ├── v-bind (:attr) → 属性绑定
│   ├── v-on (@event) → 事件监听
│   ├── v-model → 双向绑定
│   ├── v-if → 条件渲染（销毁/创建）
│   ├── v-show → 显示隐藏（CSS切换）
│   └── v-for → 列表渲染
├── 缩写
│   ├── v-bind:attr → :attr
│   ├── v-on:event → @event
│   └── v-slot:name → #name
├── 修饰符
│   ├── .prevent → 阻止默认行为
│   ├── .stop → 阻止冒泡
│   ├── .trim → 去空格
│   ├── .number → 转数字
│   └── .lazy → 延迟更新
└── 动态参数
    └── :[expr]="value" → 动态属性名
```


## 🔄 举一反三

| 模板语法 | 生活类比 | 说明 |
|----------|---------|------|
| 插值 {{ }} | 模具填空 | 占位符被真实数据替换 |
| v-bind | 贴标签 | 动态给元素贴上不同属性标签 |
| v-model | 对讲机 | 双向通信，一边说另一边听到 |
| v-if | 开关 | 真正切断/接通电路（创建/销毁DOM） |
| v-show | 百叶窗 | 不拆除，只是拉上/拉开 |
| 修饰符 | 快捷键后缀 | Ctrl+C中C是修饰，.prevent是后缀 |
| 动态参数 | 变量变量名 | 名字本身也是动态的 |

## 🚀 代码演进

### v1 - 基础插值和属性绑定

```vue
<template>
  <h1>{{ title }}</h1>
  <img v-bind:src="imageUrl" />
</template>
<script>
export default {
  data() { return { title: 'Hello', imageUrl: '/logo.png' } }
}
</script>
```

### v2 - 使用缩写和指令

```vue
<template>
  <h1>{{ title }}</h1>
  <img :src="imageUrl" :alt="title" />
  <input v-model="title" @input="onInput" />
</template>
<script>
export default {
  data() { return { title: 'Hello', imageUrl: '/logo.png' } },
  methods: { onInput() { console.log(this.title) } }
}
</script>
```

### v3 - Composition API + 完整指令运用

```vue
<template>
  <h1>{{ title }}</h1>
  <img :src="imageUrl" :alt="title" />
  <input v-model.trim="title" />
  <div :class="{ active: isActive }" @click.prevent="toggle">
    Click me
  </div>
</template>
<script setup>
import { ref } from 'vue'
const title = ref('Hello')
const imageUrl = ref('/logo.png')
const isActive = ref(false)
function toggle() { isActive.value = !isActive.value }
</script>
```
