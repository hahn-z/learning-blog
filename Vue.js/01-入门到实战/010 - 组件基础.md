# 组件基础

> 📊 **难度：中级** | 🏷️ Vue.js 3.x | ⏱️ 阅读约 25 分钟

---

## 📖 概念讲解

组件是Vue应用的构建块。每个.vue文件就是一个组件，包含template、script和style三个部分。

**组件通信的三大机制：**

| 机制 | 方向 | 用途 |
|------|------|------|
| Props | 父→子 | 传递数据 |
| Events ($emit) | 子→父 | 子组件通知父组件 |
| Slots | 父→子 | 传递模板内容（HTML结构） |

**组件注册：**

- **全局注册**：`app.component('MyComp', ...)` — 所有地方直接用，但增加打包体积
- **局部注册**：在`<script setup>`中直接import — 按需引入，推荐方式

**Props验证：**

Props支持类型验证、默认值、必填等约束：
```js
defineProps({
  title: { type: String, required: true },
  count: { type: Number, default: 0 },
  items: { type: Array, default: () => [] },
  status: { type: String, validator: (v) => ['active', 'inactive'].includes(v) }
})
```

**defineEmits：**

子组件通过emit向父组件发送事件：
```js
const emit = defineEmits(['update', 'delete'])
emit('update', newValue)
```

**插槽(Slots)：**

- 默认插槽：`<slot></slot>` — 传入任意内容
- 具名插槽：`<slot name="header">` — 指定位置
- 作用域插槽：`<slot :data="item">` — 子组件向父组件传数据

## 💻 完整代码

```vue
<!-- ComponentBasicsDemo.vue - Parent Component -->
<template>
  <div class="demo">
    <h1>Component Communication Demo</h1>

    <!-- 1. Props: pass data to child -->
    <h2>Props Demo</h2>
    <UserCard
      name="Alice"
      :age="25"
      :skills="['Vue', 'React']"
      theme="dark"
      @toggle-theme="toggleTheme"
    >
      <!-- Default slot content -->
      <p>Custom description for Alice</p>
      <!-- Named slot -->
      <template #badge>⭐ Senior Dev</template>
    </UserCard>

    <!-- 2. Events: child notifies parent -->
    <h2>Events Demo</h2>
    <Counter :initial="0" @change="onCountChange" />
    <p>Last change: {{ lastChange }}</p>

    <!-- 3. Scoped Slots -->
    <h2>Scoped Slot Demo</h2>
    <DataList :items="users">
      <template #default="{ item, index }">
        <span :style="{ color: index % 2 ? 'blue' : 'green' }">
          #{{ index + 1 }} {{ item.name }} ({{ item.email }})
        </span>
      </template>
    </DataList>

    <!-- 4. Dynamic Components -->
    <h2>Dynamic Component</h2>
    <button v-for="tab in tabs" :key="tab"
      @click="currentTab = tab"
      :class="{ active: currentTab === tab }">
      {{ tab }}
    </button>
    <keep-alive>
      <component :is="tabComponents[currentTab]" />
    </keep-alive>
  </div>
</template>

<script setup>
import { ref, markRaw, shallowRef } from 'vue'

// --- Child components defined inline for demo ---
import { defineComponent, h } from 'vue'

// UserCard component
const UserCard = defineComponent({
  props: {
    name: { type: String, required: true },
    age: { type: Number, default: 0 },
    skills: { type: Array, default: () => [] },
    theme: { type: String, default: 'light' }
  },
  emits: ['toggleTheme'],
  setup(props, { slots, emit }) {
    return () => h('div', {
      class: `user-card ${props.theme}`,
      style: { padding: '1rem', border: '1px solid #42b883', margin: '0.5rem 0' }
    }, [
      h('h3', `${props.name}, ${props.age} years old`),
      h('p', `Skills: ${props.skills.join(', ')}`),
      slots.default?.(),
      slots.badge?.(),
      h('button', { onClick: () => emit('toggleTheme') }, 'Toggle Theme')
    ])
  }
})

// Counter component
const Counter = defineComponent({
  props: { initial: { type: Number, default: 0 } },
  emits: ['change'],
  setup(props, { emit }) {
    const count = ref(props.initial)
    const increment = () => {
      count.value++
      emit('change', count.value)
    }
    const decrement = () => {
      count.value--
      emit('change', count.value)
    }
    return () => h('div', { style: { margin: '0.5rem 0' } }, [
      h('button', { onClick: decrement }, '-'),
      h('span', { style: { margin: '0 1rem' } }, `Count: ${count.value}`),
      h('button', { onClick: increment }, '+')
    ])
  }
})

// DataList component with scoped slot
const DataList = defineComponent({
  props: { items: { type: Array, required: true } },
  setup(props, { slots }) {
    return () => h('ul', {},
      props.items.map((item, index) =>
        h('li', { key: item.id }, slots.default?.({ item, index }))
      )
    )
  }
})

// Tab components
const TabA = defineComponent({
  setup() { const v = ref(0); return () => h('p', `Tab A - count: ${v.value} `, [h('button', { onClick: () => v.value++ }, '+')]) }
})
const TabB = defineComponent({
  setup() { const v = ref(''); return () => h('div', [h('input', { onInput: (e) => v.value = e.target.value, value: v.value }), h('p', `Input: ${v.value}`)]) }
})

// Parent state
const theme = ref('light')
function toggleTheme() {
  theme.value = theme.value === 'light' ? 'dark' : 'light'
}

const lastChange = ref('none')
function onCountChange(val) { lastChange.value = `Counter changed to ${val}` }

const users = ref([
  { id: 1, name: 'Alice', email: 'alice@example.com' },
  { id: 2, name: 'Bob', email: 'bob@example.com' },
  { id: 3, name: 'Charlie', email: 'charlie@example.com' }
])

const tabs = ['Counter', 'Input']
const currentTab = ref('Counter')
const tabComponents = { Counter: markRaw(TabA), Input: markRaw(TabB) }
</script>
```

## ⚠️ 注意事项

| 注意点 | 说明 | 建议 |
|--------|------|------|
| Props单向流 | 子组件不应修改props | 需要修改时emit事件让父组件改 |
| 组件命名 | 模板中用kebab-case或PascalCase | 推荐PascalCase |
| emit声明 | 建议用defineEmits声明事件 | 方便维护和类型推导 |
| v-model组件 | 本质是modelValue+update:modelValue | Vue 3支持多个v-model |
| key on components | 强制替换组件实例 | 用于重置组件状态 |
| markRaw | 动态组件不需要响应式 | 避免不必要的性能开销 |

## 🚫 避坑指南

### 1. 直接修改Props
❌ **错误：** `props.count++`（单向数据流，不能直接修改）
✅ **正确：** `emit('update:count', props.count + 1)`

### 2. Props默认值用对象/数组
❌ **错误：** `default: []` 或 `default: {}`（多个实例共享同一引用）
✅ **正确：** `default: () => []` 或 `default: () => ({})`

### 3. 动态组件不用keep-alive
❌ **错误：** 切换组件后状态丢失
✅ **正确：** `<keep-alive><component :is="comp" /></keep-alive>`

### 4. 忘记声明emits
❌ **错误：** 直接`emit('xxx')`不声明，难以追踪
✅ **正确：** `const emit = defineEmits(['xxx'])`

### 5. 循环引用组件
❌ **错误：** A组件import B，B组件import A
✅ **正确：** 用异步组件`defineAsyncComponent`或重构组件关系

## 🔄 举一反三

| 概念 | 生活类比 | 说明 |
|------|---------|------|
| 组件 | 函数 | 输入(props)→处理→输出(template) |
| Props | 函数参数 | 父组件传给子组件的数据 |
| Events | 回调函数 | 子组件说"这事发生了，你来处理" |
| Slot | 插槽/口袋 | 父组件往子组件的口袋里塞内容 |
| 作用域插槽 | 预填表 | 子组件提供数据，父组件决定怎么显示 |
| keep-alive | 休眠舱 | 切走时不销毁，切回来继续用 |
| 动态组件 | 变形金刚 | 一个位置，根据条件变不同组件 |

## 🚀 代码演进

### v1 - 基础Props和Events

```vue
<!-- Child.vue -->
<template><button @click="$emit('click')">{{ label }}</button></template>
<script>
export default {
  props: ['label'],
  emits: ['click']
}
</script>
```

### v2 - script setup + defineProps/defineEmits

```vue
<!-- Child.vue -->
<template><button @click="emit('click')">{{ label }}</button></template>
<script setup>
const props = defineProps({ label: String })
const emit = defineEmits(['click'])
</script>
```

### v3 - 完整组件通信（Slots + v-model + 验证）

```vue
<!-- AdvancedChild.vue -->
<template>
  <div class="card">
    <slot name="header"><h3>{{ title }}</h3></slot>
    <slot :data="item">{{ item.name }}</slot>
    <input :value="modelValue" @input="emit('update:modelValue', $event.target.value)" />
  </div>
</template>
<script setup>
import { defineProps, defineEmits } from 'vue' // auto-imported in <script setup>
const props = defineProps({
  title: { type: String, required: true },
  item: { type: Object, required: true },
  modelValue: { type: String, default: '' }
})
const emit = defineEmits(['update:modelValue'])
</script>
```
