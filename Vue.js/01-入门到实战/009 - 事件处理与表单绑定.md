# 事件处理与表单绑定

> 📊 **难度：初级** | 🏷️ Vue.js 3.x | ⏱️ 阅读约 15 分钟

---

## 📖 概念讲解

事件处理和表单绑定是用户交互的核心。Vue提供了优雅的方式来监听DOM事件和处理表单输入。

**事件处理：**

Vue使用`v-on`指令（缩写`@`）监听DOM事件，可以在模板中直接写内联语句，也可以绑定一个方法。

```html
<!-- 内联语句 -->
<button @click="count++">{{ count }}</button>
<!-- 绑定方法 -->
<button @click="handleClick">Click</button>
<!-- 调用方法并传参 -->
<button @click="greet('Hello')">Greet</button>
<!-- 访问原生事件对象 -->
<button @click="handleClick($event)">Click</button>
```

**事件修饰符链：**

修饰符可以链式使用，按顺序执行：
```html
<a @click.stop.prevent="handler">Link</a>
```

常用修饰符：
- `.stop` - 阻止事件冒泡
- `.prevent` - 阻止默认行为
- `.capture` - 使用捕获模式
- `.self` - 仅在event.target是自身时触发
- `.once` - 只触发一次
- `.passive` - 提升滚动性能

**按键修饰符：**
```html
<input @keyup.enter="submit" />
<input @keyup.esc="cancel" />
<!-- 支持组合键 -->
<input @keyup.ctrl.enter="submit" />
```

**表单绑定 v-model：**

v-model根据表单元素类型自动选择正确的方式更新数据：
- `<input type="text">` → value + input事件
- `<textarea>` → value + input事件
- `<input type="checkbox">` → checked + change事件
- `<input type="radio">` → value + change事件
- `<select>` → value + change事件

## 💻 完整代码

```vue
<!-- EventFormDemo.vue -->
<template>
  <div class="demo">
    <!-- 1. Event handling -->
    <h2>1. Event Handling</h2>
    <button @click="count++">Inline: {{ count }}</button>
    <button @click="increment(5)">Method: +{{ step }}</button>
    <button @click="logEvent($event)">Log Event</button>
    <button @click="count = 0; clickHistory = []">Reset</button>
    <p>Click history: {{ clickHistory.join(', ') || 'none' }}</p>

    <!-- Event modifiers -->
    <h2>2. Event Modifiers</h2>
    <div @click="log('outer')" class="box outer">
      Outer (click me)
      <div @click.stop="log('inner')" class="box inner">
        Inner (.stop prevents bubble)
      </div>
    </div>
    <a href="https://vuejs.org" @click.prevent="log('prevented')">
      Link with .prevent
    </a>
    <button @click.once="log('once')">Fire once only</button>

    <!-- Key modifiers -->
    <h2>3. Key Modifiers</h2>
    <input @keyup.enter="log('Enter pressed')"
      @keyup.esc="log('Esc pressed')"
      @keyup.ctrl.enter="log('Ctrl+Enter')"
      placeholder="Try Enter / Esc / Ctrl+Enter" />

    <!-- 2. Form binding -->
    <h2>4. Form Controls</h2>
    <form @submit.prevent="submitForm">
      <!-- Text input -->
      <label>Name: <input v-model.trim="form.name" /></label>
      <!-- Number input -->
      <label>Age: <input v-model.number="form.age" type="number" /></label>
      <!-- Textarea -->
      <label>Bio: <textarea v-model="form.bio"></textarea></label>

      <!-- Checkboxes - single -->
      <label><input type="checkbox" v-model="form.agree" /> Agree to terms</label>

      <!-- Checkboxes - multiple -->
      <p>Skills:</p>
      <label v-for="skill in skillOptions" :key="skill">
        <input type="checkbox" :value="skill" v-model="form.skills" />
        {{ skill }}
      </label>

      <!-- Radio buttons -->
      <p>Gender:</p>
      <label v-for="g in genders" :key="g.value">
        <input type="radio" :value="g.value" v-model="form.gender" />
        {{ g.label }}
      </label>

      <!-- Select -->
      <label>City:
        <select v-model="form.city">
          <option disabled value="">Select city</option>
          <option v-for="c in cities" :key="c" :value="c">{{ c }}</option>
        </select>
      </label>

      <!-- Multi-select -->
      <label>Hobbies:
        <select v-model="form.hobbies" multiple>
          <option v-for="h in hobbyOptions" :key="h" :value="h">{{ h }}</option>
        </select>
      </label>

      <!-- Lazy input -->
      <label>Lazy input: <input v-model.lazy="form.lazyValue" /></label>
      <p>Lazy value updates on blur/enter: {{ form.lazyValue }}</p>

      <button type="submit">Submit Form</button>
    </form>

    <!-- Form data preview -->
    <h2>5. Form Data Preview</h2>
    <pre>{{ JSON.stringify(form, null, 2) }}</pre>
  </div>
</template>

<script setup>
import { ref, reactive } from 'vue'

// Event handling
const count = ref(0)
const step = 5
const clickHistory = ref([])

function increment(n) {
  count.value += n
  clickHistory.value.push(`+${n}`)
}

function logEvent(event) {
  clickHistory.value.push(`${event.type}@${event.target.tagName}`)
}

function log(msg) {
  clickHistory.value.push(msg)
}

// Form data
const form = reactive({
  name: '',
  age: null,
  bio: '',
  agree: false,
  skills: [],
  gender: '',
  city: '',
  hobbies: [],
  lazyValue: ''
})

const skillOptions = ['Vue', 'React', 'Angular', 'Node.js']
const genders = [
  { value: 'male', label: 'Male' },
  { value: 'female', label: 'Female' }
]
const cities = ['Beijing', 'Shanghai', 'Shenzhen', 'Hangzhou']
const hobbyOptions = ['Coding', 'Gaming', 'Reading', 'Travel']

function submitForm() {
  console.log('Form submitted:', JSON.stringify(form, null, 2))
  alert('Form submitted! Check console for data.')
}
</script>

<style scoped>
.box { padding: 1rem; margin: 0.5rem 0; border: 2px solid #42b883; cursor: pointer; }
.outer { background: #e8f5e9; }
.inner { background: #c8e6c9; }
label { display: block; margin: 0.3rem 0; }
input, textarea, select { margin: 0.3rem 0; padding: 0.3rem; }
pre { background: #f5f5f5; padding: 1rem; overflow: auto; }
</style>
```

## ⚠️ 注意事项

| 注意点 | 说明 | 建议 |
|--------|------|------|
| v-model.trim | 自动去除首尾空格 | 用户名等字段推荐使用 |
| v-model.number | 输入值自动转数字 | type="number"输入框配合使用 |
| v-model.lazy | change事件而非input事件 | 减少更新频率，适合实时搜索 |
| checkbox多选 | 绑定数组，value属性指定值 | 单选绑boolean，多选绑数组 |
| select多选 | 需要multiple属性 | 按住Ctrl/Cmd多选 |
| 修饰符顺序 | 按从左到右顺序执行 | .stop .prevent 顺序有影响 |

## 🚫 避坑指南

### 1. 忘记$event导致报错
❌ **错误：** `@click="handler"` 但handler需要event对象
✅ **正确：** `@click="handler($event)"` 或 `@click="(e) => handler(e)"`

### 2. checkbox v-model绑定错误
❌ **错误：** 多个checkbox绑定同一个字符串ref
✅ **正确：** 多选绑定数组，单选绑定boolean

### 3. select默认选中不生效
❌ **错误：** `<option value="">选择</option>` 但v-model初始值不是""
✅ **正确：** 确保v-model初始值与某个option的value匹配

### 4. 事件修饰符滥用
❌ **错误：** 所有事件都加.stop.prevent
✅ **正确：** 按需使用，理解每个修饰符的含义

### 5. 表单提交不阻止默认行为
❌ **错误：** `<form>` 没有`@submit.prevent`，提交后页面刷新
✅ **正确：** `<form @submit.prevent="handleSubmit">`

## 🔄 举一反三

| 概念 | 生活类比 | 说明 |
|------|---------|------|
| v-on | 对讲机 | "听到枪声就跑" → 事件触发执行 |
| .stop | 隔音墙 | 声音不再向上传递（阻止冒泡） |
| .prevent | 拦截 | 阻止链接跳转等默认行为 |
| v-model | 对话 | 双方都能说能听（双向绑定） |
| .trim | 剪刀 | 自动剪掉首尾多余空格 |
| .number | 翻译器 | 字符串自动翻译为数字 |
| .lazy | 缓冲 | 不急着发，等说完再传 |
