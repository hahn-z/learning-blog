概念讲解

什么是自定义指令？

自定义指令（Custom Directives）允许你封装底层 DOM 操作，以声明式的方式应用到元素上。适用于需要直接操作 DOM 的场景。

指令钩子函数

钩子 触发时机
created 元素创建后，属性/事件绑定前
beforeMount 元素挂载到 DOM 前
mounted 元素挂载后（最常用）
beforeUpdate 父组件更新前
updated 父组件更新后
beforeUnmount 元素卸载前
unmounted 元素卸载后

指令与组件的选择

优先用组件，只在需要复用底层 DOM 操作时才用指令。

脑图

自定义指令
├── 钩子函数
│   ├── mounted — 最常用
│   ├── updated — 值变化时
│   └── unmounted — 清理
├── 注册方式
│   ├── app.directive() — 全局
│   └── directives: {} — 局部（setup 中 vXxx）
├── 实用指令

│   ├── v-permission — 权限控制
│   ├── v-debounce — 防抖点击
│   ├── v-lazy — 图片懒加载
│   └── v-copy — 一键复制
└── 简写
    └── (el, binding) => { ... }
       等同于 mounted + updated

完整代码

// directives/index.js — Common custom directives

// Auto-focus directive
export const vFocus = {
  mounted(el) {
    el.focus()
  }
}

// Permission control directive
// Usage: v-permission="['admin', 'editor']"
export const vPermission = {
  mounted(el, binding) {
    const requiredRoles = binding.value
    const userRoles = JSON.parse(localStorage.getItem('roles') || '[]')
    const hasPermission = requiredRoles.some(role => userRoles.includes(role))

    if (!hasPermission) {
      // Remove element from DOM
      el.parentNode?.removeChild(el)
    }
  }
}

// Debounce click directive
// Usage: v-debounce="handleSubmit" or v-debounce:500="handleSubmit"
export const vDebounce = {
  mounted(el, binding) {
    const delay = binding.arg ? parseInt(binding.arg) : 300
    let timer = null

    el.addEventListener('click', () => {
      if (timer) clearTimeout(timer)
      timer = setTimeout(() => {
        binding.value()
      }, delay)
    })
  },

  unmounted(el) {
    // Clean up is handled by element removal
  }
}

// Copy to clipboard directive
// Usage: v-copy="'text to copy'"
export const vCopy = {
  mounted(el, binding) {
    el.style.cursor = 'pointer'
    el.addEventListener('click', () => {
      navigator.clipboard.writeText(binding.value)
        .then(() => {
          // Visual feedback
          const original = el.textContent
          el.textContent = 'Copied!'
          setTimeout(() => { el.textContent = original }, 1500)
        })
        .catch(() => {
          // Fallback for older browsers
          const textarea = document.createElement('textarea')
          textarea.value = binding.value
          document.body.appendChild(textarea)
          textarea.select()
          document.execCommand('copy')
          document.body.removeChild(textarea)
        })
    })
  }
}

// Image lazy loading directive
// Usage: v-lazy="imageUrl"
export const vLazy = {
  mounted(el, binding) {
    const observer = new IntersectionObserver(
      (entries) => {
        entries.forEach(entry => {
          if (entry.isIntersecting) {
            el.src = binding.value
            el.classList.add('loaded')
            observer.unobserve(el)
          }
        })
      },
      { rootMargin: '100px' }
    )
    observer.observe(el)
    el._lazyObserver = observer
  },

  unmounted(el) {
    el._lazyObserver?.disconnect()
  }
}

// Long press directive
// Usage: v-longpress="handleLongPress"
export const vLongpress = {
  mounted(el, binding) {
    let timer = null

    el.addEventListener('mousedown', () => {
      timer = setTimeout(() => binding.value(), 500)
    })

    el.addEventListener('mouseup', () => {
      if (timer) clearTimeout(timer)
    })

    el.addEventListener('mouseleave', () => {
      if (timer) clearTimeout(timer)
    })
  }
}
// main.js — Register directives globally
import { createApp } from 'vue'
import App from './App.vue'

const app = createApp(App)

// Register globally
app.directive('focus', vFocus)
app.directive('permission', vPermission)
app.directive('debounce', vDebounce)
app.directive('copy', vCopy)
app.directive('lazy', vLazy)

app.mount('#app')
<!-- components/DirectiveDemo.vue — Using custom directives -->
<script setup>
import { ref } from 'vue'

const imageUrl = ref('https://picsum.photos/400/300')
const copyText = ref('Hello from v-copy!')

function handleClick() {
  console.log('Debounced click!')
}

function handleLongPress() {
  alert('Long pressed!')
}
</script>

<template>
  <div>
    <!-- Auto-focus -->
    <input v-focus placeholder="Auto-focused" />

    <!-- Permission control -->
    <button v-permission="['admin']">Admin Only Button</button>

    <!-- Debounce click -->
    <button v-debounce:500="handleClick">Debounced Button (500ms)</button>

    <!-- Copy text -->
    <span v-copy="copyText">Click to copy: {{ copyText }}</span>

    <!-- Lazy load image -->
    <img v-lazy="imageUrl" alt="Lazy loaded" style="width: 400px" />

    <!-- Long press -->
    <button v-longpress="handleLongPress">Hold me (500ms)</button>
  </div>
</template>

页面加载:

用户角色 ["user"]:
  → v-permission="['admin']": 按钮被移除

用户角色 ["admin"]:
  → v-permission="['admin']": 按钮正常显示

点击 debounced 按钮:
  → 500ms 内重复点击只执行最后一次

点击 copy 文本:
  → 文本复制到剪贴板
  → 显示 "Copied!" 1.5秒后恢复

滚动到图片位置:
  → IntersectionObserver 检测到可见
  → 设置 src 加载图片

注意事项

注意点 说明
指令命名 vFocus 对应 v-focus（自动转换）
简写形式 (el, binding) => {} 等同于 mounted + updated
清理资源 unmounted 中清理 observer、事件监听等
binding.value 指令绑定值，如 v-xxx="expr" 中的 expr
binding.arg 指令参数，如 v-xxx:500 中的 "500"
binding.modifiers 修饰符，如 v-xxx.prevent 中的 { prevent: true }

避坑指南

❌ 错误做法 ✅ 正确做法
指令中修改组件状态 指令只做 DOM 操作，状态管理用组件
不清理副作用 unmounted 中 disconnect observer / removeEventListener
用指令实现 v-if 功能 逻辑显示/隐藏用 v-if，指令做 DOM 操作
全局注册太多指令 常用的全局，不常用的局部注册
指令中硬编码数据 通过 binding.value 动态传入
