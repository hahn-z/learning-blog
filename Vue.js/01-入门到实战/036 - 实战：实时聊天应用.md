## 概念讲解

### 实时聊天技术方案

实时聊天的核心技术是 **WebSocket**，它提供全双工通信通道，服务端可以主动推送消息。

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| WebSocket | 全双工、低延迟 | 需要服务端支持 | 实时聊天 |
| SSE | 单向推送、简单 | 只能服务端→客户端 | 通知/消息流 |
| 轮询 | 实现简单 | 延迟高、浪费带宽 | 兼容性方案 |
| Socket.IO | 自动降级、房间 | 依赖特定库 | 快速开发 |

### 聊天应用核心功能

1. **连接管理**：建立/断开/重连 WebSocket
2. **消息收发**：实时发送和接收消息
3. **聊天室**：加入/离开房间
4. **在线状态**：显示用户在线/离线
5. **消息历史**：加载历史消息，滚动加载
6. **未读消息**：计数和提醒

## 完整代码

### WebSocket 封装

```ts
// src/utils/websocket.ts
type MessageHandler = (data: any) => void

export class WebSocketClient {
  private ws: WebSocket | null = null
  private url: string
  private handlers: Map<string, MessageHandler[]> = new Map()
  private reconnectAttempts = 0
  private maxReconnectAttempts = 5
  private heartbeatInterval: ReturnType<typeof setInterval> | null = null

  constructor(url: string) {
    this.url = url
  }

  // Connect to WebSocket server
  connect(token: string) {
    this.ws = new WebSocket(`${this.url}?token=${token}`)

    this.ws.onopen = () => {
      console.log('[WS] Connected')
      this.reconnectAttempts = 0
      this.startHeartbeat()
    }

    this.ws.onmessage = (event) => {
      const data = JSON.parse(event.data)
      // Dispatch message by type
      const handlers = this.handlers.get(data.type) || []
      handlers.forEach(handler => handler(data.payload))
    }

    this.ws.onclose = () => {
      console.log('[WS] Disconnected')
      this.stopHeartbeat()
      this.attemptReconnect(token)
    }

    this.ws.onerror = (error) => {
      console.error('[WS] Error:', error)
    }
  }

  // Send message to server
  send(type: string, payload: any) {
    if (this.ws?.readyState === WebSocket.OPEN) {
      this.ws.send(JSON.stringify({ type, payload }))
    }
  }

  // Register event handler
  on(type: string, handler: MessageHandler) {
    if (!this.handlers.has(type)) {
      this.handlers.set(type, [])
    }
    this.handlers.get(type)!.push(handler)
  }

  // Remove event handler
  off(type: string, handler: MessageHandler) {
    const handlers = this.handlers.get(type)
    if (handlers) {
      const index = handlers.indexOf(handler)
      if (index > -1) handlers.splice(index, 1)
    }
  }

  // Disconnect
  disconnect() {
    this.stopHeartbeat()
    this.ws?.close()
    this.ws = null
  }

  // Heartbeat to keep connection alive
  private startHeartbeat() {
    this.heartbeatInterval = setInterval(() => {
      this.send('ping', { timestamp: Date.now() })
    }, 30000)
  }

  private stopHeartbeat() {
    if (this.heartbeatInterval) {
      clearInterval(this.heartbeatInterval)
      this.heartbeatInterval = null
    }
  }

  // Reconnect with exponential backoff
  private attemptReconnect(token: string) {
    if (this.reconnectAttempts >= this.maxReconnectAttempts) {
      console.error('[WS] Max reconnection attempts reached')
      return
    }
    const delay = Math.min(1000 * 2 ** this.reconnectAttempts, 30000)
    this.reconnectAttempts++
    console.log(`[WS] Reconnecting in ${delay}ms (attempt ${this.reconnectAttempts})`)
    setTimeout(() => this.connect(token), delay)
  }
}
```

### Chat Store

```ts
// src/stores/chat.ts
import { defineStore } from 'pinia'
import { ref, computed } from 'vue'
import { WebSocketClient } from '@/utils/websocket'
import { useAuthStore } from '@/stores/auth'

export interface Message {
  id: string
  roomId: string
  senderId: number
  senderName: string
  senderAvatar: string
  content: string
  type: 'text' | 'image' | 'system'
  createdAt: string
}

export interface Room {
  id: string
  name: string
  avatar: string
  type: 'private' | 'group'
  members: number[]
  lastMessage?: Message
  unreadCount: number
}

export const useChatStore = defineStore('chat', () => {
  const auth = useAuthStore()
  const ws = ref<WebSocketClient | null>(null)

  const rooms = ref<Room[]>([])
  const currentRoomId = ref<string>('')
  const messages = ref<Map<string, Message[]>>(new Map())
  const onlineUsers = ref<Set<number>>(new Set())

  // Current room messages
  const currentMessages = computed(() =>
    messages.value.get(currentRoomId.value) || []
  )

  // Total unread count
  const totalUnread = computed(() =>
    rooms.value.reduce((sum, r) => sum + r.unreadCount, 0)
  )

  // Initialize WebSocket connection
  function connect() {
    const client = new WebSocketClient(
      import.meta.env.VITE_WS_URL || 'ws://localhost:8080/ws'
    )

    // Listen for new messages
    client.on('message', (msg: Message) => {
      const roomMessages = messages.value.get(msg.roomId) || []
      roomMessages.push(msg)
      messages.value.set(msg.roomId, roomMessages)

      // Update unread count if not in current room
      if (msg.roomId !== currentRoomId.value) {
        const room = rooms.value.find(r => r.id === msg.roomId)
        if (room) room.unreadCount++
      }
    })

    // Listen for online status updates
    client.on('presence', (data: { userId: number; online: boolean }) => {
      if (data.online) onlineUsers.value.add(data.userId)
      else onlineUsers.value.delete(data.userId)
    })

    client.connect(auth.token!)
    ws.value = client
  }

  // Send a text message
  function sendMessage(content: string) {
    ws.value?.send('message', {
      roomId: currentRoomId.value,
      content,
      type: 'text'
    })
  }

  // Switch to a room
  function selectRoom(roomId: string) {
    currentRoomId.value = roomId
    // Clear unread
    const room = rooms.value.find(r => r.id === roomId)
    if (room) room.unreadCount = 0
    // Mark messages as read
    ws.value?.send('read', { roomId })
  }

  // Disconnect
  function disconnect() {
    ws.value?.disconnect()
    ws.value = null
  }

  return {
    rooms, currentRoomId, messages, onlineUsers,
    currentMessages, totalUnread,
    connect, sendMessage, selectRoom, disconnect
  }
})
```

  await nextTick()
  if (messageListRef.value) {
    messageListRef.value.scrollTop = messageListRef.value.scrollHeight

  if (!inputText.value.trim()) return
  chatStore.sendMessage(inputText.value.trim())

  return new Date(date).toLocaleTimeString('en-US', { hour: '2-digit', minute: '2-digit' })

## 执行预览

```
打开聊天页 → WebSocket 连接 ws://localhost:8080/ws?token=xxx
           → 接收房间列表 + 在线用户
           → 渲染左侧会话列表

选择房间 → 加载历史消息 → 滚动到底部
发送 "Hello" → WS send { type: "message", payload: { roomId, content: "Hello" } }
            → 接收消息回调 → 追加到消息列表 → 自动滚动

对方发消息 → WS onmessage → 显示新消息 → 非当前房间则未读+1
```

## 注意事项

| 事项 | 说明 | 建议 |
|------|------|------|
| 连接安全 | wss 加密 | 生产环境必须 wss |
| 心跳机制 | 防止连接断开 | 30s ping/pong |
| 断线重连 | 网络不稳定 | 指数退避重连 |
| 消息去重 | 网络重发可能重复 | 消息 ID 去重 |
| 历史消息 | 初始加载 + 滚动加载 | 分页拉取 |
| 敏感内容 | XSS 防护 | 渲染前转义 HTML |

## 避坑指南

- ❌ 组件内直接 `new WebSocket()` → ✅ 封装 WebSocketClient 类统一管理
- ❌ 不做断线重连 → ✅ 指数退避 + 最大重试次数
- ❌ 不做心跳 → ✅ 定时 ping，超时主动重连
- ❌ 消息列表不用虚拟滚动 → ✅ 大量消息时用虚拟列表
- ❌ 组件卸载不断开 → ✅ onUnmounted 中 disconnect
