# Vue 3简介与环境搭建

> 📊 **难度：初级** | 🏷️ Vue.js 3.x | ⏱️ 阅读约 15 分钟

---

## 📖 概念讲解

Vue.js是一个用于构建用户界面的渐进式JavaScript框架。Vue 3于2020年9月发布，带来了Composition API、更好的TypeScript支持、更小的包体积和更快的渲染性能。

**核心特点：**

- **渐进式框架**：可以从小到大，逐步引入功能。你可以在页面中只用一个`<script>`标签引入Vue，也可以使用完整的工程化方案。
- **响应式系统**：基于Proxy实现的响应式系统，自动追踪依赖关系，数据变化时自动更新DOM。
- **组件化开发**：将页面拆分为独立、可复用的组件，每个组件包含自己的模板、逻辑和样式。
- **虚拟DOM**：通过虚拟DOM diff算法，最小化真实DOM操作，提升性能。

**Vue 3的三大API风格：**

| 风格 | 说明 | 适用场景 |
|------|------|---------|
| Options API | 选项式API，Vue 2风格 | 简单组件、初学者 |
| Composition API | 组合式API，setup函数 | 复杂逻辑复用 |
| `<script setup>` | Composition API的语法糖 | 推荐的开发方式 |

**开发环境需求：**

- Node.js ≥ 16.x
- npm ≥ 7.x 或 pnpm
- 编辑器推荐 VS Code + Volar插件

## 💻 完整代码

```vue
<!-- App.vue - Vue 3 Application Entry -->
<template>
  <div class="app">
    <h1>{{ greeting }}</h1>
    <!-- v-model for two-way binding -->
    <input v-model="name" placeholder="Enter your name" />
    <!-- Conditional rendering -->
    <p v-if="name">Hello, {{ name }}! Welcome to Vue 3!</p>
    <p v-else>Please enter your name above.</p>

    <!-- List rendering -->
    <h2>Vue 3 Core Features</h2>
    <ul>
      <li v-for="feature in features" :key="feature.name">
        {{ feature.name }}: {{ feature.desc }}
      </li>
    </ul>

    <!-- Event handling -->
    <button @click="counter++">Clicked {{ counter }} times</button>
    <button @click="resetAll">Reset All</button>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'

// Reactive state declarations
const name = ref('')
const counter = ref(0)

// Computed property
const greeting = computed(() => {
  return name.value ? `Hello, ${name.value}!` : 'Welcome to Vue 3'
})

// Feature list data
const features = ref([
  { name: 'Composition API', desc: 'Flexible logic composition' },
  { name: 'Proxy Reactivity', desc: 'Deep reactive tracking' },
  { name: 'Fragment', desc: 'Multiple root elements' },
  { name: 'Teleport', desc: 'Render content anywhere' },
])

// Method: reset all state
function resetAll() {
  name.value = ''
  counter.value = 0
}
</script>

<style scoped>
.app {
  max-width: 600px;
  margin: 2rem auto;
  padding: 1rem;
  font-family: sans-serif;
}
input {
  padding: 0.5rem;
  margin: 1rem 0;
  width: 100%;
  font-size: 1rem;
}
button {
  padding: 0.5rem 1rem;
  margin: 0.5rem;
  cursor: pointer;
}
</style>
```

## ⚠️ 注意事项

| 注意点 | 说明 | 建议 |
|--------|------|------|
| Node.js版本 | Vue 3需要Node.js ≥ 16 | 使用nvm管理Node版本 |
| Vite vs Webpack | Vue 3官方推荐Vite | 新项目用Vite，老项目可保留Webpack |
| 浏览器兼容 | Vue 3不支持IE11 | 如需IE兼容，使用Vue 2 |
| Volar vs Vetur | Vetur是Vue 2插件 | Vue 3项目必须用Volar |
| 开发工具 | 需要Vue DevTools浏览器扩展 | 安装Vue DevTools beta版本 |

## 🚫 避坑指南

### 1. Node版本过低
❌ **错误：** 直接用系统自带Node，版本过旧导致依赖安装失败
✅ **正确：** 使用nvm安装Node.js 18+，确保版本兼容

### 2. 混用Vetur和Volar
❌ **错误：** 同时安装Vetur和Volar插件，导致语法高亮冲突
✅ **正确：** Vue 3项目禁用Vetur，只使用Volar

### 3. CDN引入方式开发
❌ **错误：** 用`<script src="vue.js">`开发完整项目，无法使用SFC
✅ **正确：** 使用Vite创建项目，享受SFC和HMR

### 4. 修改生成的文件后不生效
❌ **错误：** 修改了文件但浏览器没有更新，疯狂刷新
✅ **正确：** 检查终端是否报错，Vite的HMR会在控制台显示错误信息

## 🔄 举一反三

| Vue 3概念 | 生活类比 | 说明 |
|----------|---------|------|
| 渐进式框架 | 乐高积木 | 从一块积木开始，逐步搭建复杂结构 |
| 响应式 | Excel公式 | 单元格数据变了，公式结果自动更新 |
| 组件 | 函数 | 输入(props)→处理(逻辑)→输出(模板) |
| Vite | 快餐店厨房 | 开发时按需编译，不一次性全煮好 |
| HMR | 实时预览 | 修改代码后页面局部刷新，不丢失状态 |
| 虚拟DOM | 草稿纸 | 先在草稿上比较差异，再修改正式文档 |

## 🚀 代码演进

### v1 - 最简Vue应用（CDN方式）

```html
<!-- index.html - Simplest Vue 3 app via CDN -->
<div id="app">{{ message }}</div>
<script src="https://unpkg.com/vue@3"></script>
<script>
  Vue.createApp({
    data() { return { message: 'Hello Vue 3!' } }
  }).mount('#app')
</script>
```

### v2 - Options API单文件组件

```vue
<!-- App.vue - Options API style -->
<template>
  <h1>{{ message }}</h1>
</template>

<script>
export default {
  data() {
    return { message: 'Hello Vue 3!' }
  }
}
</script>
```

### v3 - Composition API + script setup（推荐）

```vue
<!-- App.vue - Modern <script setup> style -->
<template>
  <h1>{{ message }}</h1>
</template>

<script setup>
import { ref } from 'vue'
const message = ref('Hello Vue 3!')
</script>
```
