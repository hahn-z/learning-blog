022 - 模板引用ref

📂 Vue.js入门到实战 🏷️ Vue.js

难度标注

🟢 初级 — Vue 3 基础即可

概念讲解

什么是模板引用？

模板引用（Template Ref）是 Vue 中直接访问 DOM 元素或子组件实例的方式。通过 ref attribute 和 ref() 函数配合使用。

何时使用？

  • 需要直接操作 DOM（聚焦、滚动、测量尺寸）
  • 需要调用子组件的方法
  • 使用第三方 DOM 库（图表、地图等）
  • v-for 场景获取元素列表

    • 核心规则

    模板引用是逃生舱，优先使用声明式的 Vue 方式（v-bind、v-on），只在必要时使用 ref。

    脑图

    模板引用 ref
├── DOM 引用
│   ├── <input ref="inputRef">
│   └── const inputRef = ref(null)
├── 组件引用
│   ├── <Child ref="childRef">
│   └── childRef.value.childMethod()
├── v-for 中的 ref
│   ├── <div v-for ref="itemRefs">
│   └── itemRefs.value = [] (自动填充)
├── 函数 ref
│   ├── <div :ref="el => node = el">
│   └── 动态绑定
└── 注意事项
    ├── onMounted 后才能访问
    ├── defineExpose 暴露组件方法
    └── v-if 导致 ref 为 null

    完整代码

    <!-- components/TemplateRefDemo.vue — Basic DOM ref -->
<script setup>
import { ref, onMounted } from 'vue'

// Declare a ref to hold the DOM element
const inputRef = ref(null)

onMounted(() => {
  // Auto-focus the input after mount
  inputRef.value?.focus()
})

function clearInput() {
  inputRef.value.value = ''
  inputRef.value.focus()
}
</script>

<template>
  <div>
    <input ref="inputRef" placeholder="Auto-focused on mount" />
    <button @click="clearInput">Clear</button>
  </div>
</template>
    <!-- components/MeasureBox.vue — Measuring DOM dimensions -->
<script setup>
import { ref, onMounted } from 'vue'

const boxRef = ref(null)
const dimensions = ref({ width: 0, height: 0 })

onMounted(() => {
  updateDimensions()
  // Observe size changes
  const observer = new ResizeObserver(() => updateDimensions())
  observer.observe(boxRef.value)
})

function updateDimensions() {
  if (boxRef.value) {
    const rect = boxRef.value.getBoundingClientRect()
    dimensions.value = {
      width: Math.round(rect.width),
      height: Math.round(rect.height)
    }
  }
}
</script>

<template>
  <div
    ref="boxRef"
    class="resize-box"
    :style="{ width: '50%', minHeight: '100px' }"
  >
    <p>Width: {{ dimensions.width }}px</p>
    <p>Height: {{ dimensions.height }}px</p>
    <p>Try resizing the window!</p>
  </div>
</template>

<style scoped>
.resize-box {
  border: 2px dashed #42b883;
  padding: 20px;
  resize: both;
  overflow: auto;
}
</style>
    <!-- components/ChildWithExpose.vue — Component ref with defineExpose -->
<script setup>
import { ref } from 'vue'

const count = ref(0)

// Internal state — not exposed by default
const secret = ref('hidden')

// Expose specific methods and state to parent
defineExpose({
  count,
  increment() { count.value++ },
  reset() { count.value = 0 }
})
</script>

<template>
  <div>Child Count: {{ count }}</div>
</template>
    <!-- components/ParentComponent.vue — Using child component ref -->
<script setup>
import { ref } from 'vue'
import ChildWithExpose from './ChildWithExpose.vue'

const childRef = ref(null)

function callChildMethod() {
  // Access exposed child methods
  childRef.value?.increment()
}

function resetChild() {
  childRef.value?.reset()
}
</script>

<template>
  <div>
    <ChildWithExpose ref="childRef" />
    <button @click="callChildMethod">Call Child.increment()</button>
    <button @click="resetChild">Reset Child</button>
  </div>
</template>
    <!-- components/VForRefs.vue — Refs in v-for -->
<script setup>
import { ref, onMounted } from 'vue'

const items = ref(['Apple', 'Banana', 'Cherry'])
const itemRefs = ref([])

// itemRefs is automatically populated after render
onMounted(() => {
  console.log('All items:', itemRefs.value)
  // Scroll to the second item
  itemRefs.value[1]?.scrollIntoView({ behavior: 'smooth' })
})

function scrollToItem(index) {
  itemRefs.value[index]?.scrollIntoView({ behavior: 'smooth' })
}
</script>

<template>
  <div class="list-container">
    <div
      v-for="(item, index) in items"
      :key="item"
      :ref="el => { if (el) itemRefs[index] = el }"
      class="list-item"
    >
      {{ item }}
      <button @click="scrollToItem(index)">Scroll here</button>
    </div>
  </div>
</template>

<style scoped>
.list-container { height: 200px; overflow-y: auto; }
.list-item { padding: 40px; border: 1px solid #eee; }
</style>

    执行预览

    页面加载:
  → inputRef.value.focus() 自动聚焦输入框
  → MeasureBox 显示初始尺寸: 400x100

用户调整窗口:
  → ResizeObserver 触发
  → dimensions 更新为新尺寸

点击 "Call Child.increment()":
  → childRef.value.increment() 调用子组件方法
  → 子组件 count: 0 → 1

v-for refs:
  → itemRefs = [div, div, div]
  → scrollToItem(1) 平滑滚动到 Banana

    注意事项

    注意点 说明
    初始值 null ref 挂载前为 null，使用可选链 ?.
    onMounted 访问 DOM ref 只在 onMounted 后可用
    defineExpose
