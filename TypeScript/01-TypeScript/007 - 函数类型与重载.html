一、概念讲解

函数类型基础

在TypeScript中，函数的每个参数和返回值都可以（也应该）有类型。

// Named function
function add(a: number, b: number): number {
  return a + b;
}

// Arrow function
const multiply = (a: number, b: number): number => a * b;

// Function type expression
type MathOp = (a: number, b: number) => number;
const divide: MathOp = (a, b) => a / b;

参数变体

// Optional parameters (must come last)
function greet(name: string, title?: string): string {
  return title ? `${title} ${name}` : name;
}

// Default parameters
function createUser(name: string, role: string = 'viewer'): object {
  return { name, role };
}

// Rest parameters
function sum(...nums: number[]): number {
  return nums.reduce((a, b) => a + b, 0);
}

函数重载（Function Overloading）

TypeScript的函数重载通过多个签名+一个实现来实现：

// Overload signatures
function format(value: string): string;
function format(value: number): string;
// Implementation
function format(value: string | number): string {
  if (typeof value === 'string') return value.trim();
  return value.toFixed(2);
}

三、完整代码示例

// ============================================
// Function Types and Overloads
// ============================================

// --- 1. Basic Function Types ---
function greet(name: string): string {
  return `Hello, ${name}!`;
}

// Function type alias
type Callback = (data: string) => void;
type AsyncFn = (url: string) => Promise<Response>;

// --- 2. Optional & Default Parameters ---
function search(
  query: string,
  limit: number = 10,
  offset?: number
): string[] {
  const results: string[] = [];
  // Simulate search results
  for (let i = 0; i < limit; i++) {
    results.push(`${query}-result-${i}`);
  }
  return results;
}

console.log(search('ts'));           // 10 results
console.log(search('ts', 5));        // 5 results
console.log(search('ts', 5, 10));    // 5 results with offset

// --- 3. Rest Parameters ---
function log(level: string, ...messages: string[]): void {
  console.log(`[${level}]`, messages.join(' '));
}

log('INFO', 'Server', 'started', 'on', 'port', '3000');

// --- 4. Function Overloading ---

// Example 1: Type-safe event handler
function on(event: 'click', handler: (e: MouseEvent) => void): void;
function on(event: 'input', handler: (e: InputEvent) => void): void;
function on(event: string, handler: (e: Event) => void): void {
  console.log(`Registered handler for ${event}`);
}

on('click', (e) => {
  console.log('Click at', e.clientX, e.clientY); // e is MouseEvent
});

// Example 2: Utility function with overloads
function pluck<T, K extends keyof T>(obj: T, key: K): T[K];
function pluck<T>(obj: T, keys: (keyof T)[]): unknown[];
function pluck<T>(obj: T, keyOrKeys: keyof T | (keyof T)[]): unknown {
  if (Array.isArray(keyOrKeys)) {
    return keyOrKeys.map(k => obj[k]);
  }
  return obj[keyOrKeys];
}

const user = { name: 'Alice', age: 30, email: 'a@b.com' };
const name = pluck(user, 'name');      // type: string
const vals = pluck(user, ['name', 'age']); // type: unknown[]

// --- 5. 'this' Parameter ---
interface DB {
  users: string[];
  addUser(this: DB, name: string): void;
}

const db: DB = {
  users: [],
  addUser(name: string) {
    this.users.push(name);
  }
};

db.addUser('Alice');
console.log(db.users); // ['Alice']

// --- 6. Assertion Functions ---
function assertDefined<T>(value: T | undefined | null, msg?: string): asserts value is T {
  if (value == null) throw new Error(msg ?? 'Value is null/undefined');
}

let maybeName: string | undefined = 'Alice';
assertDefined(maybeName);
console.log(maybeName.toUpperCase()); // TS knows it's string now

// --- 7. Predicate Functions ---
function isString(value: unknown): value is string {
  return typeof value === 'string';
}

const items: unknown[] = ['hello', 42, 'world', true];
const strings = items.filter(isString); // type: string[]
console.log(strings); // ['hello', 'world']

// --- 8. Constructor Type ---
type Constructor<T> = new (...args: any[]) => T;

function createInstance<T>(Ctor: Constructor<T>, ...args: any[]): T {
  return new Ctor(...args);
}

class Person {
  constructor(public name: string) {}
}

const p = createInstance(Person, 'Alice');
console.log(p.name); // "Alice"

console.log('All function type examples completed!');

五、注意事项

注意点 说明
可选参数必须在必选参数后面 fn(a?: string, b: string)会报错
重载的实现在内部是any式处理 实现签名对外不可见
void返回值不阻止返回值 只是告诉调用者不要使用返回值
箭头函数不能重载 只有用function声明的函数支持重载
this参数是编译时擦除的 只用于类型检查，不影响运行时

六、避坑指南

❌ 常见错误 ✅ 正确做法
重载签名顺序随意 最具体的签名放最前面
实现签名与调用签名不匹配 实现参数必须是所有重载的联合
用any做回调参数 用泛型或具体函数类型
忽略返回值推断 复杂函数显式标注返回值类型
在重载中返回不同类型 用联合返回值或泛型保持一致性

              v1 — 简单函数

              function processInput(input) {
  if (typeof input === 'string') return input.toUpperCase();
  if (typeof input === 'number') return input.toFixed(2);
  return String(input);
}

              v2 — 添加类型注解

              function processInput(input: unknown): string {
  if (typeof input === 'string') return input.toUpperCase();
  if (typeof input === 'number') return input.toFixed(2);
  return String(input);
}

              v3 — 函数重载+泛型

              function process(input: string): string;
function process(input: number): string;
function process(input: string | number): string {
  if (typeof input === 'string') return input.toUpperCase();
  return input.toFixed(2);
}

// Generic version with return type inference
function transform<T extends string | number>(input: T): string {
  return typeof input === 'string' ? input.toUpperCase() : input.toFixed(2);
}
