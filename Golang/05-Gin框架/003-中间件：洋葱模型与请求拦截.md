# 中间件：洋葱模型与请求拦截

> **难度：⭐⭐ 进阶级** | 预计阅读：25分钟 | 适合：理解Gin基础路由，需要掌握请求拦截与处理的开发者

## 一、概念讲解

### 1.1 什么是中间件？

中间件是**包裹在Handler外层的函数链**，在请求到达最终Handler之前和之后执行逻辑。可以理解为"请求处理流水线上的工位"。

### 1.2 洋葱模型

Gin中间件采用**洋葱模型(Onion Model)**：

```
请求 →
  ┌─────────────────────────────┐
  │     Middleware A (Before)    │
  │  ┌───────────────────────┐  │
  │  │   Middleware B (Before)│  │
  │  │  ┌─────────────────┐  │  │
  │  │  │  Final Handler   │  │  │
  │  │  └─────────────────┘  │  │
  │  │   Middleware B (After) │  │
  │  └───────────────────────┘  │
  │     Middleware A (After)     │
  └─────────────────────────────┘
← 响应

执行顺序：
1. A before c.Next()
2. B before c.Next()
3. Handler 执行
4. B after c.Next()
5. A after c.Next()
```

关键API：
- **`c.Next()`**：暂停当前中间件，执行链中下一个handler，完成后返回
- **`c.Abort()`**：终止后续handler执行，直接返回响应

### 1.3 中间件的分类

| 类型 | 作用范围 | 示例 |
|------|---------|------|
| 全局中间件 | 所有路由 | Logger, Recovery, CORS |
| 分组中间件 | 某组路由 | `/api/v1`下的认证 |
| 单路由中间件 | 单个路由 | 特定接口的限流 |

## 三、完整代码

### 3.1 自定义认证中间件

```go
package main

import (
    "fmt"
    "net/http"
    "time"
    "github.com/gin-gonic/gin"
)

// Auth middleware validates Bearer token
func AuthMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        token := c.GetHeader("Authorization")
        if token == "" {
            c.AbortWithStatusJSON(http.StatusUnauthorized, gin.H{
                "error": "Authorization header is required",
            })
            return
        }
        // In real projects, validate JWT token here
        if token != "Bearer my-secret-token" {
            c.AbortWithStatusJSON(http.StatusUnauthorized, gin.H{
                "error": "Invalid token",
            })
            return
        }
        // Pass user info to downstream handlers
        c.Set("userID", "user-001")
        c.Set("role", "admin")
        c.Next()
    }
}
```

### 3.2 请求耗时中间件

```go
// Timer middleware logs request duration
func TimerMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        start := time.Now()
        
        // Process request
        c.Next()
        
        // After handler completes
        duration := time.Since(start)
        status := c.Writer.Status()
        fmt.Printf("[%s] %s %s - %d (%v)\n",
            start.Format("15:04:05"),
            c.Request.Method,
            c.Request.URL.Path,
            status,
            duration,
        )
    }
}
```

### 3.3 CORS跨域中间件

```go
// CORS middleware for cross-origin requests
func CORSMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        c.Header("Access-Control-Allow-Origin", "*")
        c.Header("Access-Control-Allow-Methods", "GET,POST,PUT,DELETE,PATCH,OPTIONS")
        c.Header("Access-Control-Allow-Headers", "Content-Type,Authorization")
        c.Header("Access-Control-Max-Age", "86400")
        
        // Handle preflight requests
        if c.Request.Method == "OPTIONS" {
            c.AbortWithStatus(http.StatusNoContent)
            return
        }
        
        c.Next()
    }
}
```

### 3.4 中间件传值与goroutine安全

```go
package main

import (
    "fmt"
    "net/http"
    "time"
    "github.com/gin-gonic/gin"
)

func main() {
    r := gin.Default()
    
    // Middleware that sets values in context
    r.Use(func(c *gin.Context) {
        c.Set("requestID", "req-"+time.Now().Format("20060102150405"))
        c.Set("startTime", time.Now())
        c.Next()
    })
    
    // Handler that reads middleware values
    r.GET("/info", func(c *gin.Context) {
        // Safe: reading values in the same goroutine
        reqID, _ := c.Get("requestID")
        startTime, _ := c.Get("startTime")
        
        c.JSON(http.StatusOK, gin.H{
            "requestID": reqID,
            "startTime": startTime,
        })
    })
    
    // Goroutine-safe access with c.Copy()
    r.GET("/async", func(c *gin.Context) {
        // IMPORTANT: c is NOT safe across goroutines!
        // Must use c.Copy() for goroutine access
        cCopy := c.Copy()
        
        go func() {
            // Safe: using copied context
            reqID, _ := cCopy.Get("requestID")
            fmt.Println("Background task for request:", reqID)
        }()
        
        c.JSON(http.StatusOK, gin.H{"status": "processing"})
    })
    
    // Protected routes with auth middleware
    authorized := r.Group("/api")
    authorized.Use(AuthMiddleware())
    {
        authorized.GET("/profile", func(c *gin.Context) {
            // Retrieve values set by auth middleware
            userID, _ := c.Get("userID")
            role, _ := c.Get("role")
            c.JSON(http.StatusOK, gin.H{
                "userID": userID,
                "role":   role,
            })
        })
    }
    
    r.Run(":8080")
}
```

## 五、注意事项

| 事项 | 说明 | 建议 |
|------|------|------|
| 中间件顺序 | 先注册先执行（洋葱外层） | 认证放在业务中间件之前 |
| c.Abort() | 阻止后续handler执行 | 配合`AbortWithStatusJSON`使用更简洁 |
| goroutine安全 | Context不是goroutine安全的 | 跨goroutine必须`c.Copy()` |
| c.Set类型 | 存储值为`interface{}` | 取出时需要类型断言 |
| 性能影响 | 每个中间件都有开销 | 避免在中间件做DB查询等重操作 |

## 六、避坑指南

### ❌ 在goroutine中直接使用Context
```go
r.GET("/bad", func(c *gin.Context) {
    go func() {
        // DANGER! c is not goroutine-safe
        fmt.Println(c.Request.URL.Path) // Race condition!
    }()
    c.JSON(200, gin.H{"status": "ok"})
})
```
### ✅ 使用c.Copy()复制Context
```go
r.GET("/good", func(c *gin.Context) {
    cCopy := c.Copy() // Create a safe copy
    go func() {
        // Safe: using the copied context
        fmt.Println(cCopy.Request.URL.Path)
    }()
    c.JSON(200, gin.H{"status": "ok"})
})
```

### ❌ 忘记调用c.Next()
```go
func BadMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        // Do something before
        fmt.Println("before")
        // Oops! Forgot c.Next() - all subsequent handlers won't execute!
    }
}
```
### ✅ 始终调用c.Next()（除非有意Abort）
```go
func GoodMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        fmt.Println("before")
        c.Next() // Pass control to next handler
        fmt.Println("after")
    }
}
```

### ❌ 中间件中panic未恢复
```go
func RiskyMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        // If this panics, server crashes without Recovery middleware
        result := mustNotFail()
        c.Next()
    }
}
```
### ✅ 使用Recovery中间件 + 优雅错误处理
```go
r := gin.New()
r.Use(gin.Recovery()) // Always include Recovery!
r.Use(RiskyMiddleware())
```

## 八、知识点总结

```
Gin中间件知识树
├── 核心机制
│   ├── HandlerChain (handlers []HandlerFunc + index)
│   ├── c.Next() → index++, 执行下一个
│   └── c.Abort() → index = abortIndex, 停止
├── 注册方式
│   ├── r.Use() → 全局
│   ├── group.Use() → 分组
│   └── r.GET(path, mid1, mid2, handler) → 单路由
├── 数据传递
│   ├── c.Set(key, value)
│   ├── c.Get(key) → (value, exists)
│   ├── c.GetString/GetInt/GetBool...
│   └── c.Copy() → goroutine安全副本
├── 常用中间件
│   ├── gin.Logger()
│   ├── gin.Recovery()
│   ├── 自定义CORS
│   ├── JWT认证
│   └── 请求耗时
└── 注意点
    ├── 顺序决定执行层级
    ├── Abort不返回响应，需手动写
    └── goroutine中用c.Copy()
```

## 九、举一反三

| Gin中间件模式 | 其他框架/语言 | 差异 |
|--------------|-------------|------|
| `r.Use(middleware)` | Express: `app.use(middleware)` | 几乎一样 |
| `c.Next()` | Express: `next()` | 概念相同 |
| `c.Abort()` | Koa: 无直接等价 | Koa用throw/return |
| 洋葱模型 | Koa的洋葱模型 | 完全相同的设计 |
| `c.Set/Get` | Express: `req.app.set/get` | Gin用Context更优雅 |

### v3: 完整中间件栈
```go
package main

import (
    "fmt"
    "log"
    "net/http"
    "sync"
    "time"
    "github.com/gin-gonic/gin"
)

// RequestID middleware adds unique ID to each request
func RequestIDMiddleware() gin.HandlerFunc {
    var counter int64
    return func(c *gin.Context) {
        counter++
        id := fmt.Sprintf("req-%d-%d", time.Now().Unix(), counter)
        c.Set("requestID", id)
        c.Header("X-Request-ID", id)
        c.Next()
    }
}

// RateLimit middleware limits requests per IP
func RateLimitMiddleware(maxRequests int, window time.Duration) gin.HandlerFunc {
    type visitor struct {
        count    int
        expireAt time.Time
    }
    var (
        mu       sync.Mutex
        visitors = make(map[string]*visitor)
    )
    
    // Cleanup expired entries periodically
    go func() {
        for range time.Tick(window) {
            mu.Lock()
            now := time.Now()
            for ip, v := range visitors {
                if now.After(v.expireAt) {
                    delete(visitors, ip)
                }
            }
            mu.Unlock()
        }
    }()
    
    return func(c *gin.Context) {
        ip := c.ClientIP()
        mu.Lock()
        now := time.Now()
        v, exists := visitors[ip]
        if !exists || now.After(v.expireAt) {
            visitors[ip] = &visitor{count: 1, expireAt: now.Add(window)}
            mu.Unlock()
            c.Next()
            return
        }
        v.count++
        if v.count > maxRequests {
            mu.Unlock()
            c.AbortWithStatusJSON(http.StatusTooManyRequests, gin.H{
                "error": "rate limit exceeded",
            })
            return
        }
        mu.Unlock()
        c.Next()
    }
}

// Auth middleware validates JWT token (simplified)
func AuthMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        token := c.GetHeader("Authorization")
        if token == "" {
            c.AbortWithStatusJSON(http.StatusUnauthorized, gin.H{
                "error": "missing authorization header",
            })
            return
        }
        // Simplified: accept any "Bearer xxx" with length > 10
        if len(token) < 16 || token[:7] != "Bearer " {
            c.AbortWithStatusJSON(http.StatusUnauthorized, gin.H{
                "error": "invalid token format",
            })
            return
        }
        c.Set("userID", "user-from-token")
        c.Next()
    }
}

func main() {
    gin.SetMode(gin.ReleaseMode)
    r := gin.New()
    
    // Global middleware stack (outer → inner)
    r.Use(gin.Recovery())
    r.Use(RequestIDMiddleware())
    r.Use(RateLimitMiddleware(60, time.Minute))
    r.Use(CORSMiddleware())
    
    // Public routes
    r.GET("/health", func(c *gin.Context) {
        c.JSON(http.StatusOK, gin.H{"status": "ok"})
    })
    
    // Protected routes
    api := r.Group("/api")
    api.Use(AuthMiddleware())
    {
        api.GET("/profile", func(c *gin.Context) {
            userID, _ := c.Get("userID")
            c.JSON(http.StatusOK, gin.H{"user": userID})
        })
    }
    
    log.Println("Server starting on :8080")
    r.Run(":8080")
}
```
