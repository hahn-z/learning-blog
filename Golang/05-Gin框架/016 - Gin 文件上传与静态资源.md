# 016 - Gin 文件上传与静态资源

> **难度：⭐⭐⭐** | **预计阅读：15 分钟**
>
> 掌握 Gin 框架中文件上传（单文件/多文件）、大小限制、类型校验、存储策略以及静态文件服务的完整方案。

---

## 一、概念讲解

文件上传是 Web 开发中最常见的功能之一。Gin 框架基于 `net/http` 的 `multipart/form-data` 解析，提供了简洁的文件上传 API。

**核心流程：**

```
客户端 (multipart/form-data) → Gin Handler → 解析文件 → 校验 → 存储 → 返回URL
```

**关键类型：**
- `*multipart.FileHeader` — 文件元信息（文件名、大小、MIME类型）
- `multipart.File` — 文件内容读取器
- `FormFile()` — 单文件快捷方法
- `MultipartForm()` — 多文件/表单数据

**静态文件服务：** Gin 提供 `Static()`、`StaticFile()`、`StaticFS()` 方法，将本地目录映射为 HTTP 路径。

## 三、完整代码

### v1：基础单文件上传

```go
package main

import (
	"fmt"
	"net/http"
	"os"
	"path/filepath"
	"time"

	"github.com/gin-gonic/gin"
)

func main() {
	r := gin.Default()

	// Set max multipart memory (default 32MiB)
	r.MaxMultipartMemory = 8 << 20 // 8 MiB

	// v1: single file upload
	r.POST("/upload", func(c *gin.Context) {
		file, err := c.FormFile("file")
		if err != nil {
			c.JSON(http.StatusBadRequest, gin.H{"error": err.Error()})
			return
		}

		// Generate unique filename
		ext := filepath.Ext(file.Filename)
		newName := fmt.Sprintf("%d%s", time.Now().UnixMilli(), ext)

		// Ensure upload directory exists
		os.MkdirAll("./uploads", 0755)

		dst := filepath.Join("./uploads", newName)
		if err := c.SaveUploadedFile(file, dst); err != nil {
			c.JSON(http.StatusInternalServerError, gin.H{"error": "save failed"})
			return
		}

		c.JSON(http.StatusOK, gin.H{
			"filename": newName,
			"size":     file.Size,
			"url":      "/static/" + newName,
		})
	})

	// Serve static files
	r.Static("/static", "./uploads")

	r.Run(":8080")
}
```

### v2：多文件上传 + 类型校验 + 大小限制

```go
package main

import (
	"fmt"
	"net/http"
	"os"
	"path/filepath"
	"strings"
	"time"

	"github.com/gin-gonic/gin"
)

// Allowed MIME types for upload
var allowedTypes = map[string]bool{
	"image/jpeg": true,
	"image/png":  true,
	"image/gif":  true,
	"image/webp": true,
	"application/pdf": true,
}

// Max file size: 10MB
const maxFileSize = 10 << 20

func main() {
	r := gin.Default()
	r.MaxMultipartMemory = 8 << 20

	// Middleware: check content type
	uploadGroup := r.Group("/upload")
	uploadGroup.Use(func(c *gin.Context) {
		if !strings.HasPrefix(c.ContentType(), "multipart/form-data") {
			c.JSON(http.StatusBadRequest, gin.H{"error": "must be multipart/form-data"})
			c.Abort()
			return
		}
		c.Next()
	})

	// Single file upload with validation
	uploadGroup.POST("/single", uploadSingle)
	// Multiple file upload
	uploadGroup.POST("/multi", uploadMulti)

	r.Static("/static", "./uploads")
	r.Run(":8080")
}

func uploadSingle(c *gin.Context) {
	file, err := c.FormFile("file")
	if err != nil {
		c.JSON(http.StatusBadRequest, gin.H{"error": err.Error()})
		return
	}

	// Validate file size
	if file.Size > maxFileSize {
		c.JSON(http.StatusBadRequest, gin.H{"error": "file too large, max 10MB"})
		return
	}

	// Validate file type
	if !allowedTypes[file.Header.Get("Content-Type")] {
		c.JSON(http.StatusBadRequest, gin.H{
			"error": "file type not allowed",
			"got":   file.Header.Get("Content-Type"),
		})
		return
	}

	filename, err := saveFile(file)
	if err != nil {
		c.JSON(http.StatusInternalServerError, gin.H{"error": err.Error()})
		return
	}

	c.JSON(http.StatusOK, gin.H{
		"filename": filename,
		"size":     file.Size,
		"url":      "/static/" + filename,
	})
}

func uploadMulti(c *gin.Context) {
	form, err := c.MultipartForm()
	if err != nil {
		c.JSON(http.StatusBadRequest, gin.H{"error": err.Error()})
		return
	}

	files := form.File["files"]
	var results []map[string]interface{}
	os.MkdirAll("./uploads", 0755)

	for _, file := range files {
		if file.Size > maxFileSize {
			results = append(results, map[string]interface{}{
				"original": file.Filename,
				"error":    "file too large",
			})
			continue
		}

		filename, err := saveFile(file)
		if err != nil {
			results = append(results, map[string]interface{}{
				"original": file.Filename,
				"error":    err.Error(),
			})
			continue
		}

		results = append(results, map[string]interface{}{
			"original": file.Filename,
			"filename": filename,
			"size":     file.Size,
			"url":      "/static/" + filename,
		})
	}

	c.JSON(http.StatusOK, gin.H{
		"total":  len(files),
		"saved":  len(results),
		"files":  results,
	})
}

// saveFile generates a unique name and saves the uploaded file
func saveFile(file *multipart.FileHeader) (string, error) {
	ext := filepath.Ext(file.Filename)
	newName := fmt.Sprintf("%d_%s%s", time.Now().UnixMilli(), randomStr(6), ext)
	os.MkdirAll("./uploads", 0755)
	dst := filepath.Join("./uploads", newName)
	return newName, c.SaveUploadedFile(file, dst)
}
```

### v3：生产级文件上传（按日期存储 + 图片处理）

```go
package main

import (
	"fmt"
	"image"
	"image/jpeg"
	"image/png"
	"log"
	"net/http"
	"os"
	"path/filepath"
	"strings"
	"time"

	"github.com/gin-gonic/gin"
	"github.com/nfnt/resize"
)

const (
	maxUploadSize = 10 << 20 // 10MB
	uploadDir     = "./uploads"
	maxThumbWidth = 300
)

var imageTypes = map[string]bool{
	"image/jpeg": true,
	"image/png":  true,
	"image/gif":  true,
	"image/webp": true,
}

func main() {
	r := gin.Default()
	r.MaxMultipartMemory = 8 << 20

	r.POST("/api/upload", uploadHandler)
	r.POST("/api/upload/batch", batchUploadHandler)

	// Static file server with cache headers
	r.Use(func(c *gin.Context) {
		c.Header("Cache-Control", "public, max-age=86400")
		c.Next()
	})
	r.Static("/static", uploadDir)

	log.Println("Server running on :8080")
	r.Run(":8080")
}

// datePath returns a path like ./uploads/2024/01/15/
func datePath() string {
	now := time.Now()
	p := filepath.Join(uploadDir, now.Format("2006"), now.Format("01"), now.Format("02"))
	os.MkdirAll(p, 0755)
	return p
}

// uniqueName generates a unique filename preserving extension
func uniqueName(original string) string {
	ext := filepath.Ext(original)
	return fmt.Sprintf("%d%s", time.Now().UnixNano(), ext)
}

// validateFile checks size and type
func validateFile(header *multipart.FileHeader) error {
	if header.Size > maxUploadSize {
		return fmt.Errorf("file too large: %d bytes (max %d)", header.Size, maxUploadSize)
	}
	ct := header.Header.Get("Content-Type")
	if !imageTypes[ct] {
		return fmt.Errorf("type %s not allowed", ct)
	}
	return nil
}

// createThumbnail generates a smaller version of the image
func createThumbnail(srcPath, thumbPath string) error {
	file, err := os.Open(srcPath)
	if err != nil {
		return err
	}
	defer file.Close()

	img, _, err := image.Decode(file)
	if err != nil {
		return err
	}

	thumb := resize.Thumbnail(maxThumbWidth, 0, img, resize.Lanczos3)
	out, err := os.Create(thumbPath)
	if err != nil {
		return err
	}
	defer out.Close()

	ext := strings.ToLower(filepath.Ext(srcPath))
	if ext == ".png" {
		return png.Encode(out, thumb)
	}
	return jpeg.Encode(out, thumb, &jpeg.Options{Quality: 85})
}

func uploadHandler(c *gin.Context) {
	file, err := c.FormFile("file")
	if err != nil {
		c.JSON(http.StatusBadRequest, gin.H{"error": "no file provided"})
		return
	}

	if err := validateFile(file); err != nil {
		c.JSON(http.StatusBadRequest, gin.H{"error": err.Error()})
		return
	}

	dir := datePath()
	name := uniqueName(file.Filename)
	fullPath := filepath.Join(dir, name)

	if err := c.SaveUploadedFile(file, fullPath); err != nil {
		c.JSON(http.StatusInternalServerError, gin.H{"error": "save failed"})
		return
	}

	// Generate thumbnail
	thumbName := "thumb_" + name
	thumbPath := filepath.Join(dir, thumbName)
	_ = createThumbnail(fullPath, thumbPath) // non-fatal

	// Build relative URL from uploadDir
	relPath := strings.TrimPrefix(fullPath, uploadDir+"/")

	c.JSON(http.StatusOK, gin.H{
		"url":       "/static/" + relPath,
		"thumbnail": "/static/" + strings.TrimPrefix(thumbPath, uploadDir+"/"),
		"size":      file.Size,
		"filename":  name,
	})
}

func batchUploadHandler(c *gin.Context) {
	form, err := c.MultipartForm()
	if err != nil {
		c.JSON(http.StatusBadRequest, gin.H{"error": err.Error()})
		return
	}

	files := form.File["files"]
	type result struct {
		URL       string `json:"url"`
		Original  string `json:"original"`
		Size      int64  `json:"size"`
		Error     string `json:"error,omitempty"`
	}

	var results []result
	for _, f := range files {
		if err := validateFile(f); err != nil {
			results = append(results, result{Original: f.Filename, Error: err.Error()})
			continue
		}

		dir := datePath()
		name := uniqueName(f.Filename)
		dst := filepath.Join(dir, name)

		if err := c.SaveUploadedFile(f, dst); err != nil {
			results = append(results, result{Original: f.Filename, Error: "save failed"})
			continue
		}

		relPath := strings.TrimPrefix(dst, uploadDir+"/")
		results = append(results, result{
			URL:      "/static/" + relPath,
			Original: f.Filename,
			Size:     f.Size,
		})
	}

	c.JSON(http.StatusOK, gin.H{"results": results})
}
```

## 五、注意事项

| 项目 | 说明 |
|------|------|
| `MaxMultipartMemory` | 内存缓冲区，超出部分写入临时文件，建议 8-32MB |
| 并发写入 | 同一文件名会覆盖，必须生成唯一文件名 |
| 路径穿越 | 永远不要用用户提供的文件名直接拼接路径 |
| MIME 校验 | `Content-Type` 可伪造，生产环境建议读取文件头验证 |
| 临时文件 | `SaveUploadedFile` 后临时文件会被清理，无需手动处理 |
| 目录权限 | `os.MkdirAll` 需要 0755，确保运行用户有写权限 |

## 六、避坑指南

| ❌ 错误做法 | ✅ 正确做法 |
|------------|------------|
| `c.SaveUploadedFile(file, file.Filename)` | 用 UUID/时间戳生成唯一文件名 |
| 不限制上传大小 | 设置 `r.MaxMultipartMemory` 并检查 `file.Size` |
| 只校验文件扩展名 | 校验 MIME 类型 + 文件头魔数 |
| 文件存在 public 目录无任何保护 | 按日期分目录 + 随机文件名 + 权限控制 |
| 前端不压缩图片直接上传 | 后端做缩略图 + 前端预压缩 |

## 八、知识点总结

```
文件上传知识树
├── 核心 API
│   ├── FormFile("key") → 单文件
│   ├── MultipartForm() → 整个表单
│   └── SaveUploadedFile() → 保存
├── 安全
│   ├── 大小限制 (MaxMultipartMemory + Size)
│   ├── 类型校验 (MIME + 魔数)
│   └── 文件名过滤 (唯一名 + 路径过滤)
├── 存储
│   ├── 本地: 按日期分目录
│   └── OSS: 直接流式上传
├── 图片处理
│   ├── 缩略图生成
│   └── 格式转换
└── 静态服务
    ├── Static(path, dir)
    ├── StaticFile(path, file)
    └── StaticFS(path, fs)
```

## 九、举一反三

| 场景 | 方案 | 关键点 |
|------|------|--------|
| 头像上传 | 单文件 + 裁剪 + 压缩 | 固定尺寸输出 |
| 批量图片 | 多文件 + 异步处理 | 用 channel/goroutine 并发生成缩略图 |
| 文档上传 | PDF/Word 校验 + 病毒扫描 | 调用 ClamAV 扫描 |
| 视频上传 | 分片上传 + 进度回调 | tus 协议或自实现 |
| 对象存储 | 流式上传到 S3/OSS | 不落盘，直接 pipe |
