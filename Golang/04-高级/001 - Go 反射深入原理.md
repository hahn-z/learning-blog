## 概念讲解

### 什么是反射（Reflection）？

反射是程序在**运行时**检查自身结构的能力——变量的类型、值、方法、字段，统统可以在运行时动态获取和操作。Go 的反射由 `reflect` 包提供，核心围绕两个类型：

- `reflect.Type`：类型的元信息（名称、种类、字段、方法……）
- `reflect.Value`：运行时的值（可以读取、设置、调用方法）

### 为什么需要反射？

| 场景 | 没有反射 | 有反射 |
|------|---------|--------|
| JSON 序列化 | 为每个类型手写 Marshal | 一个 `json.Marshal` 通吃 |
| ORM 映射 | 为每个 Model 写 CRUD | 结构体 Tag 自动映射字段 |
| 依赖注入 | 手动组装 | 按类型自动注入 |
| 配置解析 | 逐字段赋值 | 结构体 Tag 自动填充 |

### 接口到反射的桥梁

Go 中**每个接口值**内部都是一个二元组 `(type, value)`。当你把一个变量传给 `reflect.ValueOf()` 或 `reflect.TypeOf()` 时，实际上是在做**接口装箱**，然后 reflect 包拆箱拿到内部的 type 和 value。

```
var x int = 42
var i interface{} = x        // 装箱：(type=int, value=42)
v := reflect.ValueOf(i)       // 拆箱：拿到 (Type, Value)
```

## 代码演进

### v1：基础类型检查

```go
package main

import (
    "fmt"
    "reflect"
)

func describe(v interface{}) {
    t := reflect.TypeOf(v)
    val := reflect.ValueOf(v)

    fmt.Printf("Type:  %s\n", t.Name())
    fmt.Printf("Kind:  %s\n", t.Kind())
    fmt.Printf("Value: %v\n", val.Interface())
}

func main() {
    describe(42)
    describe("hello")
    describe([]int{1, 2, 3})
}
```

**执行预览：**
```
Type:  int
Kind:  int
Value: 42
Type:  string
Kind:  string
Value: hello
Type:  
Kind:  slice
Value: [1 2 3]
```

### v2：结构体字段遍历与方法调用

```go
package main

import (
    "fmt"
    "reflect"
)

// User demonstrates struct reflection
type User struct {
    Name string `json:"name" validate:"required"`
    Age  int    `json:"age" validate:"min=0"`
    // unexported field — reflect can read but cannot set
    secret string
}

// Greet is an exported method for reflection demo
func (u User) Greet(greeting string) string {
    return fmt.Sprintf("%s, I'm %s (age %d)", greeting, u.Name, u.Age)
}

// Birthday is a pointer receiver method
func (u *User) Birthday() {
    u.Age++
}

func inspectStruct(u interface{}) {
    t := reflect.TypeOf(u)
    v := reflect.ValueOf(u)

    fmt.Println("=== Fields ===")
    for i := 0; i < t.NumField(); i++ {
        field := t.Field(i)
        fieldVal := v.Field(i)

        fmt.Printf("  %s (%s) = %v  tags=%s  exported=%v\n",
            field.Name, field.Type, fieldVal.Interface(),
            field.Tag, field.PkgPath == "")
    }

    fmt.Println("\n=== Methods ===")
    for i := 0; i < t.NumMethod(); i++ {
        m := t.Method(i)
        fmt.Printf("  %s(%v) -> %v\n", m.Name, m.Type, m.Type.Out(0))
    }
}

func callMethod(u interface{}, methodName string, args ...interface{}) {
    v := reflect.ValueOf(u)
    m := v.MethodByName(methodName)
    if !m.IsValid() {
        fmt.Printf("method %s not found\n", methodName)
        return
    }

    // Convert args to reflect.Value slice
    in := make([]reflect.Value, len(args))
    for i, arg := range args {
        in[i] = reflect.ValueOf(arg)
    }

    results := m.Call(in)
    for _, r := range results {
        fmt.Println("Result:", r.Interface())
    }
}

func main() {
    u := User{Name: "Alice", Age: 30, secret: "hidden"}

    inspectStruct(u)
    fmt.Println()
    callMethod(u, "Greet", "Hi")
}
```

**执行预览：**
```
=== Fields ===
  Name (string) = Alice  tags=json:"name" validate:"required"  exported=true
  Age (int) = 30  tags=json:"age" validate:"min=0"  exported=true
  secret (string) = hidden  tags=  exported=false

=== Methods ===
  Greet(func(main.User, string) string) -> string

Result: Hi, I'm Alice (age 30)
```

### v3：通用结构体验证器（实战）

```go
package main

import (
    "fmt"
    "reflect"
    "strconv"
    "strings"
)

// Validator uses reflect to check struct tags
type Validator struct{}

// Validate checks fields against "validate" tags
func (vd *Validator) Validate(s interface{}) []error {
    v := reflect.ValueOf(s)
    
    // Dereference pointer if needed
    if v.Kind() == reflect.Ptr {
        v = v.Elem()
    }
    if v.Kind() != reflect.Struct {
        return []error{fmt.Errorf("Validate expects a struct, got %s", v.Kind())}
    }

    t := v.Type()
    var errs []error

    for i := 0; i < t.NumField(); i++ {
        field := t.Field(i)
        fieldVal := v.Field(i)

        tag, ok := field.Tag.Lookup("validate")
        if !ok {
            continue
        }

        rules := strings.Split(tag, ",")
        for _, rule := range rules {
            parts := strings.SplitN(rule, "=", 2)
            ruleName := parts[0]
            var ruleVal string
            if len(parts) == 2 {
                ruleVal = parts[1]
            }

            switch ruleName {
            case "required":
                if fieldVal.IsZero() {
                    errs = append(errs, fmt.Errorf("field %s is required", field.Name))
                }
            case "min":
                min, _ := strconv.Atoi(ruleVal)
                if fieldVal.Kind() == reflect.Int && int(fieldVal.Int()) < min {
                    errs = append(errs, fmt.Errorf("field %s must be >= %d", field.Name, min))
                }
            case "max":
                max, _ := strconv.Atoi(ruleVal)
                if fieldVal.Kind() == reflect.Int && int(fieldVal.Int()) > max {
                    errs = append(errs, fmt.Errorf("field %s must be <= %d", field.Name, max))
                }
            case "len":
                length, _ := strconv.Atoi(ruleVal)
                if fieldVal.Kind() == reflect.String && fieldVal.Len() < length {
                    errs = append(errs, fmt.Errorf("field %s must be at least %d chars", field.Name, length))
                }
            }
        }
    }
    return errs
}

// Form is the struct to validate
type Form struct {
    Name string `validate:"required,len=2"`
    Age  int    `validate:"min=0,max=150"`
    City string `validate:"required"`
}

func main() {
    vd := &Validator{}

    // Valid case
    ok := Form{Name: "Bob", Age: 25, City: "Shanghai"}
    if errs := vd.Validate(ok); len(errs) > 0 {
        for _, e := range errs {
            fmt.Println("ERR:", e)
        }
    } else {
        fmt.Println("✅ Valid form")
    }

    // Invalid case
    bad := Form{Name: "", Age: -1, City: ""}
    for _, e := range vd.Validate(bad) {
        fmt.Println("❌", e)
    }
}
```

**执行预览：**
```
✅ Valid form
❌ field Name is required
❌ field Age must be >= 0
❌ field City is required
```

## 注意事项

| 要点 | 说明 |
|------|------|
| 性能开销 | 反射比直接调用慢 10-100 倍，热路径避免使用 |
| 类型安全 | 编译器无法检查反射代码，错误只会在运行时暴露 |
| 可导出性 | 只有导出字段（大写开头）才能被 `Set()` 修改 |
| 指针传递 | 要修改值，必须传指针 `reflect.ValueOf(&x)` |
| 代码可读性 | 过度使用反射让代码难懂，适度使用 |
| 递归嵌套 | 处理嵌套结构体时注意 `Kind() == reflect.Ptr` 解引用 |

---

## 避坑指南

| ❌ 错误做法 | ✅ 正确做法 |
|------------|-----------|
| `reflect.ValueOf(x).SetInt(1)` 直接改值 | `reflect.ValueOf(&x).Elem().SetInt(1)` 传指针 |
| 忽略 `CanSet()` 检查 | 设置前先检查 `v.CanSet()` |
| 用反射做简单类型判断 | 用 type switch：`switch v := x.(type)` |
| 对未导出字段调用 `Set()` | 只操作导出字段，或使用 `unsafe`（不推荐） |
| 反射结果不检查 `IsValid()` | 始终检查 `IsValid()` 和 `IsZero()` |
| `MethodByName` 拼写错误无编译提示 | 用常量或代码生成替代硬编码方法名 |

## 知识点总结

```
Go 反射
├── 核心类型
│   ├── reflect.Type   → 类型元信息
│   └── reflect.Value  → 运行时值
├── 三大定律
│   ├── 1. 接口值 → 反射对象
│   ├── 2. 反射对象 → 接口值
│   └── 3. 修改反射对象需可设置性（指针）
├── 常用操作
│   ├── Kind()         → 底层类型种类
│   ├── NumField()     → 结构体字段数
│   ├── Field(i)       → 按索引取字段
│   ├── MethodByName() → 按名取方法
│   ├── Call()         → 调用函数/方法
│   └── Elem()         → 解引用指针/slice/接口
├── 实战场景
│   ├── JSON/XML 序列化
│   ├── ORM 字段映射
│   ├── 结构体 Tag 解析
│   ├── 依赖注入
│   └── 通用验证器
└── 注意事项
    ├── 性能差（避免热路径）
    ├── 无编译时类型安全
    └── 仅导出字段可 Set
```

## 举一反三

| 技术点 | 本文学到 | 延伸应用 |
|--------|---------|---------|
| Type / Value | 运行时类型检查 | 实现 `fmt.Sprintf` 的通用格式化 |
