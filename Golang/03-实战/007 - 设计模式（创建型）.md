# 设计模式（创建型） ⭐⭐⭐中级

> **难度标注：** ⭐⭐⭐ 中级 | 预计阅读时间：20分钟 | 前置知识：Go接口、结构体、并发基础

## 📖 概念讲解

创建型设计模式关注**"如何创建对象"**，核心思想是将对象的创建与使用分离，让系统不依赖于具体类的实例化方式。Go语言没有类和继承，但通过**接口组合**和**函数一等公民**的特性，可以优雅地实现这些模式。

本章覆盖5种创建型模式：**单例(Singleton)、工厂方法(Factory Method)、抽象工厂(Abstract Factory)、建造者(Builder)、原型(Prototype)**。

```
创建型设计模式脑图
├── Singleton 单例
│   ├── sync.Once 保证唯一
│   ├── 懒加载 vs 饿加载
│   └── 并发安全
├── Factory Method 工厂方法
│   ├── 简单工厂（静态工厂）
│   ├── 接口多态
│   └── 配置驱动创建
├── Abstract Factory 抽象工厂
│   ├── 产品族概念
│   ├── 接口组合
│   └── 数据库驱动切换
├── Builder 建造者
│   ├── 链式调用
│   ├── 参数校验
│   └── 可选参数处理
└── Prototype 原型
    ├── 深拷贝 vs 浅拷贝
    ├── JSON序列化克隆
    └── 原型管理器
```

## 💻 完整Go代码

### v1 基础版本：简单工厂 + 基础单例

```go
// v1_basic.go - Simple factory and basic singleton
package main

import "fmt"

// ===== Simple Factory =====

// Animal interface defines behavior
type Animal interface {
    Speak() string
}

type Dog struct{ Name string }
func (d *Dog) Speak() string { return d.Name + ": Woof!" }

type Cat struct{ Name string }
func (c *Cat) Speak() string { return c.Name + ": Meow!" }

// Simple factory function
func NewAnimal(kind, name string) Animal {
    switch kind {
    case "dog":
        return &Dog{Name: name}
    case "cat":
        return &Cat{Name: name}
    default:
        return nil
    }
}

// ===== Basic Singleton (NOT concurrent-safe) =====

type Database struct {
    connection string
}

var dbInstance *Database

func GetDatabase() *Database {
    if dbInstance == nil {
        dbInstance = &Database{connection: "mysql://localhost:3306"}
    }
    return dbInstance
}

func main() {
    // Factory usage
    animals := []Animal{
        NewAnimal("dog", "Buddy"),
        NewAnimal("cat", "Kitty"),
    }
    for _, a := range animals {
        fmt.Println(a.Speak())
    }

    // Singleton usage
    db1 := GetDatabase()
    db2 := GetDatabase()
    fmt.Printf("Same instance: %v\n", db1 == db2)
}
```

### v2 进阶版本：完整5种创建型模式

```go
// v2_patterns.go - All 5 creational patterns with Go idioms
package main

import (
    "encoding/json"
    "fmt"
    "sync"
)

// ===== 1. Singleton (concurrent-safe with sync.Once) =====

type Config struct {
    AppName string `json:"app_name"`
    Port    int    `json:"port"`
    Debug   bool   `json:"debug"`
}

var (
    configInstance *Config
    configOnce     sync.Once
)

// GetConfig returns singleton config, thread-safe
func GetConfig() *Config {
    configOnce.Do(func() {
        configInstance = &Config{
            AppName: "MyApp",
            Port:    8080,
            Debug:   false,
        }
        fmt.Println("Config initialized (once)")
    })
    return configInstance
}

// ===== 2. Factory Method =====

// Storage interface for different backends
type Storage interface {
    Save(key string, data []byte) error
    Load(key string) ([]byte, error)
    String() string
}

type MemoryStorage struct {
    data map[string][]byte
}

func NewMemoryStorage() *MemoryStorage {
    return &MemoryStorage{data: make(map[string][]byte)}
}

func (m *MemoryStorage) Save(key string, data []byte) error {
    m.data[key] = data
    return nil
}

func (m *MemoryStorage) Load(key string) ([]byte, error) {
    return m.data[key], nil
}

func (m *MemoryStorage) String() string { return "MemoryStorage" }

type FileStorage struct {
    baseDir string
}

func NewFileStorage(dir string) *FileStorage {
    return &FileStorage{baseDir: dir}
}

func (f *FileStorage) Save(key string, data []byte) error {
    fmt.Printf("Saved %s to %s\n", key, f.baseDir)
    return nil
}

func (f *FileStorage) Load(key string) ([]byte, error) {
    return []byte("data"), nil
}

func (f *FileStorage) String() string { return "FileStorage:" + f.baseDir }

// Factory function - creates storage by type
func NewStorage(storageType string) Storage {
    switch storageType {
    case "memory":
        return NewMemoryStorage()
    case "file":
        return NewFileStorage("/tmp/data")
    default:
        return NewMemoryStorage() // Default fallback
    }
}

// ===== 3. Abstract Factory =====

// UIFactory creates a family of UI components
type Button interface {
    Render() string
}

type TextBox interface {
    Render() string
}

type UIFactory interface {
    CreateButton() Button
    CreateTextBox() TextBox
}

// Windows family
type WinButton struct{}
func (b *WinButton) Render() string { return "[Windows Button]" }

type WinTextBox struct{}
func (t *WinTextBox) Render() string { return "[Windows TextBox]" }

type WindowsFactory struct{}

func (f *WindowsFactory) CreateButton() Button  { return &WinButton{} }
func (f *WindowsFactory) CreateTextBox() TextBox { return &WinTextBox{} }

// Linux family
type LinuxButton struct{}
func (b *LinuxButton) Render() string { return "<Linux Button>" }

type LinuxTextBox struct{}
func (t *LinuxTextBox) Render() string { return "<Linux TextBox>" }

type LinuxFactory struct{}

func (f *LinuxFactory) CreateButton() Button  { return &LinuxButton{} }
func (f *LinuxFactory) CreateTextBox() TextBox { return &LinuxTextBox{} }

// Factory selector
func GetUIFactory(platform string) UIFactory {
    if platform == "linux" {
        return &LinuxFactory{}
    }
    return &WindowsFactory{}
}

// ===== 4. Builder =====

// ServerConfig builds complex server config step by step
type ServerConfig struct {
    Host      string
    Port      int
    Timeout   int
    TLS       bool
    CertFile  string
    MaxConn   int
    LogLevel  string
}

// ServerBuilder provides fluent API for building ServerConfig
type ServerBuilder struct {
    config *ServerConfig
}

func NewServerBuilder() *ServerBuilder {
    return &ServerBuilder{
        config: &ServerConfig{
            Host:    "localhost",
            Port:    8080,
            Timeout: 30,
            MaxConn: 100,
            LogLevel: "info",
        },
    }
}

func (b *ServerBuilder) WithHost(host string) *ServerBuilder {
    b.config.Host = host
    return b
}

func (b *ServerBuilder) WithPort(port int) *ServerBuilder {
    b.config.Port = port
    return b
}

func (b *ServerBuilder) WithTLS(certFile string) *ServerBuilder {
    b.config.TLS = true
    b.config.CertFile = certFile
    return b
}

func (b *ServerBuilder) WithMaxConn(n int) *ServerBuilder {
    b.config.MaxConn = n
    return b
}

// Build validates and returns the config
func (b *ServerBuilder) Build() (*ServerConfig, error) {
    if b.config.Port < 1 || b.config.Port > 65535 {
        return nil, fmt.Errorf("invalid port: %d", b.config.Port)
    }
    if b.config.TLS && b.config.CertFile == "" {
        return nil, fmt.Errorf("TLS requires cert file")
    }
    return b.config, nil
}

// ===== 5. Prototype =====

// Document supports cloning
type Document struct {
    Title   string
    Content string
    Tags    []string
}

// Clone creates a deep copy via JSON serialization
func (d *Document) Clone() *Document {
    data, _ := json.Marshal(d)
    clone := &Document{}
    json.Unmarshal(data, clone)
    return clone
}

// PrototypeManager manages reusable prototypes
type PrototypeManager struct {
    prototypes map[string]*Document
}

func NewPrototypeManager() *PrototypeManager {
    return &PrototypeManager{prototypes: make(map[string]*Document)}
}

func (m *PrototypeManager) Register(name string, doc *Document) {
    m.prototypes[name] = doc
}

func (m *PrototypeManager) Create(name string) *Document {
    if proto, ok := m.prototypes[name]; ok {
        return proto.Clone()
    }
    return nil
}

// ===== Demo =====
func main() {
    fmt.Println("=== 1. Singleton ===")
    c1 := GetConfig()
    c2 := GetConfig()
    fmt.Printf("Same: %v, Port: %d\n", c1 == c2, c1.Port)

    fmt.Println("\n=== 2. Factory Method ===")
    s := NewStorage("memory")
    s.Save("key1", []byte("hello"))
    fmt.Println("Storage:", s)

    fmt.Println("\n=== 3. Abstract Factory ===")
    factory := GetUIFactory("linux")
    btn := factory.CreateButton()
    txt := factory.CreateTextBox()
    fmt.Println(btn.Render(), txt.Render())

    fmt.Println("\n=== 4. Builder ===")
    cfg, err := NewServerBuilder().
        WithHost("0.0.0.0").
        WithPort(443).
        WithTLS("/certs/server.pem").
        WithMaxConn(1000).
        Build()
    if err != nil {
        panic(err)
    }
    fmt.Printf("Config: %+v\n", cfg)

    fmt.Println("\n=== 5. Prototype ===")
    mgr := NewPrototypeManager()
    mgr.Register("report", &Document{Title: "Monthly Report", Tags: []string{"finance", "monthly"}})
    doc := mgr.Create("report")
    doc.Title = "April Report" // Modify clone
    fmt.Printf("Clone: %+v\n", doc)
}
```

### v3 生产级版本：泛型工厂 + 配置驱动

```go
// v3_production.go - Generic factory with config-driven creation (Go 1.18+)
package main

import (
    "encoding/json"
    "fmt"
    "os"
    "sync"
)

// ===== Generic Factory with Go Generics =====

// Factory registry using generics
type Factory[T any] struct {
    builders map[string]func() T
    mu       sync.RWMutex
}

func NewFactory[T any]() *Factory[T] {
    return &Factory[T]{
        builders: make(map[string]func() T),
    }
}

func (f *Factory[T]) Register(name string, builder func() T) {
    f.mu.Lock()
    defer f.mu.Unlock()
    f.builders[name] = builder
}

func (f *Factory[T]) Create(name string) (T, error) {
    f.mu.RLock()
    builder, ok := f.builders[name]
    f.mu.RUnlock()
    if !ok {
        var zero T
        return zero, fmt.Errorf("builder not found: %s", name)
    }
    return builder(), nil
}

// ===== Config-driven creation =====

type CacheConfig struct {
    Type string `json:"type"` // "memory" | "redis"
    DSN  string `json:"dsn"`
    TTL  int    `json:"ttl"`
}

type Cache interface {
    Get(key string) (string, bool)
    Set(key, value string)
}

type MemoryCache struct {
    data map[string]string
}

func (m *MemoryCache) Get(key string) (string, bool) {
    v, ok := m.data[key]
    return v, ok
}

func (m *MemoryCache) Set(key, value string) { m.data[key] = value }

type RedisCache struct {
    dsn string
}

func (r *RedisCache) Get(key string) (string, bool) { return "", false }
func (r *RedisCache) Set(key, value string)          {}

// Setup factory with config file
func setupFactory() *Factory[Cache] {
    f := NewFactory[Cache]()
    f.Register("memory", func() Cache {
        return &MemoryCache{data: make(map[string]string)}
    })
    f.Register("redis", func() Cache {
        return &RedisCache{dsn: "localhost:6379"}
    })
    return f
}

func main() {
    // Load config
    config := CacheConfig{Type: "memory", TTL: 300}

    // Create via factory
    factory := setupFactory()
    cache, err := factory.Create(config.Type)
    if err != nil {
        fmt.Println("Error:", err)
        os.Exit(1)
    }

    cache.Set("user:1", "Alice")
    val, ok := cache.Get("user:1")
    fmt.Printf("Cache hit: %v = %s\n", ok, val)

    // Demonstrate generic factory with different type
    animalFactory := NewFactory[string]()
    animalFactory.Register("dog", func() string { return "Woof!" })
    animalFactory.Register("cat", func() string { return "Meow!" })

    sound, _ := animalFactory.Create("dog")
    fmt.Println("Dog says:", sound)
}
```

## ⚠️ 注意事项

| 事项 | 说明 | 严重度 |
|------|------|--------|
| sync.Once不可重置 | Once.Do只执行一次，测试时需用不同实例 | ⚠️ 中 |
| 工厂返回接口 | 始终返回接口类型，不要返回具体类型 | ⚠️ 中 |
| Builder校验 | Build()必须校验参数合法性，别信任调用者 | 🔴 高 |
| Prototype深拷贝 | slice/map是指针类型，浅拷贝会共享底层数据 | 🔴 高 |
| 过度设计 | 不是每个场景都需要工厂模式，简单new就够了 | ⚠️ 中 |

## 🚫→✅ 避坑指南

| ❌ 错误做法 | ✅ 正确做法 |
|------------|------------|
| 手写单例加锁 `if instance == nil { mu.Lock(); ... }` | 使用 `sync.Once`，简洁且无竞态 |
| 工厂函数返回具体类型 `*Dog` | 返回接口类型 `Animal`，保持灵活性 |
| Builder不用指针接收者 | Builder方法用 `*Builder` 指针接收者支持链式调用 |
| 原型模式浅拷贝slice/map | 用 `json.Marshal/Unmarshal` 或手动深拷贝 |
| 一个工厂管所有类型 | 按产品族拆分工厂，符合单一职责 |
| 单例存储可变状态不加锁 | 单例内部的可变数据必须加锁保护 |

## 🌳 知识点总结

```
创建型设计模式
├── Singleton 单例
│   ├── sync.Once → 标准实现
│   ├── init() → 包级初始化
│   └── 懒加载 → 按需创建
├── Factory Method 工厂方法
│   ├── 简单工厂 → switch/if
│   ├── 工厂接口 → 策略模式配合
│   └── 泛型工厂 → Go 1.18+
├── Abstract Factory 抽象工厂
│   ├── 产品族 → 一组相关对象
│   └── 工厂选择 → 运行时切换
├── Builder 建造者
│   ├── 链式调用 → return *Builder
│   ├── 参数校验 → Build() 校验
│   └── 函数选项 → WithXxx 模式
└── Prototype 原型
    ├── 深拷贝 → JSON序列化
    ├── 浅拷贝 → 结构体值拷贝
    └── 管理器 → 注册+克隆
```

## 📈 代码演进总结

| 版本 | 核心改进 | 特点 | 适用场景 |
|------|---------|------|---------|
| v1 基础版 | 简单工厂+基础单例 | 直观易懂，非并发安全 | 学习、原型 |
| v2 完整版 | 5种模式+sync.Once+Builder | Go惯用写法，生产可用 | 实际项目 |

> **设计原则：** 在Go中，优先用接口组合代替继承，用函数代替策略类，用闭包代替模板方法。Go的设计模式比Java/C++简洁得多，别过度设计。
