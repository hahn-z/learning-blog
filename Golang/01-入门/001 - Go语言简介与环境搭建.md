## 一句话定义

Go（Golang）是 Google 于 2009 年发布的开源编程语言，专为构建简洁、高效、可靠的软件而设计。

**现实类比：** 如果 C 语言是一辆手动挡跑车（快但难开），Python 是一辆自动驾驶出租车（方便但慢），那 Go 就是一辆带自动挡的性能车——既有速度，又不难驾驭。

**技术场景：** 云原生基础设施（Docker、Kubernetes）、微服务 API、CLI 工具、网络编程、DevOps 工具链。

## Go 语言核心特点

| 特点 | 说明 |
|------|------|
| **编译速度快** | 大型项目秒级编译，告别等待 |
| **静态类型** | 编译期捕获错误，运行时更安全 |
| **内置并发** | goroutine + channel，原生支持高并发 |
| **垃圾回收** | 自动内存管理，无需手动 free |
| **单一二进制** | 编译产出一个可执行文件，无外部依赖 |
| **跨平台** | 支持 Linux / macOS / Windows / ARM 等 |
| **简洁语法** | 25 个关键字，上手快 |
| **强大标准库** | HTTP、JSON、加密、测试等开箱即用 |

## 安装 Go

### 1. 下载安装

访问 [https://go.dev/dl/](https://go.dev/dl/) 下载对应平台安装包。

**Linux 安装示例：**

```bash
# Download Go (adjust version as needed)
wget https://go.dev/dl/go1.22.0.linux-amd64.tar.gz

# Remove any previous Go installation
sudo rm -rf /usr/local/go

# Extract to /usr/local
sudo tar -C /usr/local -xzf go1.22.0.linux-amd64.tar.gz

# Verify installation
/usr/local/go/bin/go version
```

**macOS 安装：**

```bash
# Using Homebrew (recommended)
brew install go

# Or download the .pkg installer from go.dev/dl
```

**Windows 安装：**

下载 `.msi` 安装包，双击安装，默认路径 `C:\Go`。

### 2. 配置环境变量

```bash
# Add Go binary to PATH (add to ~/.bashrc or ~/.zshrc)
export PATH=$PATH:/usr/local/go/bin

# Verify
go version
# Output: go version go1.22.0 linux/amd64
```

### 3. 关键环境变量

```bash
# View all Go environment variables
go env

# Key variables:
# GOROOT  - Go installation directory (usually /usr/local/go)
# GOPATH  - Go workspace directory (default: $HOME/go)
# GOBIN   - Binary output directory (default: $GOPATH/bin)
# GOOS    - Target operating system (linux, darwin, windows)
# GOARCH  - Target architecture (amd64, arm64)
```

## Go Modules（模块管理）

Go Modules 是 Go 1.16+ 的官方依赖管理方案。

```bash
# Create a new project
mkdir myproject && cd myproject

# Initialize a Go module
go mod init github.com/yourname/myproject
# Creates go.mod file

# Add dependencies (auto-downloaded)
go get github.com/gin-gonic/gin

# Tidy up unused dependencies
go mod tidy
```

**go.mod 文件示例：**

```go
module github.com/yourname/myproject

go 1.22

require github.com/gin-gonic/gin v1.9.1
```

## 第一个 Go 程序

```go
// main.go - Your first Go program
package main

import "fmt"

func main() {
    // Print a greeting message
    fmt.Println("Hello, Go!")

    // Print Go version info
    fmt.Printf("Running Go version: %s\n", "1.22+")
}
```

**运行方式：**

```bash
# Run directly (compile + execute, no binary left behind)
go run main.go

# Build to binary
go build -o myapp main.go

# Run the binary
./myapp

# Install to $GOPATH/bin
go install
```

## 注意事项

| 事项 | 说明 |
|------|------|
| Go 版本 | 建议使用 Go 1.21+，体验最新特性 |
| GOPATH vs Modules | 新项目一律用 Go Modules，不要用 GOPATH 模式 |
| 文件命名 | 文件名使用 `snake_case`，如 `hello_world.go` |
| 入口文件 | 可执行程序的入口必须是 `package main` + `func main()` |
| 代理加速 | 国内用户设置 `GOPROXY=https://goproxy.cn,direct` |
| 格式化 | 使用 `gofmt` 或 `go fmt` 格式化代码，不要自定义风格 |

## 避坑指南

| ❌ 错误做法 | ✅ 正确做法 |
|------------|-----------|
| 从 APT/YUM 安装 Go（版本太旧） | 从 go.dev 下载最新版或用 Homebrew |
| 不设置 GOPATH | Go 1.16+ 默认即可，了解但不需手动设置 |
| 用 GOPATH 模式管理依赖 | 新项目必须用 `go mod init` |
| 不用 gofmt 格式化 | 所有代码都应 gofmt，团队统一风格 |
| 把项目放在 `$GOROOT/src` 下 | 项目放任意位置，用 Go Modules 管理 |

## 代码演进

### v1: 最简单的程序

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}
```

### v2: 加入版本信息和模块

```go
// main.go - with module info
package main

import (
    "fmt"
    "runtime"
)

func main() {
    fmt.Println("Hello, Go!")
    fmt.Printf("Go version: %s\n", runtime.Version())
    fmt.Printf("OS/Arch: %s/%s\n", runtime.GOOS, runtime.GOARCH)
}
```

### v3: 加入命令行参数处理

```go
// main.go - with command line arguments
package main

import (
    "flag"
    "fmt"
    "os"
    "runtime"
)

func main() {
    // Define command line flags
    name := flag.String("name", "Go", "Your name")
    version := flag.Bool("version", false, "Show version info")
    flag.Parse()

    if *version {
        fmt.Printf("Go version: %s\n", runtime.Version())
        fmt.Printf("OS/Arch: %s/%s\n", runtime.GOOS, runtime.GOARCH)
        os.Exit(0)
    }

    fmt.Printf("Hello, %s!\n", *name)
    fmt.Printf("Remaining args: %v\n", flag.Args())
}
```

```bash
# v3 运行示例
$ go run main.go -name 世界 -version false 剩余参数
Hello, 世界!
Remaining args: [剩余参数]
```
