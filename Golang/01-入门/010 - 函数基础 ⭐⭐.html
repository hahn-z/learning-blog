概念讲解

函数是 Go 程序的基本构建块。Go 的函数设计简洁而强大：一等公民、多返回值、defer 机制，这些都是 Go 的核心特性。

核心要点：

  • 函数定义：func name(params) returnType { body }
  • 参数传递：值传递（副本）和指针传递（共享）
  • 返回值：单返回值、命名返回值
  • defer：延迟执行，函数返回前触发
  • 函数是一等公民：可以赋值给变量、作为参数传递

    完整 Go 代码

    v1：函数定义与基本调用

    package main

import "fmt"

// Basic function with parameters and return value
func add(a int, b int) int {
	return a + b
}

// Multiple parameters of the same type can share the type declaration
func multiply(a, b int) int {
	return a * b
}

// Function with no return value
func greet(name string) {
	fmt.Printf("Hello, %s!\n", name)
}

// Function with no parameters
func getTime() string {
	return "2024-01-01 00:00:00"
}

func main() {
	// Basic function calls
	sum := add(3, 5)
	fmt.Println("3 + 5 =", sum)

	product := multiply(4, 6)
	fmt.Println("4 * 6 =", product)

	greet("Gopher")
	fmt.Println("Time:", getTime())
}

    执行预览：

    3 + 5 = 8
4 * 6 = 24
Hello, Gopher!
Time: 2024-01-01 00:00:00

    v2：参数传递 —— 值 vs 指针

    package main

import "fmt"

// Value passing: changes inside function don't affect the original
func doubleVal(n int) {
	n = n * 2
	fmt.Println("inside doubleVal:", n)
}

// Pointer passing: can modify the original value
func doublePtr(n *int) {
	*n = *n * 2
	fmt.Println("inside doublePtr:", *n)
}

// Swap two values using pointers
func swap(a, b *int) {
	*a, *b = *b, *a
}

func main() {
	// --- Value passing demo ---
	x := 10
	doubleVal(x)
	fmt.Println("after doubleVal:", x) // still 10

	// --- Pointer passing demo ---
	y := 10
	doublePtr(&y)
	fmt.Println("after doublePtr:", y) // now 20

	// --- Swap demo ---
	a, b := 1, 2
	fmt.Printf("before swap: a=%d, b=%d\n", a, b)
	swap(&a, &b)
	fmt.Printf("after swap:  a=%d, b=%d\n", a, b)
}

    执行预览：

    inside doubleVal: 20
after doubleVal: 10
inside doublePtr: 20
after doublePtr: 20
before swap: a=1, b=2
after swap:  a=2, b=1

    v3：命名返回值、defer 与函数作为值

    package main

import "fmt"

// Named return value: "result" is initialized to 0
func factorial(n int) (result int) {
	if n <= 1 {
		return 1 // bare return, returns result
	}
	for i := 2; i <= n; i++ {
		result *= i
	}
	return // bare return
}

// Defer basics: deferred calls execute before function returns
func demoDefer() {
	fmt.Println("step 1")
	defer fmt.Println("deferred: cleanup")
	fmt.Println("step 2")
	fmt.Println("step 3")
}

// Function as a value
func compute(a, b int, op func(int, int) int) int {
	return op(a, b)
}

func main() {
	// --- Named return ---
	fmt.Println("5! =", factorial(5))

	// --- Defer ---
	fmt.Println("--- defer demo ---")
	demoDefer()

	// --- Function as value ---
	add := func(a, b int) int { return a + b }
	sub := func(a, b int) int { return a - b }

	fmt.Println("compute(10,3, add) =", compute(10, 3, add))
	fmt.Println("compute(10,3, sub) =", compute(10, 3, sub))

	// --- Function variable ---
	greet := func(name string) string {
		return "Hi, " + name
	}
	fmt.Println(greet("Gopher"))
}

    执行预览：

    5! = 120
--- defer demo ---
step 1
step 2
step 3
deferred: cleanup
compute(10,3, add) = 13
compute(10,3, sub) = 7
Hi, Gopher

    注意事项

    项目 说明
    参数类型缩写 a int, b int 可简写为 a, b int，但 a int, b string 不可
    值传递本质 Go 只有值传递，指针传递本质上是传递地址值的副本
    命名返回值 建议仅在短函数中使用，长函数用显式 return 更清晰
    defer 时序 defer 在 return 语句之后、函数返回之前执行
    函数签名 函数作为参数时，签名必须完全匹配（参数类型+返回类型）

    避坑指南

    ❌ 错误写法 ✅ 正确写法 说明
    func add(a, b) func add(a, b int) int 参数和返回值类型不能省略
    func swap(a, b int) 内直接交换 func swap(a, b *int) 值传递无法修改原值
    defer 中的参数立即求值被忽略 记住 defer 的参数在声明时就确定了 defer fmt.Println(n) 中的 n 是声明时的值
    命名返回值在长函数中滥用 短函数用命名返回值，长函数显式 return 裸 return 降低可读性
    函数变量类型推断失败 显式声明函数类型 var fn func(int) int 复杂签名建议显式声明

    练习题

    🟢 入门：求最大值

    编写函数 max(a, b int) int，返回两个整数中的较大值。

    func max(a, b int) int {
    if a > b {
        return a
    }
    return b
}

    🟡 进阶：指针计数器

    编写函数 increment(counter *int, delta int)，通过指针修改计数器值。

    func increment(counter *int, delta int) {
    *counter += delta
}

    🔴 挑战：函数式过滤器

    编写 filter(nums []int, fn func(int) bool) []int，返回满足条件的元素。

    func filter(nums []int, fn func(int) bool) []int {
    var result []int
    for _, n := range nums {
        if fn(n) {
            result = append(result, n)
        }
    }
    return result
}

    知识点总结

    Go 函数基础
├── 定义: func name(params) returnType { body }
├── 参数
│   ├── 值传递: 创建副本，不影响原值
│   ├── 指针传递: 传地址，可修改原值
│   └── 类型共享: a, b int
├── 返回值
│   ├── 单返回值: func f() int
│   ├── 命名返回值: func f() (r int)
│   └── 裸 return: 直接 return
├── defer
│   ├── 延迟执行
│   └── 参数立即求值
└── 一等公民
    ├── 赋值: fn := func() { }
    ├── 参数: func do(f func())
    └── 返回值: func factory() func()

    举一反三

    概念 Go 实现 类比语言
    函数定义 func add(a, b int) int Python: def add(a, b):
    指针参数 func f(p *int) C: void f(int *p)
    匿名函数 fn := func() { } JS: const fn = () => {}
    defer defer cleanup() Python: with / finally
    函数作参数 func do(f func()) JS: 高阶函数
