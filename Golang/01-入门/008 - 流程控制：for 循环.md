# 008 - 流程控制：for 循环 ⭐⭐

> 难度：⭐⭐ | 预计学习时间：20 分钟

## 一、概念讲解

Go 只有一个循环关键字：`for`。没有 `while`、`do-while`、`foreach`，但 `for` 能胜任所有场景。

核心形式：
1. **经典三段式**：`for init; condition; post { }`
2. **while 模式**：`for condition { }`
3. **无限循环**：`for { }`
4. **range 迭代**：`for i, v := range collection { }`

## 二、实时脑图

```
for 循环
├── 经典 for
│   └── for i := 0; i < n; i++ {}
├── while 模式
│   └── for condition {}
├── 无限循环
│   └── for {} + break
├── for-range
│   ├── 数组/切片 → index, value
│   ├── map → key, value
│   ├── string → index, rune
│   └── channel → value
├── 控制语句
│   ├── break (跳出循环)
│   ├── continue (跳过本轮)
│   └── 标签 (label) + break/continue
└── 性能优化
    ├── 避免在循环内分配
    └── for-range 会复制值
```

## 三、完整 Go 代码

```go
package main

import "fmt"

func main() {
	// --- 1. Classic for loop ---
	fmt.Println("=== Classic for ===")
	for i := 0; i < 5; i++ {
		fmt.Print(i, " ")
	}
	fmt.Println()

	// --- 2. While-style loop ---
	fmt.Println("=== While-style ===")
	n := 1
	for n < 100 {
		n *= 2
	}
	fmt.Println("n =", n)

	// --- 3. Infinite loop with break ---
	fmt.Println("=== Infinite loop ===")
	count := 0
	for {
		count++
		if count >= 3 {
			break
		}
	}
	fmt.Println("broke at", count)

	// --- 4. for-range over slice ---
	fmt.Println("=== Range slice ===")
	fruits := []string{"apple", "banana", "cherry"}
	for i, fruit := range fruits {
		fmt.Printf("[%d] %s\n", i, fruit)
	}

	// --- 5. for-range over map ---
	fmt.Println("=== Range map ===")
	colors := map[string]string{"red": "#f00", "green": "#0f0"}
	for k, v := range colors {
		fmt.Printf("%s = %s\n", k, v)
	}

	// --- 6. for-range over string (runes!) ---
	fmt.Println("=== Range string ===")
	for i, ch := range "Go语言" {
		fmt.Printf("[%d] %c\n", i, ch)
	}

	// --- 7. break with label ---
	fmt.Println("=== Label break ===")
outer:
	for i := 0; i < 3; i++ {
		for j := 0; j < 3; j++ {
			if i == 1 && j == 1 {
				break outer // breaks both loops
			}
			fmt.Printf("(%d,%d) ", i, j)
		}
	}
	fmt.Println()

	// --- 8. continue with label ---
	fmt.Println("=== Label continue ===")
	for i := 0; i < 3; i++ {
	next:
		for j := 0; j < 3; j++ {
			if j == 1 {
				continue next
			}
			fmt.Printf("(%d,%d) ", i, j)
		}
	}
	fmt.Println()

	// --- 9. Range copies values (important!) ---
	fmt.Println("=== Range value copy ===")
	type Point struct{ X, Y int }
	points := []Point{{1, 2}, {3, 4}}
	for _, p := range points {
		p.X = 0 // modifies COPY, not original
	}
	fmt.Println(points) // [{1 2} {3 4}] unchanged

	// --- 10. Modify slice via index ---
	for i := range points {
		points[i].X = 0
	}
	fmt.Println(points) // [{0 2} {0 4}]
}
```

## 五、注意事项

| 要点 | 说明 |
|------|------|
| Go 只有 `for` | 没有 while，用 `for condition {}` 替代 |
| range 返回值是副本 | 修改 range 的 value 不影响原数据 |
| range string 返回 rune | 索引是字节位置，值是 rune |
| range map 顺序随机 | 每次遍历 map 的顺序可能不同 |
| for-range 中 `i` 可省略 | `for _, v := range slice` |
| range nil 切片/map 安全 | 循环体不执行，不会 panic |

## 六、避坑指南

| ❌ 错误写法 | ✅ 正确写法 | 原因 |
|------------|-----------|------|
| `for _, p := range pts { p.X = 0 }` | `for i := range pts { pts[i].X = 0 }` | range 值是副本 |
| `for i := 0; i <= len(s); i++` | `for i := 0; i < len(s); i++` | 越界 panic |
| 循环内 `defer` | 循环外处理或用匿名函数 | defer 在函数返回时执行，循环内会堆积 |
| goroutine 中用 range 变量 | 传入参数 `go func(v int) {...}(v)` | 闭包捕获变量引用 |

## 七、练习题

🟢 **基础题：** 用 for 循环计算 1+2+3+...+100 的和。

🟡 **进阶题：** 遍历一个字符串，分别统计其中英文字母、数字、其他字符的个数。

🔴 **挑战题：** 实现冒泡排序，对 `[]int{64, 34, 25, 12, 22, 11, 90}` 排序，使用带标签的 break 在已排序时提前退出。

## 九、举一反三

| 场景 | 用法 |
|------|------|
| 遍历切片 | `for i, v := range slice` |
| 只要索引 | `for i := range slice` |
| 遍历 map | `for k, v := range m` |
| 统计字符 | `for _, r := range str` |
| 批量处理 | `for i := 0; i < len(data); i += batchSize` |
| 重试逻辑 | `for retries > 0 { ... retries-- }` |

## 十一、代码演进

**v1 — 基础遍历：**
```go
nums := []int{1, 2, 3}
for i := 0; i < len(nums); i++ {
    fmt.Println(nums[i])
}
```

**v2 — range 迭代：**
```go
for i, v := range nums {
    fmt.Printf("[%d] %d\n", i, v)
}
```

**v3 — 修改元素 + 并发安全：**
```go
// Modify via index
for i := range nums {
    nums[i] *= 2
}

// Parallel processing with local copy
for _, v := range nums {
    v := v // shadow loop variable
    go func() { process(v) }()
}
```
