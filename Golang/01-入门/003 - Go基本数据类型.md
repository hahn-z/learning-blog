# 003 - Go 基本数据类型

> **难度：** ⭐ | **前置知识：** Go 程序基本结构（002篇） | **预估用时：** 45分钟

---

## 一句话定义

Go 是静态类型语言，每个变量在编译时就确定了类型——基本类型包括布尔、整数、浮点、复数、字符串等。

**现实类比：** 数据类型就像不同尺寸的收纳盒——装戒指用小盒（int8），装被子用大盒（int64），标签贴对了才能正确存放和取出。

**技术场景：** 选择正确的数据类型影响内存占用、计算精度和程序性能。

## 完整类型一览

```go
// types_overview.go - All basic types in Go
package main

import (
    "fmt"
    "math/cmplx"
    "unsafe"
)

func main() {
    // === Boolean ===
    var isActive bool = true
    var isDeleted bool // zero value: false
    fmt.Printf("bool: %v (%T), zero: %v\n", isActive, isActive, isDeleted)

    // === Signed Integers ===
    var i8 int8 = 127           // -128 to 127
    var i16 int16 = 32767      // -32768 to 32767
    var i32 int32 = 2147483647 // -2^31 to 2^31-1
    var i64 int64 = 1<<62      // -2^63 to 2^63-1
    var i int = 100            // platform dependent: 32 or 64 bit
    fmt.Printf("int8: %d, int16: %d, int32: %d, int64: %d, int: %d\n", i8, i16, i32, i64, i)

    // === Unsigned Integers ===
    var u8 uint8 = 255         // 0 to 255
    var u16 uint16 = 65535     // 0 to 65535
    var u32 uint32 = 1<<31     // 0 to 2^32-1
    var u64 uint64 = 1<<63     // 0 to 2^64-1
    var u uint = 200           // platform dependent
    fmt.Printf("uint8: %d, uint16: %d, uint32: %d, uint64: %d, uint: %d\n", u8, u16, u32, u64, u)

    // === byte and rune (aliases) ===
    var b byte = 'A'    // alias for uint8
    var r rune = '中'    // alias for int32, represents a Unicode code point
    fmt.Printf("byte: %c(%d), rune: %c(%d)\n", b, b, r, r)

    // === Floating Point ===
    var f32 float32 = 3.141592653589793 // ~7 decimal digits precision
    var f64 float64 = 3.141592653589793 // ~15 decimal digits precision
    fmt.Printf("float32: %.15f\n", f32) // note the precision loss
    fmt.Printf("float64: %.15f\n", f64)

    // === Complex Numbers ===
    var c64 complex64 = complex(1, 2)       // float32 real + imag
    var c128 complex128 = complex(3.5, -4.2) // float64 real + imag
    fmt.Printf("complex64: %v, real: %v, imag: %v\n", c64, real(c64), imag(c64))
    fmt.Printf("complex128: %v, real: %v, imag: %v\n", c128, real(c128), imag(c128))
    fmt.Printf("abs(c128): %v\n", cmplx.Abs(c128))

    // === String ===
    var s string = "Hello, 世界"
    fmt.Printf("string: %s, len: %d bytes\n", s, len(s)) // len=12 (each Chinese char=3 bytes)

    // === Size of types ===
    fmt.Printf("\nType sizes:\n")
    fmt.Printf("bool: %d bytes\n", unsafe.Sizeof(isActive))
    fmt.Printf("int: %d bytes (platform)\n", unsafe.Sizeof(i))
    fmt.Printf("int8: %d, int16: %d, int32: %d, int64: %d\n",
        unsafe.Sizeof(i8), unsafe.Sizeof(i16), unsafe.Sizeof(i32), unsafe.Sizeof(i64))
    fmt.Printf("float32: %d, float64: %d\n", unsafe.Sizeof(f32), unsafe.Sizeof(f64))
}
```

## 类型范围与大小

### 整数类型

| 类型 | 大小 | 范围 |
|------|------|------|
| `int8` | 1 byte | -128 ~ 127 |
| `int16` | 2 bytes | -32768 ~ 32767 |
| `int32` | 4 bytes | -2³¹ ~ 2³¹-1 |
| `int64` | 8 bytes | -2⁶³ ~ 2⁶³-1 |
| `int` | 平台相关 | 32 或 64 位 |
| `uint8` (`byte`) | 1 byte | 0 ~ 255 |
| `uint16` | 2 bytes | 0 ~ 65535 |
| `uint32` | 4 bytes | 0 ~ 2³²-1 |
| `uint64` | 8 bytes | 0 ~ 2⁶⁴-1 |
| `uint` | 平台相关 | 32 或 64 位 |
| `uintptr` | 平台相关 | 存放指针 |

### 浮点类型

| 类型 | 大小 | 精度 |
|------|------|------|
| `float32` | 4 bytes | ~7 位有效数字 |
| `float64` | 8 bytes | ~15 位有效数字 |

### 字符串特殊说明

```go
// string_details.go - String specifics
package main

import "fmt"

func main() {
    s := "Hello, 世界"

    // len() returns byte count, not character count
    fmt.Printf("Bytes: %d\n", len(s)) // 12

    // To count characters (runes), convert to []rune
    fmt.Printf("Characters: %d\n", len([]rune(s))) // 9

    // Strings are immutable
    // s[0] = 'h' // Compile error!

    // Access bytes by index
    fmt.Printf("First byte: %c (%d)\n", s[0], s[0]) // H (72)

    // Raw string literal (backtick, no escape processing)
    raw := `C:\Users\name\file.txt
Line 2
    "quoted"`
    fmt.Println(raw)

    // Interpreted string literal (double quote, supports escapes)
    interpreted := "Line1\nLine2\tTabbed"
    fmt.Println(interpreted)
}
```

## 类型转换

Go **没有隐式类型转换**，所有转换必须显式声明。

```go
// conversion.go - Type conversions in Go
package main

import (
    "fmt"
    "math"
    "strconv"
)

func main() {
    // === Numeric to Numeric ===
    var i int = 42
    var f float64 = float64(i)    // int -> float64
    var u uint = uint(f)          // float64 -> uint
    var i8 int8 = int8(i)         // int -> int8 (may overflow!)
    fmt.Printf("int→float64: %v, float64→uint: %v, int→int8: %v\n", f, u, i8)

    // === Numeric to String ===
    // Method 1: strconv (recommended)
    s := strconv.Itoa(42)           // int -> string
    fmt.Printf("Itoa: %q\n", s)

    s2 := strconv.FormatFloat(3.14, 'f', 2, 64) // float -> string
    fmt.Printf("FormatFloat: %q\n", s2)

    // Method 2: fmt.Sprintf (flexible but slower)
    s3 := fmt.Sprintf("%d", 42)
    fmt.Printf("Sprintf: %q\n", s3)

    // === String to Numeric ===
    n, err := strconv.Atoi("42")    // string -> int
    if err != nil {
        fmt.Println("Parse error:", err)
    }
    fmt.Printf("Atoi: %v (%T)\n", n, n)

    f, err := strconv.ParseFloat("3.14159", 64) // string -> float64
    if err != nil {
        fmt.Println("Parse error:", err)
    }
    fmt.Printf("ParseFloat: %v\n", f)

    // === byte/rune and string ===
    // string from bytes
    bytes := []byte("Hello")
    fmt.Printf("bytes: %v → string: %s\n", bytes, string(bytes))

    // string to runes
    runes := []rune("Hello, 世界")
    fmt.Printf("runes: %v\n", runes)
    for i, r := range runes {
        fmt.Printf("  [%d] %c (U+%04X)\n", i, r, r)
    }

    // === Overflow warning ===
    big := math.MaxInt64
    var small int8 = int8(big) // Overflow! Undefined behavior
    fmt.Printf("Overflow: %d → %d (garbage!)\n", big%128, small)
}
```

## 零值机制

Go 的每个类型都有默认的零值，声明但不赋值的变量自动获得零值。

```go
// zero_values.go - Go zero value mechanism
package main

import "fmt"

func main() {
    var b bool        // false
    var i int         // 0
    var f float64     // 0
    var s string      // "" (empty string)
    var p *int        // nil
    var sl []int      // nil (slice)
    var m map[string]int // nil (map)

    fmt.Printf("bool:        %v\n", b)
    fmt.Printf("int:         %v\n", i)
    fmt.Printf("float64:     %v\n", f)
    fmt.Printf("string:      %q\n", s)
    fmt.Printf("pointer:     %v\n", p)
    fmt.Printf("slice:       %v, nil=%v\n", sl, sl == nil)
    fmt.Printf("map:         %v, nil=%v\n", m, m == nil)

    // Practical use: check if a string was set
    name := ""
    if name == "" {
        fmt.Println("Name is not set (zero value)")
    }
}
```

**各类型零值汇总：**

| 类型 | 零值 |
|------|------|
| `bool` | `false` |
| 所有整数类型 | `0` |
| 所有浮点类型 | `0.0` |
| 所有复数类型 | `0+0i` |
| `string` | `""` (空字符串) |
| 指针、切片、映射、通道、接口、函数 | `nil` |

## 注意事项

| 事项 | 说明 |
|------|------|
| `int` 大小不确定 | 32 位系统上是 32 位，64 位系统上是 64 位，需要确定大小时用 `int64` |
| `float32` 精度不足 | 科学计算、金额计算务必用 `float64` |
| 字符串不可变 | Go 字符串是只读的，修改需转换为 `[]byte` 或 `[]rune` |
| `len(string)` 返回字节数 | 不是字符数，中文字符占 3 字节（UTF-8） |
| 溢出不会 panic | 整数溢出是未定义行为，不会报错但结果错误 |
| 隐式转换不存在 | `int` 和 `int32` 是不同类型，必须显式转换 |
| `string(byte)` 有效范围 | `byte` 范围 0-255，超出部分的 rune 转为 string 需要 `string(rune)` |

## 避坑指南

| ❌ 错误做法 | ✅ 正确做法 |
|------------|-----------|
| `var i int32 = 10; var j int64 = i` | `var j int64 = int64(i)` 显式转换 |
| `len("中文")` 认为返回 2 | `utf8.RuneCountInString("中文")` 返回 2 |
| 用 `float32` 做精确计算 | 金额用整数（分），科学计算用 `float64` |
| `int8(200)` 期望得到 200 | 会溢出，200-256=-56 |
| 字符串比较用 `==` 比较引用 | Go 的 `==` 比较值，字符串可直接 `==` 比较 |
| `for i := 0; i < len(s); i++` 遍历中文 | 用 `for i, r := range s` 正确遍历 rune |

## 举一反三

| 场景 | Go 做法 | 其他语言对比 |
|------|---------|------------|
| 类型转换 | `int(x)` 显式 | C 隐式提升 / Python 动态无转换 |
| 字符串遍历 | `for i, r := range s` | Python `for c in s` / JS `for...of` |
| 金额计算 | 用整数（分）或 decimal 库 | Java BigDecimal / Python Decimal |
| 未初始化变量 | 自动零值 | C 随机值 / Java 默认值 / Python None |
| 获取类型 | `fmt.Printf("%T", x)` | Python `type(x)` / JS `typeof x` |
| 整数溢出 | 静默溢出（配合 `math/bits` 检测） | Rust panic / Python 无限精度 |

### v3: 实用类型工具函数

```go
// main.go - practical type utilities
package main

import (
    "fmt"
    "strconv"
    "unicode/utf8"
)

// Safe integer parse with range check
func safeParseInt(s string, bits int) (int64, bool) {
    n, err := strconv.ParseInt(s, 10, bits)
    return n, err == nil
}

// Count different character types in string
func analyzeString(s string) (letters, digits, spaces, others int) {
    for _, r := range s {
        switch {
        case r >= 'a' && r <= 'z' || r >= 'A' && r <= 'Z':
            letters++
        case r >= '0' && r <= '9':
            digits++
        case r == ' ' || r == '\t':
            spaces++
        default:
            others++
        }
    }
    return
}

func main() {
    // Safe parsing
    examples := []string{"42", "99999999999999999999", "abc", "-1"}
    for _, s := range examples {
        if n, ok := safeParseInt(s, 32); ok {
            fmt.Printf("Parsed %q → %d\n", s, n)
        } else {
            fmt.Printf("Failed to parse %q\n", s)
        }
    }

    // String analysis
    test := "Hello 世界 123! @#$"
    l, d, sp, o := analyzeString(test)
    fmt.Printf("\nString: %q\n", test)
    fmt.Printf("Bytes: %d, Runes: %d\n", len(test), utf8.RuneCountInString(test))
    fmt.Printf("Letters: %d, Digits: %d, Spaces: %d, Others: %d\n", l, d, sp, o)

    // Type conversion chain
    var i int = 100
    f := float64(i)
    s2 := strconv.FormatFloat(f, 'f', 2, 64)
    fmt.Printf("\nConversion chain: int(%d) → float64(%v) → string(%q)\n", i, f, s2)
}
```
