# ES6+新增数据结构与迭代器完全解析

> **难度标注：** 🟡 中级→🔴 高级 | **前置知识：** 对象、数组、函数、ES6基础
> **阅读时间：** 约20分钟 | **代码量：** 约170行

---

## 一、概念讲解

### 1.1 新数据结构

- **Map：** 键值对集合，任意类型作键（vs Object只允许字符串/Symbol）
- **Set：** 值的唯一集合，自动去重
- **WeakMap/WeakSet：** 键必须是对象，且是弱引用（不阻止GC）

### 1.2 迭代协议

- **可迭代协议：** 对象实现`[Symbol.iterator]()`方法，返回迭代器
- **迭代器协议：** 对象实现`next()`方法，返回`{ value, done }`

### 1.3 Generator

`function*`声明的函数，可通过`yield`暂停和恢复执行。天然实现迭代器协议。

## 三、完整代码示例

```javascript
// ============================================
// 1. Map — key-value pairs with any key type
// ============================================
const userRoles = new Map();

// Set entries
userRoles.set("alice", "admin");
userRoles.set("bob", "editor");
userRoles.set(42, "the answer"); // non-string key

console.log(userRoles.get("alice"));  // "admin"
console.log(userRoles.has("bob"));    // true
console.log(userRoles.size);          // 3

// Iteration
for (const [key, value] of userRoles) {
  console.log(`${key} => ${value}`);
}

// Initialize from entries
const config = new Map([
  ["host", "localhost"],
  ["port", 3000],
]);
console.log(config.get("port")); // 3000

// Convert to object/array
console.log(Object.fromEntries(userRoles));
console.log([...userRoles.keys()]);
console.log([...userRoles.values()]);

// ============================================
// 2. Set — unique values
// ============================================
const tags = new Set(["js", "es6", "js", "web"]);
console.log(tags.size); // 3 (duplicates removed)

tags.add("node");
tags.delete("web");
console.log(tags.has("es6")); // true

// Set operations
const a = new Set([1, 2, 3]);
const b = new Set([2, 3, 4]);

const union = new Set([...a, ...b]);
console.log([...union]); // [1, 2, 3, 4]

const intersection = new Set([...a].filter(x => b.has(x)));
console.log([...intersection]); // [2, 3]

const difference = new Set([...a].filter(x => !b.has(x)));
console.log([...difference]); // [1]

// ============================================
// 3. WeakMap — weak-referenced keys
// ============================================
const privateData = new WeakMap();

function createSecret(obj, secret) {
  privateData.set(obj, secret);
}
function getSecret(obj) {
  return privateData.get(obj);
}

const myObj = { id: 1 };
createSecret(myObj, "hidden value");
console.log(getSecret(myObj)); // "hidden value"
// When myObj is GC'd, the WeakMap entry is also cleaned up

// ============================================
// 4. Iterators & Iterable protocol
// ============================================
// Custom iterable
class NumberRange {
  constructor(start, end, step = 1) {
    this.start = start;
    this.end = end;
    this.step = step;
  }

  [Symbol.iterator]() {
    let current = this.start;
    const { end, step } = this;
    return {
      next() {
        if (current < end) {
          const value = current;
          current += step;
          return { value, done: false };
        }
        return { value: undefined, done: true };
      },
    };
  }
}

const range = new NumberRange(0, 5, 2);
console.log([...range]); // [0, 2, 4]

for (const num of range) {
  console.log(num); // 0, 2, 4
}

// ============================================
// 5. Generator functions
// ============================================
function* idGenerator() {
  let id = 1;
  while (true) {
    yield id++;
  }
}

const gen = idGenerator();
console.log(gen.next()); // { value: 1, done: false }
console.log(gen.next()); // { value: 2, done: false }
console.log(gen.next()); // { value: 3, done: false }

// Fibonacci generator
function* fibonacci() {
  let [prev, curr] = [0, 1];
  while (true) {
    yield curr;
    [prev, curr] = [curr, prev + curr];
  }
}

// Take first 8 Fibonacci numbers
const fib = fibonacci();
const first8 = Array.from({ length: 8 }, () => fib.next().value);
console.log(first8); // [1, 1, 2, 3, 5, 8, 13, 21]

// Generator with delegation
function* concatIterables(...iterables) {
  for (const iter of iterables) {
    yield* iter;
  }
}

const combined = concatIterables([1, 2], "hi", new Set([3, 4]));
console.log([...combined]); // [1, 2, 'h', 'i', 3, 4]

// ============================================
// 6. Async generators (ES2018)
// ============================================
async function* asyncCounter(max) {
  for (let i = 1; i <= max; i++) {
    await new Promise(r => setTimeout(r, 10));
    yield i;
  }
}

(async () => {
  const results = [];
  for await (const num of asyncCounter(3)) {
    results.push(num);
  }
  console.log("Async gen:", results); // [1, 2, 3]
})();
```

## 五、注意事项

| 结构 | 特点 | 适用场景 | 注意 |
|------|------|----------|------|
| Map | 任意键类型 | 频繁增删的键值对 | 比Object更安全 |
| Set | 自动去重 | 去重、成员检测 | NaN === NaN在Set中 |
| WeakMap | 弱引用键 | 私有数据、缓存 | 不可遍历 |
| WeakSet | 弱引用值 | 标记对象 | 不可遍历 |
| Generator | 惰性求值 | 无限序列、异步流 | 需手动调用next() |

## 六、避坑指南

### ❌ 用Object做频繁增删的字典
```javascript
const dict = {};
dict[key] = value;
delete dict[key]; // slow, triggers V8 deoptimization
```
### ✅ 用Map
```javascript
const dict = new Map();
dict.set(key, value);
dict.delete(key); // optimized for frequent changes
```

### ❌ Generator只遍历一次
```javascript
function* gen() { yield 1; yield 2; }
const g = gen();
console.log([...g]); // [1, 2]
console.log([...g]); // [] — exhausted!
```
### ✅ 创建新的Generator实例
```javascript
const g1 = gen();
const g2 = gen();
console.log([...g1]); // [1, 2]
console.log([...g2]); // [1, 2]
```

## 十一、代码演进

### v1: 用Object模拟Map
```javascript
const cache = {};
cache["key1"] = "value1";
// Problem: keys are always strings, prototype pollution risk
```

### v2: ES6 Map
```javascript
const cache = new Map();
cache.set(objKey, "value1");
cache.set(42, "value2");
// Any key type, no prototype issues
```

### v3: WeakMap私有 + Map缓存 + Generator
```javascript
const privateStore = new WeakMap();
const cache = new Map();

function* cachedIterator(key, generator) {
  if (cache.has(key)) yield* cache.get(key);
  const results = [];
  for (const item of generator()) {
    results.push(item);
    yield item;
  }
  cache.set(key, results);
}
```

## 十二、总结

| 核心要点 | 说明 |
|----------|------|
| **Map vs Object** | Map键类型灵活、有序、大小可查 |
| **Set去重** | 自动去重，比`[...new Set(arr)]`更优雅 |
| **WeakMap** | 私有数据存储，不阻止垃圾回收 |
| **迭代协议** | 自定义可迭代对象的关键 |
| **Generator** | 惰性求值、无限序列、异步流 |
