# 正则表达式实战

> **难度：** ⭐⭐⭐ 中等 | **分类：** JavaScript 核心 | **阅读时间：** ~15分钟

## 一、概念讲解

正则表达式（Regular Expression）是文本模式匹配的强大工具。JavaScript 使用 `RegExp` 对象支持正则，配合 `String` 的 `match`、`replace`、`search`、`split` 等方法使用。

核心要素：
- **字面量 vs 构造函数**：`/pattern/flags` vs `new RegExp('pattern', 'flags')`
- **字符类**：`\d`(数字)、`\w`(单词字符)、`\s`(空白)、`.`(任意)
- **量词**：`*`(0+)、`+`(1+)、`?`(0/1)、`{n,m}`(n到m次)
- **锚点**：`^`(开头)、`$`(结尾)、`\b`(单词边界)
- **分组**：`()`捕获、`(?:)`非捕获、`(?<name>)`命名捕获
- **标志**：`g`(全局)、`i`(忽略大小写)、`m`(多行)、`s`(dotAll)、`u`(Unicode)

## 三、完整 JS 代码

```javascript
// ==================== Common Patterns ====================

// Email validation (practical, not RFC-compliant)
const emailReg = /^[\w.-]+@[\w.-]+\.\w{2,}$/;
console.log(emailReg.test('user@example.com')); // true

// Chinese mobile number
const phoneReg = /^1[3-9]\d{9}$/;
console.log(phoneReg.test('13800138000')); // true

// URL parsing with named groups
const urlReg = /^(?<protocol>https?):\/\/(?<host>[^/]+)(?<path>\/.*)?$/;
const urlMatch = 'https://example.com/path/to/page'.match(urlReg);
console.log(urlMatch.groups);
// { protocol: 'https', host: 'example.com', path: '/path/to/page' }

// ==================== Replace with Callback ====================

// Template engine — replace {{key}} with values
function render(template, data) {
  return template.replace(/\{\{(\w+)\}\}/g, (_, key) => data[key] ?? '');
}

const html = render('Hello {{name}}, you have {{count}} messages', {
  name: 'Alice', count: 5
});
console.log(html); // 'Hello Alice, you have 5 messages'

// ==================== Advanced: Lookbehind & Lookahead ====================

// Password strength: at least 8 chars, 1 uppercase, 1 lowercase, 1 digit
const strongPwd = /^(?=.*[a-z])(?=.*[A-Z])(?=.*\d).{8,}$/;
console.log(strongPwd.test('Abc12345')); // true
console.log(strongPwd.test('password')); // false

// Match numbers not preceded by $
const priceText = 'Price: $100, discount 20, total $80';
const discounts = priceText.matchAll(/(?<!\$)\b\d+\b/g);
for (const m of discounts) {
  console.log(m[0]); // '20'
}

// ==================== String.prototype.split with Regex ====================

// Split by multiple delimiters
const csv = 'one,two;three|four';
console.log(csv.split(/[,;|]/)); // ['one', 'two', 'three', 'four']

// ==================== Practical: Parse Log Line ====================

const logReg = /^\[(?<time>\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2})\]\s*(?<level>\w+)\s*(?<msg>.+)$/;
const logLine = '[2024-01-15 10:30:00] ERROR Database connection failed';
const parsed = logLine.match(logReg);
console.log(parsed?.groups);
// { time: '2024-01-15 10:30:00', level: 'ERROR', msg: 'Database connection failed' }
```

## 五、注意事项

| 项目 | 说明 |
|------|------|
| 贪婪 vs 非贪婪 | `*` 和 `+` 默认贪婪，加 `?` 变非贪婪：`.*?` |
| `.` 不匹配换行 | 除非使用 `s` 标志（dotAll 模式） |
| `g` 标志有状态 | 带 `g` 的正则 `lastIndex` 会变化，重复 `exec` 要小心 |
| 正则不是万能的 | 复杂 HTML 解析请用 DOM parser，别用正则 |
| 转义特殊字符 | `[` `]` `(` `)` `.` `*` `+` `?` `\` `^` `$` `{` `}` `|` 需转义 |

## 六、避坑指南

❌ **用正则解析 HTML：**
```javascript
const html = '<div>content</div>';
html.match(/<div>(.*)<\/div>/); // ❌ Fragile! Nested tags break it
```

✅ **用 DOM API：**
```javascript
const doc = new DOMParser().parseFromString(html, 'text/html');
doc.querySelector('div').textContent; // ✅
```

❌ **忘记转义用户输入：**
```javascript
new RegExp(userInput); // ❌ If userInput = '.*', matches everything
```

✅ **转义特殊字符：**
```javascript
function escapeRegex(str) {
  return str.replace(/[.*+?^${}()|[\]\\]/g, '\\$&');
}
new RegExp(escapeRegex(userInput)); // ✅
```

❌ **`g` 标志导致 `test()` 反复切换：**
```javascript
const re = /a/g;
re.test('a'); // true
re.test('a'); // false! (lastIndex moved past 'a')
```

✅ **每次创建新正则或手动重置：**
```javascript
/a/g.test('a'); // true
/a/g.test('a'); // true ✅ (new regex each time)
```

## 十一、代码演进

```javascript
// v1: Basic match and replace
const str = 'Hello World';
str.match(/World/); // ['World']
str.replace(/World/, 'JS'); // 'Hello JS'

// v2: Named groups + flags
const re = /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/;
const { year, month, day } = re.exec('2024-01-15').groups;

// v3: Reusable pattern library with builder
class RegexBuilder {
  #pattern = '';
  #flags = '';

  start() { this.#pattern += '^'; return this; }
  end() { this.#pattern += '$'; return this; }
  digits(n) { this.#pattern += n ? `\\d{${n}}` : '\\d+'; return this; }
  literal(s) { this.#pattern += s.replace(/[.*+?^${}()|[\]\\]/g, '\\$&'); return this; }
  group(fn) { this.#pattern += '('; fn(this); this.#pattern += ')'; return this; }
  flags(f) { this.#flags = f; return this; }
  build() { return new RegExp(this.#pattern, this.#flags); }
}

// Build: ^\d{4}-\d{2}-\d{2}$
const dateRe = new RegexBuilder()
  .start().digits(4).literal('-').digits(2).literal('-').digits(2).end()
  .build();
console.log(dateRe.test('2024-01-15')); // true
```
