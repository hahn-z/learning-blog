一、概念讲解

箭头函数和解构赋值是ES6中使用频率最高的两个特性。它们经常组合使用，让代码更简洁。

1.1 箭头函数核心

  • 简洁语法：x => x * 2
  • 隐式返回单表达式
  • 词法this绑定

    • 1.2 解构赋值核心

    • 数组按位置：const [a, b] = [1, 2]
    • 对象按属性名：const { x, y } = { x: 1, y: 2 }
    • 默认值：const { z = 3 } = {}

      • 1.3 组合威力

      函数参数解构 + 箭头函数 = 自文档化、简洁、默认值。

      三、完整代码示例

      // ============================================
// 1. Arrow + destructuring in callbacks
// ============================================
const users = [
  { name: "Alice", age: 25, city: "Beijing" },
  { name: "Bob", age: 30, city: "Shanghai" },
  { name: "Charlie", age: 35, city: "Beijing" },
];

// Extract names with arrow + destructuring
const names = users.map(({ name }) => name);
console.log(names); // ["Alice", "Bob", "Charlie"]

// Filter and transform
const beijingAdults = users
  .filter(({ city, age }) => city === "Beijing" && age >= 18)
  .map(({ name, age }) => `${name} (${age})`);
console.log(beijingAdults); // ["Alice (25)", "Charlie (35)"]

// Configuration function
const createChart = ({ data, type = "bar", color = "blue", animated = true }) => ({
  type,

  style: { color, animated },
  render: () => `Rendering ${type} chart with ${data.length} data points`,
});

console.log(createChart({ data: [1, 2, 3] }).render());
console.log(createChart({ data: [4, 5], type: "line", color: "red" }).render());

// 4. Practical: default configs with override
// ============================================
const defaultConfig = {
  timeout: 5000,
  headers: { "Content-Type": "application/json" },
  retries: 3,
};

const createRequest = (url, userConfig = {}) => {
  const {
    timeout = defaultConfig.timeout,
    headers = defaultConfig.headers,
    retries = defaultConfig.retries,
  } = userConfig;

  return { url, timeout, headers, retries };
};

console.log(createRequest("https://api.example.com"));
console.log(createRequest("https://api.example.com", { timeout: 10000, retries: 1 }));

// ============================================
// 5. Arrow + rest/spread + destructuring
// ============================================
const mergeConfigs = (...configs) => {
  return configs.reduce((acc, { headers = {}, ...rest }) => ({
    ...acc,
    ...rest,
    headers: { ...acc.headers, ...headers },
  }), {});
};

const merged = mergeConfigs(
  { timeout: 3000, headers: { "X-Custom": "a" } },
  { retries: 2, headers: { Authorization: "Bearer token" } },
);
console.log(merged);

      五、注意事项

      组合用法 示例 注意
      参数解构 ({ a, b }) => 必须加括号包裹
      返回对象 => ({ key: val }) 括号防止被当作代码块
      嵌套解构 ({ a: { b } }) 过深会降低可读性
      默认值 ({ x = 1 }) 只在undefined时生效
      剩余参数 ({ a, ...rest }) rest不包含a

      六、避坑指南

      ❌ 忘记括号包裹解构参数

      const fn = { a, b } => a + b; // SyntaxError

      ✅ 正确写法

      const fn = ({ a, b }) => a + b; // OK

      ❌ 解构null参数

      const fn = ({ a = 1 }) => a;
fn(null); // TypeError

      ✅ 添加默认对象

      const fn = ({ a = 1 } = {}) => a;
fn(null); // still TypeError, but fn() works
const safeFn = (arg) => {
  const { a = 1 } = arg || {};
  return a;
};
